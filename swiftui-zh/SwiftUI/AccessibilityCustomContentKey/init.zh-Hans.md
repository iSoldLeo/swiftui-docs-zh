# init(_:)

**Initializer**

创建带有指定标签的`AccessibilityCustomContentKey`。

## 声明

```swift
init(_ label: LocalizedStringResource)
```

## 参数

- **label**：向用户描述此附加信息条目内容的本地化文本。例如"方向"。这也将用作标识符。

## 创建关键字

- **init(_:id:)**：用指定的标签和标识符创建`AccessibilityCustomContentKey`。

# init(_:)

**Initializer**

Create an `AccessibilityCustomContentKey` with the specified label.

## Declaration

```swift
init(_ label: LocalizedStringResource)
```

## Parameters

- **label**: Localized text describing to the user what is contained in this additional information entry. For example: “orientation”. This will also be used as the identifier.

## Creating a key

- **init(_:id:)**: Create an `AccessibilityCustomContentKey` with the specified label and identifier.
