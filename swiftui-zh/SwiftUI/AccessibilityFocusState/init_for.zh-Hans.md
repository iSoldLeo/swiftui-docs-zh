# init(for:)

**Initializer**

使用您指定的辅助功能技术为布尔值创建新的辅助功能焦点状态。

### 声明

```swift
init(for technologies: AccessibilityTechnologies) where Value == Bool
```

## 参数

- **technologies**：其中一个可用 [AccessibilityTechnologies](../AccessibilityTechnologies.zh-Hans.md)。

## 创建焦点状态

- **init()**：为布尔值创建新的无障碍焦点状态。

# init(for:)

**Initializer**

Creates a new accessibility focus state for a Boolean value, using the accessibility technologies you specify.

## Declaration

```swift
init(for technologies: AccessibilityTechnologies) where Value == Bool
```

## Parameters

- **technologies**: One of the available [AccessibilityTechnologies](../AccessibilityTechnologies.zh-Hans.md).

## Creating a focus state

- **init()**: Creates a new accessibility focus state for a Boolean value.
