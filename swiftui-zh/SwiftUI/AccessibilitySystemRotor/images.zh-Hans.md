# 图像

**类型属性**

系统旋转器，允许用户遍历所有图像。

## 声明

```swift
static var images: AccessibilitySystemRotor { get }
```

## 迭代其他元素

- **landmarks**：系统旋转器允许用户遍历所有地标。
- **lists**：允许用户遍历所有列表的系统旋转器。
- **tables**：允许用户遍历所有表格的系统旋转器。
