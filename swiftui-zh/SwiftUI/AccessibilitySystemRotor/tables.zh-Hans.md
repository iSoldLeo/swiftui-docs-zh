# 表格

**类型属性**

系统旋转器，允许用户遍历所有表格。

## 声明

```swift
static var tables: AccessibilitySystemRotor { get }
```

## 迭代其他元素

- **images**：系统旋转器允许用户遍历所有图像。
- **landmarks**：允许用户遍历所有地标的系统旋转器。
- **lists**：允许用户遍历所有列表的系统旋转器。
