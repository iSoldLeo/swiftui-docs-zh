# ActionSheet.Button

**Type Alias**

A button representing an operation of an action sheet presentation.

## Declaration

```swift
typealias Button = Alert.Button
```

## Discussion

The [ActionSheet](../ActionSheet.zh-Hans.md) button is type-aliased to the [Alert](../Alert.zh-Hans.md) button type, which provides default, cancel, and destructive styles.
