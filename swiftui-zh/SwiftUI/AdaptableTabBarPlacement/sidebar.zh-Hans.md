# 侧边栏

**类型属性**

标签视图的边栏。

## 声明

```swift
static let sidebar: AdaptableTabBarPlacement
```

## 讨论

在[sidebarAdaptable](../TabViewStyle/sidebarAdaptable.zh-Hans.md)中，标签视图可以作为边栏出现。侧边栏选项卡代表选项卡视图中的所有选项卡。

# sidebar

**Type Property**

The sidebar of a tab view.

## Declaration

```swift
static let sidebar: AdaptableTabBarPlacement
```

## Discussion

A tab view can appear as a sidebar in [sidebarAdaptable](../TabViewStyle/sidebarAdaptable.zh-Hans.md). Sidebar tabs represent all of the tabs in a tab view.
