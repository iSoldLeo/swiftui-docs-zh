## 获取文本基线参考线

- **leadingFirstTextBaseline**：用于标记视图中文本前缘和最顶部基线的参考线。

- **centerFirstTextBaseline**：用于标记视图中最顶部基线的参考线。

- **trailingFirstTextBaseline**：用于标记视图中文本尾缘和顶部基线的参考线。

- **centerLastTextBaseline**：用于标记视图中文本底部基线的参考线。

- **trailingLastTextBaseline**：用于标记视图中文本尾缘和底部基线的参考线。

# leadingLastTextBaseline

**Type Property**

A guide that marks the leading edge and bottom-most text baseline in a view.

## Declaration

```swift
static var leadingLastTextBaseline: Alignment { get }
```

## Discussion

This alignment combines the [leading](../HorizontalAlignment/leading.zh-Hans.md) horizontal guide and the [lastTextBaseline](../VerticalAlignment/lastTextBaseline.zh-Hans.md) vertical guide:



## Getting text baseline guides

- **leadingFirstTextBaseline**: A guide that marks the leading edge and top-most text baseline in a view.
- **centerFirstTextBaseline**: A guide that marks the top-most text baseline in a view.
- **trailingFirstTextBaseline**: A guide that marks the trailing edge and top-most text baseline in a view.
- **centerLastTextBaseline**: A guide that marks the bottom-most text baseline in a view.
- **trailingLastTextBaseline**: A guide that marks the trailing edge and bottom-most text baseline in a view.
