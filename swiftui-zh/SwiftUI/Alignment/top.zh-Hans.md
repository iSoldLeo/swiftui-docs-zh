# top

**Type Property**

A guide that marks the top edge of the view.

## Declaration

```swift
static let top: Alignment
```

## Discussion

This alignment combines the [center](../HorizontalAlignment/center.zh-Hans.md) horizontal guide and the [top](../VerticalAlignment/top.zh-Hans.md) vertical guide:



## Getting top guides

- **topLeading**: A guide that marks the top and leading edges of the view.
- **topTrailing**: A guide that marks the top and trailing edges of the view.
