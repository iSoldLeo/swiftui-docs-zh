**类型属性**


## 讨论

此对齐方式结合了 [trailing](../HorizontalAlignment/trailing.zh-Hans.md) 水平参考线和 [firstTextBaseline](../VerticalAlignment/firstTextBaseline.zh-Hans.md) 垂直参考线：

## 获取文本基线参考线

- **leadingFirstTextBaseline**：用于标记视图中文本前缘和顶部基线的参考线。

- **centerFirstTextBaseline**：用于标记视图中文本顶部基线的参考线。

- **leadingLastTextBaseline**：用于标记视图中文本前缘和最底部基线的参考线。

- **centerLastTextBaseline**：用于标记视图中文本最底部基线的参考线。

- **trailingLastTextBaseline**：用于标记视图中文本后缘和最底部基线的参考线。

# trailingFirstTextBaseline

**Type Property**

A guide that marks the trailing edge and top-most text baseline in a view.

## Declaration

```swift
static var trailingFirstTextBaseline: Alignment { get }
```

## Discussion

This alignment combines the [trailing](../HorizontalAlignment/trailing.zh-Hans.md) horizontal guide and the [firstTextBaseline](../VerticalAlignment/firstTextBaseline.zh-Hans.md) vertical guide:



## Getting text baseline guides

- **leadingFirstTextBaseline**: A guide that marks the leading edge and top-most text baseline in a view.
- **centerFirstTextBaseline**: A guide that marks the top-most text baseline in a view.
- **leadingLastTextBaseline**: A guide that marks the leading edge and bottom-most text baseline in a view.
- **centerLastTextBaseline**: A guide that marks the bottom-most text baseline in a view.
- **trailingLastTextBaseline**: A guide that marks the trailing edge and bottom-most text baseline in a view.
