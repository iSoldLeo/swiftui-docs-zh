# init(colors:center:angle:)

**Initializer**

从颜色集合中创建一个圆锥梯度，完成一个完整的转弯。

## 声明

```swift
init(colors: [Color], center: UnitPoint, angle: Angle = .zero)
```

## 创建一个完整的旋转角度梯度

- **init(gradient:center:angle:)**：创建完成一个完整旋转的圆锥梯度。
- **init(stops:center:angle:)**：从颜色止点集合创建一个圆锥形渐变效果，并完成一个完整的转弯。

# init(colors:center:angle:)

**Initializer**

Creates a conic gradient from a collection of colors that completes a full turn.

## Declaration

```swift
init(colors: [Color], center: UnitPoint, angle: Angle = .zero)
```

## Creating a full rotation angular gradient

- **init(gradient:center:angle:)**: Creates a conic gradient that completes a full turn.
- **init(stops:center:angle:)**: Creates a conic gradient from a collection of color stops that completes a full turn.
