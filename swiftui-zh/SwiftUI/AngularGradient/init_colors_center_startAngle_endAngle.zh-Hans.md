# init(colors:center:startAngle:endAngle:)

**Initializer**

从颜色集合中创建角度梯度。

## 声明

```swift
init(colors: [Color], center: UnitPoint, startAngle: Angle, endAngle: Angle)
```

## 创建部分旋转角梯度

- **init(gradient:center:startAngle:endAngle:)**：创建角度梯度。
- **init(stops:center:startAngle:endAngle:)**：从一系列色块创建角度渐变。

# init(colors:center:startAngle:endAngle:)

**Initializer**

Creates an angular gradient from a collection of colors.

## Declaration

```swift
init(colors: [Color], center: UnitPoint, startAngle: Angle, endAngle: Angle)
```

## Creating a partial rotation angular gradient

- **init(gradient:center:startAngle:endAngle:)**: Creates an angular gradient.
- **init(stops:center:startAngle:endAngle:)**: Creates an angular gradient from a collection of color stops.
