# init(gradient:center:angle:)

**Initializer**

创建完成一个完整转弯的圆锥梯度。

## 声明

```swift
init(gradient: Gradient, center: UnitPoint, angle: Angle = .zero)
```

## 创建一个完整的旋转角度梯度

- **init(colors:center:angle:)**：从颜色集合中创建一个圆锥形渐变，完成一个完整的旋转。
- **init(stops:center:angle:)**：从颜色停止点集合创建圆锥梯度，并完成一个完整的转弯。

# init(gradient:center:angle:)

**Initializer**

Creates a conic gradient that completes a full turn.

## Declaration

```swift
init(gradient: Gradient, center: UnitPoint, angle: Angle = .zero)
```

## Creating a full rotation angular gradient

- **init(colors:center:angle:)**: Creates a conic gradient from a collection of colors that completes a full turn.
- **init(stops:center:angle:)**: Creates a conic gradient from a collection of color stops that completes a full turn.
