# init(stops:center:angle:)

**Initializer**

从颜色止点集合中创建一个圆锥形渐变，并完成一个完整的转弯。

## 声明

```swift
init(stops: [Gradient.Stop], center: UnitPoint, angle: Angle = .zero)
```

- **init(gradient:center:angle:)**：创建完成一个完整旋转的圆锥梯度。
- **init(colors:center:angle:)**：从颜色集合中创建一个圆锥梯度，并完成一个完整的转弯。

# init(stops:center:angle:)

**Initializer**

Creates a conic gradient from a collection of color stops that completes a full turn.

## Declaration

```swift
init(stops: [Gradient.Stop], center: UnitPoint, angle: Angle = .zero)
```

## Creating a full rotation angular gradient

- **init(gradient:center:angle:)**: Creates a conic gradient that completes a full turn.
- **init(colors:center:angle:)**: Creates a conic gradient from a collection of colors that completes a full turn.
