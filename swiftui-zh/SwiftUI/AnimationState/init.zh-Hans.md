# init()

**Initializer**

Create an empty state container.

## Declaration

```swift
init()
```

## Discussion

You don’t typically create an instance of [AnimationState](../AnimationState.zh-Hans.md) directly. Instead, the [AnimationContext](../AnimationContext.zh-Hans.md) provides the animation state to an instance of [CustomAnimation](../CustomAnimation.zh-Hans.md).
