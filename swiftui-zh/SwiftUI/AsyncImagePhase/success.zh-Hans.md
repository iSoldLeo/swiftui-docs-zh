# AsyncImagePhase.success(_:)

**Case**

成功加载图像。

## 声明

```swift
case success(Image)
```

## 获取负载阶段

- **AsyncImagePhase.empty**：没有加载图像。
- **AsyncImagePhase.failure(_:)**：图像加载失败并出现错误。

# AsyncImagePhase.success(_:)

**Case**

An image successfully loaded.

## Declaration

```swift
case success(Image)
```

## Getting load phases

- **AsyncImagePhase.empty**: No image is loaded.
- **AsyncImagePhase.failure(_:)**: An image failed to load with an error.
