# AutomaticMenuBarExtraStyle

**Structure**

默认菜单栏额外样式。您也可以使用 [automatic](MenuBarExtraStyle/automatic.zh-Hans.md) 构建此样式。

### 声明

```swift
struct AutomaticMenuBarExtraStyle
```

## 创建菜单栏额外样式

- **init()**：创建自动菜单栏额外样式。

## 支持类型

- **PullDownMenuBarExtraStyle**：菜单栏额外样式，可将其内容渲染为从菜单栏图标下拉的菜单。
- **WindowMenuBarExtraStyle**：在弹出窗口中显示其内容的菜单栏额外样式。

## 符合

- 菜单栏额外样式

# AutomaticMenuBarExtraStyle

**Structure**

The default menu bar extra style. You can also use [automatic](MenuBarExtraStyle/automatic.zh-Hans.md) to construct this style.

## Declaration

```swift
struct AutomaticMenuBarExtraStyle
```

## Creating the menu bar extra style

- **init()**: Creates an automatic menu bar extra style.

## Supporting types

- **PullDownMenuBarExtraStyle**: A menu bar extra style that renders its contents as a menu that pulls down from the icon in the menu bar.
- **WindowMenuBarExtraStyle**: A menu bar extra style that renders its contents in a popover-like window.

## Conforms To

- MenuBarExtraStyle
