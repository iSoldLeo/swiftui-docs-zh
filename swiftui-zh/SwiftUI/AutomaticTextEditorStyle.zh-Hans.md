# AutomaticTextEditorStyle

**Structure**

基于文本编辑器上下文的默认文本编辑器样式。

## 声明

```swift
@MainActor @preconcurrency struct AutomaticTextEditorStyle
```

## 概述

您也可以使用 [automatic](TextEditorStyle/automatic.zh-Hans.md) 来构造此样式。

## 创建文本编辑器样式

- **init()**：创建默认文本编辑器样式的实例。

## 支持的类型

- **PlainTextEditorStyle**：无装饰的文本编辑器样式。

- **RoundedBorderTextEditorStyle**：一种带有系统定义的圆角边框的文本编辑器样式。

## 符合以下样式：

- TextEditorStyle

# AutomaticTextEditorStyle

**Structure**

The default text editor style, based on the text editor’s context.

## Declaration

```swift
@MainActor @preconcurrency struct AutomaticTextEditorStyle
```

## Overview

You can also use [automatic](TextEditorStyle/automatic.zh-Hans.md) to construct this style.

## Creating the text editor style

- **init()**: Creates an instance of the default text editor style.

## Supporting types

- **PlainTextEditorStyle**: A text editor style with no decoration.
- **RoundedBorderTextEditorStyle**: A text editor style with a system-defined rounded border.

## Conforms To

- TextEditorStyle
