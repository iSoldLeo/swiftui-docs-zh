# standard

**类型属性**

背景的标准突出显示级别

## 声明

```swift
static let standard: BackgroundProminence
```

## 讨论

这是默认的突出显示级别，无需任何调整即可与背景形成令人满意的对比度。

## 获取背景突出显示级别

- **increased**：更突出显示的背景，可能需要对其上方的视图进行一些更改。

# standard

**Type Property**

The standard prominence of a background

## Declaration

```swift
static let standard: BackgroundProminence
```

## Discussion

This is the default level of prominence and doesn’t require any adjustment to achieve satisfactory contrast with the background.

## Getting background prominence

- **increased**: A more prominent background that likely requires some changes to the views above it.
