# appRefresh

**Type Property**

A task that updates your app’s state in the background.

## Declaration

```swift
static var appRefresh: BackgroundTask<String?, Void> { get }
```

## Refreshing the app

- **appRefresh(_:)**: A task that updates your app’s state in the background for a matching identifier.
