# init(projectedValue:)

**Initializer**

根据另一个可绑定对象的值创建一个可绑定对象。

## 声明

```swift
init(projectedValue: Bindable<Value>)
```

## 创建可绑定值

- **init(_:)**：根据可观察对象创建一个可绑定对象。

- **init(wrappedValue:)**：根据可观察对象创建一个可绑定对象。

# init(projectedValue:)

**Initializer**

Creates a bindable from the value of another bindable.

## Declaration

```swift
init(projectedValue: Bindable<Value>)
```

## Creating a bindable value

- **init(_:)**: Creates a bindable object from an observable object.
- **init(wrappedValue:)**: Creates a bindable object from an observable object.
