# id

**Instance Property**

The stable identity of the entity associated with this instance, corresponding to the `id` of the binding’s wrapped value.

## Declaration

```swift
var id: Value.ID { get }
```

## Managing changes

- **animation(_:)**: Specifies an animation to perform when the binding value changes.
- **transaction(_:)**: Specifies a transaction for the binding.
- **transaction**: The binding’s transaction.
