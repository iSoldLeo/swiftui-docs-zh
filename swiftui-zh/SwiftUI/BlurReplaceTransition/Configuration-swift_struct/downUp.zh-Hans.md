# 向下向上

**类型属性**

一种配置，用于请求在移除视图时缩放视图、在插入视图时向上缩放视图的转换。

## 声明

```swift
static let downUp: BlurReplaceTransition.Configuration
```

## 获取过渡配置

- **upUp**：请求转换的配置，在移除和插入视图时都能放大视图。

# downUp

**Type Property**

A configuration that requests a transition that scales the view down while removing it and up while inserting it.

## Declaration

```swift
static let downUp: BlurReplaceTransition.Configuration
```

## Getting the transition configuration

- **upUp**: A configuration that requests a transition that scales the view up while both removing and inserting it.
