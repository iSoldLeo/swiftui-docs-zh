# init()

**Initializer**

## 声明

```swift
init()
```

## 创建表格样式

- **init(alternatesRowBackgrounds:)**：创建一个带有可选交替行背景的内嵌表格样式。

# init()

**Initializer**

Creates a default bordered table style, with alternating row backgrounds.

## Declaration

```swift
init()
```

## Creating the table style

- **init(alternatesRowBackgrounds:)**: Creates an inset table style with optional alternating row backgrounds.
