# BorderlessButtonMenuStyle

**Structure**

一种菜单样式，用于显示一个无边框按钮，按下该按钮即可切换菜单内容的显示。

## 声明

```swift
struct BorderlessButtonMenuStyle
```

## 概述

使用 [borderlessButton](MenuStyle/borderlessButton.zh-Hans.md) 来构建此样式。

## 创建无边框按钮菜单样式

- **init()**：创建无边框按钮菜单样式。

- **init(showsMenuIndicator:)**：创建无边框按钮菜单样式，并指定是否显示可视菜单指示器。

## 支持的类型

- **DefaultMenuStyle**：默认菜单样式，基于菜单上下文。

- **ButtonMenuStyle**：一种菜单样式，显示一个按钮，按下该按钮可切换菜单内容的显示。

- **BorderedButtonMenuStyle**：一种菜单样式，显示一个带边框的按钮，按下该按钮可切换菜单内容的显示。

# BorderlessButtonMenuStyle

**Structure**

A menu style that displays a borderless button that toggles the display of the menu’s contents when pressed.

## Declaration

```swift
struct BorderlessButtonMenuStyle
```

## Overview

Use [borderlessButton](MenuStyle/borderlessButton.zh-Hans.md) to construct this style.

## Creating a bordeless button menu style

- **init()**: Creates a borderless button menu style.
- **init(showsMenuIndicator:)**: Creates a borderless button menu style, specifying whether to show a visual menu indicator.

## Supporting types

- **DefaultMenuStyle**: The default menu style, based on the menu’s context.
- **ButtonMenuStyle**: A menu style that displays a button that toggles the display of the menu’s contents when pressed.
- **BorderedButtonMenuStyle**: A menu style that displays a bordered button that toggles the display of the menu’s contents when pressed.
