# automatic

**Type Property**

The automatic repeat behavior.

## Declaration

```swift
static let automatic: ButtonRepeatBehavior
```

## Getting repeat behaviors

- **enabled**: Repeating button actions will be enabled.
- **disabled**: Repeating button actions will be disabled.
