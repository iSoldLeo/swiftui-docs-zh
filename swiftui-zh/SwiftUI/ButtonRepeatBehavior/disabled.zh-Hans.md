# disabled

**Type Property**

Repeating button actions will be disabled.

## Declaration

```swift
static let disabled: ButtonRepeatBehavior
```

## Getting repeat behaviors

- **automatic**: The automatic repeat behavior.
- **enabled**: Repeating button actions will be enabled.
