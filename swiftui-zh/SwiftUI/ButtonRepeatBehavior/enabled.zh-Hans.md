# enabled

**Type Property**

Repeating button actions will be enabled.

## Declaration

```swift
static let enabled: ButtonRepeatBehavior
```

## Getting repeat behaviors

- **automatic**: The automatic repeat behavior.
- **disabled**: Repeating button actions will be disabled.
