# CarouselTabViewStyle

**Structure**

实现轮播交互和外观的样式。

## 声明

```swift
struct CarouselTabViewStyle
```

## 创建标签视图样式

- **init()**：创建轮播表格视图样式。

## 支持的类型

- **DefaultTabViewStyle**：默认标签视图样式。

- **SidebarAdaptableTabViewStyle**：适应不同平台的标签栏样式。

- **TabBarOnlyTabViewStyle**：尽可能显示标签栏的标签视图样式。

- **GroupedTabViewStyle**：一种标签视图样式，显示一个标签栏，将标签分组在一起。

- **PageTabViewStyle**：一种 `TabViewStyle`，显示分页滚动 `TabView`。

- **VerticalPageTabViewStyle**：一种 `TabViewStyle`，显示垂直 `TabView` 交互和外观。

## 符合以下规范

- TabViewStyle

# CarouselTabViewStyle

**Structure**

A style that implements the carousel interaction and appearance.

## Declaration

```swift
struct CarouselTabViewStyle
```

## Creating the tab view style

- **init()**: Creates a carousel table view style.

## Supporting types

- **DefaultTabViewStyle**: The default tab view style.
- **SidebarAdaptableTabViewStyle**: A tab bar style that adapts to each platform.
- **TabBarOnlyTabViewStyle**: A tab view style that displays a tab bar when possible.
- **GroupedTabViewStyle**: A tab view style that displays a tab bar that groups its tabs together.
- **PageTabViewStyle**: A `TabViewStyle` that displays a paged scrolling `TabView`.
- **VerticalPageTabViewStyle**: A `TabViewStyle` that displays a vertical `TabView` interaction and appearance.

## Conforms To

- TabViewStyle
