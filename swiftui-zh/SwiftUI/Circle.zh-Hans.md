# 圆

**Structure**

以包含它的视图的框架为中心绘制的圆。

## 声明

```swift
@frozen struct Circle
```

## 概述

圆的半径等于框架矩形最短边长的一半。

## 创建圆

- **init()**：创建一个新的圆形。

## 创建圆形

- **Ellipse**：在包含它的视图的框架内绘制的椭圆。

- **Capsule**：在包含它的视图的框架内绘制的胶囊形。

## 符合以下类型

- 可动画化

- 可按位复制

- 图表符号形状

- 可复制

- 可插入形状

- 圆角矩形形状

- 可发送

- 可发送元类型

- 形状

# Circle

**Structure**

A circle centered on the frame of the view containing it.

## Declaration

```swift
@frozen struct Circle
```

## Overview

The circle’s radius equals half the length of the frame rectangle’s smallest edge.

## Creating a circle

- **init()**: Creates a new circle shape.

## Creating circular shapes

- **Ellipse**: An ellipse aligned inside the frame of the view containing it.
- **Capsule**: A capsule shape aligned inside the frame of the view containing it.
