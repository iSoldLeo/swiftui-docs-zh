# Color.RGBColorSpace.displayP3

**Case**

The Display P3 color space.

## Declaration

```swift
case displayP3
```

## Discussion

This color space uses the Digital Cinema Initiatives - Protocol 3 (DCI-P3) primary colors, a D65 white point, and the [sRGB](sRGB.zh-Hans.md) transfer function.

## Getting color spaces

- **Color.RGBColorSpace.sRGB**: The extended red, green, blue (sRGB) color space.
- **Color.RGBColorSpace.sRGBLinear**: The extended sRGB color space with a linear transfer function.
