# 不透明度

**实例属性**

颜色的不透明度，取值范围为 `0` 至 `1`。

## 声明

```swift
var opacity: Float
```

## 讨论

`0`的值表示 100% 透明度，而`1`的值表示 100% 不透明度。

# opacity

**Instance Property**

The degree of opacity in the color, given in the range `0` to `1`.

## Declaration

```swift
var opacity: Float
```

## Discussion

A value of `0` means 100% transparency, while a value of `1` means 100% opacity.
