# description

**Instance Property**

A textual representation of the color.

## Declaration

```swift
var description: String { get }
```

## Discussion

Use this method to get a string that represents the color. The [doc://com.apple.documentation/documentation/Swift/print(_:separator:terminator:)] function uses this property to get a string representing an instance:

```swift
print(Color.red)
// Prints "red"
```
