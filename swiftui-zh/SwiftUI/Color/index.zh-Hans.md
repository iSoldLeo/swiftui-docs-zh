# ==(_:_:)

**Operator**

表示两种颜色是否相等。

## 声明

```swift
static func == (lhs: Color, rhs: Color) -> Bool
```

## 参数

- **lhs**：要比较的第一种颜色。
- **rhs**：要比较的第二种颜色。

## 返回值

布尔值，如果两种颜色相等，则设置为 `true`。

## 比较颜色

- **hash(into:)**：将颜色的基本成分输入给定的哈希函数，对其进行哈希处理。

# ==(_:_:)

**Operator**

Indicates whether two colors are equal.

## Declaration

```swift
static func == (lhs: Color, rhs: Color) -> Bool
```

## Parameters

- **lhs**: The first color to compare.
- **rhs**: The second color to compare.

## Return Value

A Boolean that’s set to `true` if the two colors are equal.

## Comparing colors

- **hash(into:)**: Hashes the essential components of the color by feeding them into the given hash function.
