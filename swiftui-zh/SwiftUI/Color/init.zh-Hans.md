# init(_:)

**Initializer**

创建一个常量颜色，其值由解析颜色指定。

## 声明

```swift
init(_ resolved: Color.Resolved)
```

## 创建颜色

- **init(_:bundle:)**：从您指定名称的颜色集中创建颜色。
- **resolve(in:)**：根据当前`context`，将此颜色评估为已解析的颜色。

# init(_:)

**Initializer**

Creates a constant color with the values specified by the resolved color.

## Declaration

```swift
init(_ resolved: Color.Resolved)
```

## Creating a color

- **init(_:bundle:)**: Creates a color from a color set that you indicate by name.
- **resolve(in:)**: Evaluates this color to a resolved color given the current `context`.
