# init(cgColor:)

**Initializer**

从 Core Graphics 颜色创建颜色。

## 声明

```swift
init(cgColor: CGColor)
```

## 从另一种颜色创建一种颜色

- **init(uiColor:)**：从 UIKit 颜色创建颜色。
- **init(nsColor:)**：从 AppKit 颜色创建颜色。

# init(cgColor:)

**Initializer**

Creates a color from a Core Graphics color.

## Declaration

```swift
init(cgColor: CGColor)
```

## Creating a color from another color

- **init(uiColor:)**: Creates a color from a UIKit color.
- **init(nsColor:)**: Creates a color from an AppKit color.
