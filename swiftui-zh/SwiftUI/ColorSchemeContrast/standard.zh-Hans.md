# ColorSchemeContrast.standard

**Case**

SwiftUI displays views with standard contrast between the app’s foreground and background colors.

## Declaration

```swift
case standard
```

## Getting contrast options

- **ColorSchemeContrast.increased**: SwiftUI displays views with increased contrast between the app’s foreground and background colors.
