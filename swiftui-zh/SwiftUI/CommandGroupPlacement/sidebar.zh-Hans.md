# 侧边栏

**类型属性**

用于控制应用程序侧边栏和全屏模式的命令的位置。

## 声明

```swift
static let sidebar: CommandGroupPlacement
```

## 讨论

默认情况下，该组在 macOS 中包括以下命令：

- 显示/隐藏侧边栏
- 进入/退出全屏

## 栏

- **toolbar**：工具栏操作命令的位置。

# sidebar

**Type Property**

Placement for commands that control the app’s sidebar and full-screen modes.

## Declaration

```swift
static let sidebar: CommandGroupPlacement
```

## Discussion

By default, this group includes the following commands in macOS:

- Show/Hide Sidebar
- Enter/Exit Full Screen

## Bars

- **toolbar**: Placement for commands that manipulate the toolbar.
