# 工具栏

**类型属性**

工具栏操作命令的位置。

## 声明

```swift
static let toolbar: CommandGroupPlacement
```

## 讨论

默认情况下，该组在 macOS 中包括以下命令：

- 显示/隐藏工具栏
- 自定义工具栏

## 栏

- **sidebar**：控制应用程序侧边栏和全屏模式的命令位置。

# toolbar

**Type Property**

Placement for commands that manipulate the toolbar.

## Declaration

```swift
static let toolbar: CommandGroupPlacement
```

## Discussion

By default, this group includes the following commands in macOS:

- Show/Hide Toolbar
- Customize Toolbar

## Bars

- **sidebar**: Placement for commands that control the app’s sidebar and full-screen modes.
