```swift

```

- **init()**：创建紧凑菜单控件组样式。

## 支持的类型

- **AutomaticControlGroupStyle**：默认控件组样式。

- **MenuControlGroupStyle**：一种控件组样式，当用户按下控件时，其内容会以菜单形式呈现；当嵌套在更大的菜单中时，则会以子菜单形式呈现。

- **NavigationControlGroupStyle**：导航控件组样式。

- **PaletteControlGroupStyle**：一种控件组样式，其内容会以调色板形式呈现。

# CompactMenuControlGroupStyle

**Structure**

A control group style that presents its content as a compact menu when the user presses the control, or as a submenu when nested within a larger menu.

## Declaration

```swift
struct CompactMenuControlGroupStyle
```

## Overview

Use [compactMenu](ControlGroupStyle/compactMenu.zh-Hans.md) to construct this style.

## Creating the control group style

- **init()**: Creates a compact menu control group style.

## Supporting types

- **AutomaticControlGroupStyle**: The default control group style.
- **MenuControlGroupStyle**: A control group style that presents its content as a menu when the user presses the control, or as a submenu when nested within a larger menu.
- **NavigationControlGroupStyle**: The navigation control group style.
- **PaletteControlGroupStyle**: A control group style that presents its content as a palette.

## Conforms To

- ControlGroupStyle
