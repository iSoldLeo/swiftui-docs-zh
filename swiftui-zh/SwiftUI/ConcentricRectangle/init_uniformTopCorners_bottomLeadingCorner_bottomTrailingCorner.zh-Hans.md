# init(uniformTopCorners:bottomLeadingCorner:bottomTrailingCorner:)

**Initializer**

创建一个矩形，其顶部两个角统一应用一种角样式，底部两个角分别应用两种不同的角样式。

## 声明

```swift
init(uniformTopCorners: Edge.Corner.Style = .concentric, bottomLeadingCorner: Edge.Corner.Style = .concentric, bottomTrailingCorner: Edge.Corner.Style = .concentric)
```

## 参数

- **uniformTopCorners**：应用于顶部两个角的统一角样式。

- **bottomLeadingCorner**：底部前导角的角样式。

- **bottomTrailingCorner**：底部尾角的样式。

# init(uniformTopCorners:bottomLeadingCorner:bottomTrailingCorner:)

**Initializer**

Create a rectangle with a corner style set on the top two corners uniformly, and two other styles for the bottom two corners respectively.

## Declaration

```swift
init(uniformTopCorners: Edge.Corner.Style = .concentric, bottomLeadingCorner: Edge.Corner.Style = .concentric, bottomTrailingCorner: Edge.Corner.Style = .concentric)
```

## Parameters

- **uniformTopCorners**: The corner style to be applied to the top two corners uniformly.
- **bottomLeadingCorner**: The corner style for the bottom leading corner.
- **bottomTrailingCorner**: The corner style for the bottom trailing corner.
