# init(uniformTrailingCorners:topLeadingCorner:bottomLeadingCorner:)

**Initializer**

创建一个矩形，其后两个角统一应用一种角样式，前两个角分别应用两种不同的角样式。

## 声明

```swift
init(uniformTrailingCorners: Edge.Corner.Style = .concentric, topLeadingCorner: Edge.Corner.Style = .concentric, bottomLeadingCorner: Edge.Corner.Style = .concentric)
```

## 参数

- **uniformTrailingCorners**：统一应用于后两个角的角样式。

- **topLeadingCorner**：应用于顶部前角的角样式。

- **bottomLeadingCorner**：底部前角的角样式。

# init(uniformTrailingCorners:topLeadingCorner:bottomLeadingCorner:)

**Initializer**

Create a rectangle with a corner style set on the trailing two corners uniformly, and two other styles for the leading two corners respectively.

## Declaration

```swift
init(uniformTrailingCorners: Edge.Corner.Style = .concentric, topLeadingCorner: Edge.Corner.Style = .concentric, bottomLeadingCorner: Edge.Corner.Style = .concentric)
```

## Parameters

- **uniformTrailingCorners**: The corner style to be applied to the trailing two corners uniformly.
- **topLeadingCorner**: The corner style for the top leading corner.
- **bottomLeadingCorner**: The corner style for the bottom leading corner.
