# 导航分割视图

**类型属性**

[NavigationSplitView](../NavigationSplitView.zh-Hans.md)内容后面的背景位置。

## 声明

```swift
static let navigationSplitView: ContainerBackgroundPlacement
```

## 讨论

在使用`ContentBackgroundPlacement/navigation` 放置的半透明列后面，可以看到此背景。对于展开的分割视图，该容器背景也可以作为分隔线颜色在列之间显示。

# navigationSplitView

**Type Property**

A background placement behind the content of a [NavigationSplitView](../NavigationSplitView.zh-Hans.md).

## Declaration

```swift
static let navigationSplitView: ContainerBackgroundPlacement
```

## Discussion

This is visible behind columns that are made translucent using the `ContentBackgroundPlacement/navigation` placement. For expanded split views, this container background is also visible between the columns as the divider color.
