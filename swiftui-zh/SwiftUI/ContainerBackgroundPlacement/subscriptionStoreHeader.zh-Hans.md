# subscriptionStoreHeader

**类型属性**

订阅商店视图营销内容后面的背景位置。

## 声明

```swift
static var subscriptionStoreHeader: ContainerBackgroundPlacement { get }
```

## 获取 StoreKit 放置位置

- **subscriptionStore**：根据视图的上下文，在订阅商店视图中自动放置。
- **subscriptionStoreFullHeight**：跨越订阅存储视图整个高度的背景位置。

# subscriptionStoreHeader

**Type Property**

A background placement behind the marketing content of a subscription store view.

## Declaration

```swift
static var subscriptionStoreHeader: ContainerBackgroundPlacement { get }
```

## Getting StoreKit placements

- **subscriptionStore**: An automatic placement within a subscription store view, based on the view’s context.
- **subscriptionStoreFullHeight**: A background placement that spans the full height of a subscription store view.
