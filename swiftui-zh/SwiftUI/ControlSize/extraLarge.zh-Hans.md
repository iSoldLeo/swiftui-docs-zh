# ControlSize.extraLarge

## 获取控件尺寸

- **ControlSize.mini**：尺寸最小的控件版本。

- **ControlSize.small**：尺寸按比例缩小的控件版本，适用于空间受限的视图。

- **ControlSize.regular**：默认尺寸的控件版本。

- **ControlSize.large**：一个尺寸醒目的对照版本。

# ControlSize.extraLarge

**Case**

A control version that is substantially sized. The largest control size. Resolves to [large](large.zh-Hans.md) on platforms other than visionOS.

## Declaration

```swift
case extraLarge
```

## Getting control sizes

- **ControlSize.mini**: A control version that is minimally sized.
- **ControlSize.small**: A control version that is proportionally smaller size for space-constrained views.
- **ControlSize.regular**: A control version that is the default size.
- **ControlSize.large**: A control version that is prominently sized.
