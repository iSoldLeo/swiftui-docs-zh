# ControlSize.large

**Case**

一个尺寸较大的控件版本。

## 声明

```swift
case large
```

## 获取控件尺寸

- **ControlSize.mini**：一个尺寸最小的控件版本。

- **ControlSize.small**：一个尺寸按比例缩小的控件版本，适用于空间受限的视图。

- **ControlSize.regular**：一个默认尺寸的控件版本。

- **ControlSize.extraLarge**：一个尺寸较大的控件版本。最大尺寸的控件。在除 visionOS 以外的平台上，解析为 [large](large.zh-Hans.md)。

# ControlSize.large

**Case**

A control version that is prominently sized.

## Declaration

```swift
case large
```

## Getting control sizes

- **ControlSize.mini**: A control version that is minimally sized.
- **ControlSize.small**: A control version that is proportionally smaller size for space-constrained views.
- **ControlSize.regular**: A control version that is the default size.
- **ControlSize.extraLarge**: A control version that is substantially sized. The largest control size. Resolves to [large](large.zh-Hans.md) on platforms other than visionOS.
