# ControlSize.mini

**Case**

最小尺寸的控件版本。

## 声明

```swift
case mini
```

## 获取控件尺寸

- **ControlSize.small**：适用于空间受限视图的按比例缩小的控件版本。

- **ControlSize.regular**：默认尺寸的控件版本。

- **ControlSize.large**：醒目的控件版本。

- **ControlSize.extraLarge**：尺寸极大的控件版本。最大尺寸的控件。在除 visionOS 以外的平台上，解析为 [large](large.zh-Hans.md)。

# ControlSize.mini

**Case**

A control version that is minimally sized.

## Declaration

```swift
case mini
```

## Getting control sizes

- **ControlSize.small**: A control version that is proportionally smaller size for space-constrained views.
- **ControlSize.regular**: A control version that is the default size.
- **ControlSize.large**: A control version that is prominently sized.
- **ControlSize.extraLarge**: A control version that is substantially sized. The largest control size. Resolves to [large](large.zh-Hans.md) on platforms other than visionOS.
