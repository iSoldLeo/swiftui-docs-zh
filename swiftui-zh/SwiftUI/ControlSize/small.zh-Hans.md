# ControlSize.small

**Case**

适用于空间受限视图的按比例缩小的控件版本。

## 声明

```swift
case small
```

## 获取控件大小

- **ControlSize.mini**：最小尺寸的控件版本。

- **ControlSize.regular**：默认尺寸的控件版本。

- **ControlSize.large**：醒目尺寸的控件版本。

- **ControlSize.extraLarge**：最大尺寸的控件版本。在除 visionOS 以外的平台上，解析为 [large](large.zh-Hans.md)。

# ControlSize.small

**Case**

A control version that is proportionally smaller size for space-constrained views.

## Declaration

```swift
case small
```

## Getting control sizes

- **ControlSize.mini**: A control version that is minimally sized.
- **ControlSize.regular**: A control version that is the default size.
- **ControlSize.large**: A control version that is prominently sized.
- **ControlSize.extraLarge**: A control version that is substantially sized. The largest control size. Resolves to [large](large.zh-Hans.md) on platforms other than visionOS.
