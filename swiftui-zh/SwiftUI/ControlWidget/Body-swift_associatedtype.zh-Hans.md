# Body

**Associated Type**

The type of configuration representing the content of this control.

## Declaration

```swift
associatedtype Body : ControlWidgetConfiguration
```

## Discussion

When you create a custom control, Swift infers this type from your implementation of the required [body-swift.property](body-swift_property.zh-Hans.md) property.
