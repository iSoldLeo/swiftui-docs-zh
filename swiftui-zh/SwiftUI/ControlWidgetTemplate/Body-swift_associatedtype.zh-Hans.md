# Body

**Associated Type**

The type of control widget template representing the body of this template.

## Declaration

```swift
associatedtype Body : ControlWidgetTemplate
```

## Discussion

When you create a custom control widget, Swift infers this type from your implementation of the required [body-swift.property](body-swift_property.zh-Hans.md) property.
