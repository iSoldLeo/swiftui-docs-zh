# defaultCustomization()

**Instance Method**

Configures customizable toolbar content with the default visibility and options.

## Declaration

```swift
func defaultCustomization() -> some CustomizableToolbarContent

```

## Discussion

Use the [defaultCustomization(_:options:)](defaultCustomization___options.zh-Hans.md) modifier providing either a `defaultVisibility` or `options` instead.

## Using default options

- **defaultCustomization(_:options:)**: Configures the way customizable toolbar items with the default behavior behave.
