# 日期

**类型属性**

根据地区显示日期、月份和年份

## 声明

```swift
static let date: DatePickerComponents
```

## 获取日期选择器组件

- **hourAndMinute**：根据地域显示小时和分钟组件
- **hourMinuteAndSecond**：显示基于本地语言的小时、分钟和秒分量

# date

**Type Property**

Displays day, month, and year based on the locale

## Declaration

```swift
static let date: DatePickerComponents
```

## Getting date picker components

- **hourAndMinute**: Displays hour and minute components based on the locale
- **hourMinuteAndSecond**: Displays hour, minute and second components based on the locale
