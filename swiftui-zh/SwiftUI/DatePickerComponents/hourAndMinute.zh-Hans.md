# 小时和分钟

**类型属性**

根据地区显示小时和分钟的分量

## 声明

```swift
static let hourAndMinute: DatePickerComponents
```

## 获取日期选择器组件

- **date**：根据地域显示日期、月份和年份
- **hourMinuteAndSecond**：根据地区显示小时、分钟和秒的组成部分

# hourAndMinute

**Type Property**

Displays hour and minute components based on the locale

## Declaration

```swift
static let hourAndMinute: DatePickerComponents
```

## Getting date picker components

- **date**: Displays day, month, and year based on the locale
- **hourMinuteAndSecond**: Displays hour, minute and second components based on the locale
