# 小时分钟秒

**类型属性**

显示基于本地语言的小时、分钟和秒组件

## 声明

```swift
static let hourMinuteAndSecond: DatePickerComponents
```

## 获取日期选择器组件

- **date**：根据地域显示日期、月份和年份
- **hourAndMinute**：根据地区显示小时和分钟组件

# hourMinuteAndSecond

**Type Property**

Displays hour, minute and second components based on the locale

## Declaration

```swift
static let hourMinuteAndSecond: DatePickerComponents
```

## Getting date picker components

- **date**: Displays day, month, and year based on the locale
- **hourAndMinute**: Displays hour and minute components based on the locale
