# DefaultDatePickerStyle

**Structure**

日期选择器的默认样式。

## 声明

```swift
@MainActor @preconcurrency struct DefaultDatePickerStyle
```

## 概述

您也可以使用 [automatic](DatePickerStyle/automatic.zh-Hans.md) 来构建此样式。

## 创建日期选择器样式

- **init()**：创建默认日期选择器样式的实例。

## 支持的类型

- **CompactDatePickerStyle**：以紧凑的文本格式显示组件的日期选择器样式。

- **FieldDatePickerStyle**：一种日期选择器样式，将各个组件显示在可编辑字段中。

- **GraphicalDatePickerStyle**：一种日期选择器样式，显示交互式日历或时钟。

- **StepperFieldDatePickerStyle**：一种系统样式，将各个组件显示在可编辑字段中，并带有可递增/递减所选组件的步进器。

- **WheelDatePickerStyle**：一种日期选择器样式，将每个组件显示为可滚动轮盘中的列。

## 符合以下样式

- DatePickerStyle

# DefaultDatePickerStyle

**Structure**

The default style for date pickers.

## Declaration

```swift
@MainActor @preconcurrency struct DefaultDatePickerStyle
```

## Overview

You can also use [automatic](DatePickerStyle/automatic.zh-Hans.md) to construct this style.

## Creating the date picker style

- **init()**: Creates an instance of the default date picker style.

## Supporting types

- **CompactDatePickerStyle**: A date picker style that displays the components in a compact, textual format.
- **FieldDatePickerStyle**: A date picker style that displays the components in an editable field.
- **GraphicalDatePickerStyle**: A date picker style that displays an interactive calendar or clock.
- **StepperFieldDatePickerStyle**: A system style that displays the components in an editable field, with adjoining stepper that can increment/decrement the selected component.
- **WheelDatePickerStyle**: A date picker style that displays each component as columns in a scrollable wheel.

## Conforms To

- DatePickerStyle
