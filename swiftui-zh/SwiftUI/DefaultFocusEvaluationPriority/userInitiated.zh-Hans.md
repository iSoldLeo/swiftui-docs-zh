# 用户发起

**类型属性**

当焦点移动到受影响的分支时，始终使用默认焦点首选项。

## 声明

```swift
static let userInitiated: DefaultFocusEvaluationPriority
```

## 获取优先级

- **automatic**：当焦点自动移动到受影响的分支时，使用默认焦点首选项，但当移动是由用户启动的导航命令驱动时，则忽略默认焦点首选项。

# userInitiated

**Type Property**

Always use the default focus preference when focus moves into the affected branch.

## Declaration

```swift
static let userInitiated: DefaultFocusEvaluationPriority
```

## Getting the priorities

- **automatic**: Use the default focus preference when focus moves into the affected branch automatically, but ignore it when the movement is driven by a user-initiated navigation command.
