**Structure**

## 声明


## 概述

您也可以使用 [automatic](LabelStyle/automatic.zh-Hans.md) 来构造此样式。

- **init()**：创建自动标签样式。

## 支持的类型

- **IconOnlyLabelStyle**：仅显示标签图标的标签样式。

- **TitleAndIconLabelStyle**：一种标签样式，使用系统标准布局同时显示标签标题和图标。

- **TitleOnlyLabelStyle**：一种标签样式，仅显示标签标题。

# DefaultLabelStyle

**Structure**

The default label style in the current context.

## Declaration

```swift
struct DefaultLabelStyle
```

## Overview

You can also use [automatic](LabelStyle/automatic.zh-Hans.md) to construct this style.

## Creating the label style

- **init()**: Creates an automatic label style.

## Supporting types

- **IconOnlyLabelStyle**: A label style that only displays the icon of the label.
- **TitleAndIconLabelStyle**: A label style that shows both the title and icon of the label using a system-standard layout.
- **TitleOnlyLabelStyle**: A label style that only displays the title of the label.

## Conforms To

- LabelStyle
