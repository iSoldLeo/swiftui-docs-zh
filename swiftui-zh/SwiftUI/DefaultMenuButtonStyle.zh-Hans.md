# DefaultMenuButtonStyle

**Structure**

The default menu button style.

## Declaration

```swift
struct DefaultMenuButtonStyle
```

## Creating a default menu button style

- **init()**

## Supporting types

- **BorderlessButtonMenuButtonStyle**: A menu button style which manifests as a borderless button with no visual embelishments.
- **BorderlessPullDownMenuButtonStyle**: A menu button style which manifests as a borderless pull-down button.
- **PullDownMenuButtonStyle**: A menu button style which manifests as a pull-down button.

## Conforms To

- MenuButtonStyle
