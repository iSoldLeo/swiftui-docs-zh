# DefaultMenuStyle

**Structure**

基于菜单上下文的默认菜单样式。

## 声明

```swift
struct DefaultMenuStyle
```

## 概述

您也可以使用 [automatic](MenuStyle/automatic.zh-Hans.md) 来构建此样式。

## 创建菜单样式

- **init()**：创建默认菜单样式。

## 支持的类型

- **ButtonMenuStyle**：一种菜单样式，按下按钮可切换菜单内容的显示。

- **BorderlessButtonMenuStyle**：一种菜单样式，显示一个无边框按钮，按下该按钮可切换菜单内容的显示。

- **BorderedButtonMenuStyle**：一种菜单样式，显示一个有边框按钮，按下该按钮可切换菜单内容的显示。

## 符合以下样式：

# DefaultMenuStyle

**Structure**

The default menu style, based on the menu’s context.

## Declaration

```swift
struct DefaultMenuStyle
```

## Overview

You can also use [automatic](MenuStyle/automatic.zh-Hans.md) to construct this style.

## Creating the menu style

- **init()**: Creates a default menu style.

## Supporting types

- **ButtonMenuStyle**: A menu style that displays a button that toggles the display of the menu’s contents when pressed.
- **BorderlessButtonMenuStyle**: A menu style that displays a borderless button that toggles the display of the menu’s contents when pressed.
- **BorderedButtonMenuStyle**: A menu style that displays a bordered button that toggles the display of the menu’s contents when pressed.

## Conforms To

- MenuStyle
