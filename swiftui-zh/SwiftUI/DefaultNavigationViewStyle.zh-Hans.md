# DefaultNavigationViewStyle

**Structure**

默认导航视图样式。

## 声明

```swift
struct DefaultNavigationViewStyle
```

## 概述

使用 [automatic](NavigationViewStyle/automatic.zh-Hans.md) 构造此样式。

## 创建默认导航视图样式

- **init()**：创建默认导航视图样式。

## 支持的类型

- **ColumnNavigationViewStyle**：由一系列列视图表示的导航视图样式。

- **StackNavigationViewStyle**：一种导航视图样式，由视图堆栈表示，每次只显示一个顶层视图。

- **DoubleColumnNavigationViewStyle**：一种导航视图样式，由主视图堆栈表示，可导航至详细视图。

## 符合以下规范

- NavigationViewStyle

# DefaultNavigationViewStyle

**Structure**

The default navigation view style.

## Declaration

```swift
struct DefaultNavigationViewStyle
```

## Overview

Use [automatic](NavigationViewStyle/automatic.zh-Hans.md) to construct this style.

## Creating a default navigation view style

- **init()**: Creates the default navigation view style.

## Supporting types

- **ColumnNavigationViewStyle**: A navigation view style represented by a series of views in columns.
- **StackNavigationViewStyle**: A navigation view style represented by a view stack that only shows a single top view at a time.
- **DoubleColumnNavigationViewStyle**: A navigation view style represented by a primary view stack that navigates to a detail view.

## Conforms To

- NavigationViewStyle
