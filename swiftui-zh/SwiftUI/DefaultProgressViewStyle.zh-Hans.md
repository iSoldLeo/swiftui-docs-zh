# DefaultProgressViewStyle

**Structure**

The default progress view style in the current context of the view being styled.

## Declaration

```swift
struct DefaultProgressViewStyle
```

## Overview

Use [automatic](ProgressViewStyle/automatic.zh-Hans.md) to construct this style.

## Creating the progress view style

- **init()**: Creates a default progress view style.

## Supporting types

- **CircularProgressViewStyle**: A progress view that uses a circular gauge to indicate the partial completion of an activity.
- **LinearProgressViewStyle**: A progress view that visually indicates its progress using a horizontal bar.

## Conforms To

- ProgressViewStyle
