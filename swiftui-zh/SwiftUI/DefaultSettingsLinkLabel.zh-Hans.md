# DefaultSettingsLinkLabel

**Structure**

The default label to use for a settings link.

## Declaration

```swift
struct DefaultSettingsLinkLabel
```

## Overview

You don’t use this type directly. Instead, the system creates it automatically when you construct a [SettingsLink](SettingsLink.zh-Hans.md) with the default label.
