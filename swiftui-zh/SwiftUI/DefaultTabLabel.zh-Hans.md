# DefaultTabLabel

**Structure**

The default label to use for a tab or tab section.

## Declaration

```swift
struct DefaultTabLabel
```

## Overview

You don’t use this type directly. Instead, the system creates it automatically when you construct a `Tab` or `TabSection`.
