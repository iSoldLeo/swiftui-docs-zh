# 默认窗口工具栏样式

**Structure**

默认窗口工具栏样式。

## 声明

```swift
struct DefaultWindowToolbarStyle
```

## 概览

您也可以使用 [automatic](WindowToolbarStyle/automatic.zh-Hans.md) 构建这种样式。

## 创建窗口工具栏样式

- **init()**：创建默认的窗口工具栏样式。

## 支持类型

- **ExpandedWindowToolbarStyle**：在工具栏上方显示标题栏区域的窗口工具栏样式。
- **UnifiedWindowToolbarStyle**：一种窗口工具栏样式，它以内联方式显示工具栏和标题栏。
- **UnifiedCompactWindowToolbarStyle**：与[unified](WindowToolbarStyle/unified.zh-Hans.md) 相似的窗口工具栏样式，但垂直尺寸更为紧凑。

# DefaultWindowToolbarStyle

**Structure**

The default window toolbar style.

## Declaration

```swift
struct DefaultWindowToolbarStyle
```

## Overview

You can also use [automatic](WindowToolbarStyle/automatic.zh-Hans.md) to construct this style.

## Creating the window toolbar style

- **init()**: Creates a default window toolbar style.

## Supporting types

- **ExpandedWindowToolbarStyle**: A window toolbar style which displays its title bar area above the toolbar.
- **UnifiedWindowToolbarStyle**: A window toolbar style which displays its toolbar and title bar inline.
- **UnifiedCompactWindowToolbarStyle**: A window toolbar style similar to [unified](WindowToolbarStyle/unified.zh-Hans.md), but with a more compact vertical sizing.

## Conforms To

- WindowToolbarStyle
