# DisabledTextSelectability

**Structure**

A selectability type that disables text selection by the person using your app.

## Declaration

```swift
struct DisabledTextSelectability
```

## Overview

Don’t use this type directly. Instead, use [disabled](TextSelectability/disabled.zh-Hans.md).

## Supporting types

- **EnabledTextSelectability**: A selectability type that enables text selection by the person using your app.

## Conforms To

- TextSelectability
