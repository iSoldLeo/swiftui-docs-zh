# fileURL

**Instance Property**

A URL of an open document.

## Declaration

```swift
var fileURL: URL? { get }
```

## Discussion

If the document has never been saved, returns `nil`.

## Getting configuration values

- **isEditable**: A Boolean value that indicates whether you can edit the document.
