# DoubleColumnNavigationViewStyle

**Structure**

导航视图样式，由主视图堆栈表示，可导航到详细视图。

## 声明

```swift
struct DoubleColumnNavigationViewStyle
```

## 创建双列视图样式

- **init()**：创建双列导航视图样式。

## 支持类型

- **DefaultNavigationViewStyle**：默认导航视图样式。
- **ColumnNavigationViewStyle**：由一系列列视图表示的导航视图样式。
- **StackNavigationViewStyle**：由视图堆栈表示的导航视图样式，一次只显示一个顶部视图。

## 符合

- 导航视图样式

# DoubleColumnNavigationViewStyle

**Structure**

A navigation view style represented by a primary view stack that navigates to a detail view.

## Declaration

```swift
struct DoubleColumnNavigationViewStyle
```

## Create a double column view style

- **init()**: Creates a double column navigation view style.

## Supporting types

- **DefaultNavigationViewStyle**: The default navigation view style.
- **ColumnNavigationViewStyle**: A navigation view style represented by a series of views in columns.
- **StackNavigationViewStyle**: A navigation view style represented by a view stack that only shows a single top view at a time.

## Conforms To

- NavigationViewStyle
