# DropConfiguration

**Structure**

Describes the behavior of the drop.

## Declaration

```swift
struct DropConfiguration
```

## Initializers

- **init(operation:)**: Creates a configuration value with the operation chosen by the drop destination.

## Instance Properties

- **acceptedItemCount**: Specifies the number of items that the drop side wants to accept.
- **operation**: The drop operation that the drop chooses to perform.

## Configuring drag and drop behavior

- **DragConfiguration**: The behavior of the drag, proposed by the dragging source.
