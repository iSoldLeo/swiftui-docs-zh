# DropOperation.forbidden

**Case**

在此时间或地点不允许进行投放活动。

## 声明

```swift
case forbidden
```

## 获取操作类型

- **DropOperation.cancel**：取消拖动操作，不传输数据。
- **DropOperation.copy**：将数据复制到修改后的视图中。
- **DropOperation.move**：移动拖动项所代表的数据，而不是复制数据。

# DropOperation.forbidden

**Case**

The drop activity is not allowed at this time or location.

## Declaration

```swift
case forbidden
```

## Getting operation types

- **DropOperation.cancel**: Cancel the drag operation and transfer no data.
- **DropOperation.copy**: Copy the data to the modified view.
- **DropOperation.move**: Move the data represented by the drag items instead of copying it.
