# update()

**Instance Method**

Updates the underlying value of the stored value.

## Declaration

```swift
mutating func update()
```

## Discussion

SwiftUI calls this function before rendering a view’s [body-8kl5o](../View/body-8kl5o.zh-Hans.md) to ensure the view has the most recent value.
