# init(floatLiteral:)

**Initializer**

Create a fixed corner style with a float value as its corner radius. This is a shorthand of writing `.fixed(value)`.

## Declaration

```swift
init(floatLiteral value: Edge.Corner.Style.FloatLiteralType)
```

## Parameters

- **value**: The fixed corner radius.
