An edit action that allows the user to delete one or more elements of a collection.

## Declaration

```swift
static var delete: EditActions<Data> { get }
```

- **all**: All the edit actions available on this collection.
- **all**: All the edit actions available on this collection.
- **all**: All the edit actions available on this collection.
