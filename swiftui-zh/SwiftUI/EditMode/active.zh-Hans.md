# EditMode.active

**Case**

The user can edit the view content.

## Declaration

```swift
case active
```

## Discussion

The [isEditing](isEditing.zh-Hans.md) property is `true` in this state.

## Getting edit modes

- **EditMode.inactive**: The user can’t edit the view content.
- **EditMode.transient**: The view is in a temporary edit mode.
