# EditMode.inactive

**Case**

The user can’t edit the view content.

## Declaration

```swift
case inactive
```

## Discussion

The [isEditing](isEditing.zh-Hans.md) property is `false` in this state.

## Getting edit modes

- **EditMode.active**: The user can edit the view content.
- **EditMode.transient**: The view is in a temporary edit mode.
