# Ellipse

**Structure**

An ellipse aligned inside the frame of the view containing it.

## Declaration

```swift
@frozen struct Ellipse
```

## Creating an ellipse

- **init()**: Creates a new ellipse shape.

## Creating circular shapes

- **Circle**: A circle centered on the frame of the view containing it.
- **Capsule**: A capsule shape aligned inside the frame of the view containing it.

## Conforms To

- Animatable
- BitwiseCopyable
- Copyable
- InsettableShape
- Sendable
- SendableMetatype
- Shape
- View
