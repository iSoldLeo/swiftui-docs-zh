# EnabledTextSelectability

**Structure**

A selectability type that enables text selection by the person using your app.

## Declaration

```swift
struct EnabledTextSelectability
```

## Overview

Don’t use this type directly. Instead, use [enabled](TextSelectability/enabled.zh-Hans.md).

## Supporting types

- **DisabledTextSelectability**: A selectability type that disables text selection by the person using your app.

## Conforms To

- TextSelectability
