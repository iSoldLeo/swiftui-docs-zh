# 无障碍显示按钮形状

**实例属性**

是否启用了显示按钮形状的系统偏好设置。

## 声明

```swift
var accessibilityShowButtonShapes: Bool { get }
```

## 讨论

如果此属性的值为 true，则按钮等交互式自定义控件的绘制方式应使其边缘和边界清晰可见。

## 提高可读性

- **accessibilityReduceTransparency**：是否启用了 "降低透明度 "系统偏好设置。
- **legibilityWeight**：应用于文本的字体权重。
- **LegibilityWeight**：辅助功能粗体文字用户设置选项。

# accessibilityShowButtonShapes

**Instance Property**

Whether the system preference for Show Button Shapes is enabled.

## Declaration

```swift
var accessibilityShowButtonShapes: Bool { get }
```

## Discussion

If this property’s value is true, interactive custom controls such as buttons should be drawn in such a way that their edges and borders are clearly visible.

## Improving legibility

- **accessibilityReduceTransparency**: Whether the system preference for Reduce Transparency is enabled.
- **legibilityWeight**: The font weight to apply to text.
- **LegibilityWeight**: The Accessibility Bold Text user setting options.
