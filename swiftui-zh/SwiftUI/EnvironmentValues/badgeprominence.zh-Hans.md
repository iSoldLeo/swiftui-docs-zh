# badgeProminence

**实例属性**

适用于与此环境相关的徽章的突出显示。

## 声明

```swift
var badgeProminence: BadgeProminence { get set }
```

## 讨论

默认值为 [standard](../BadgeProminence/standard.zh-Hans.md)。

## 在列表项上显示徽章

- **badge(_:)**：根据整数值为视图生成徽章。
- **badgeProminence(_:)**：指定此视图创建的徽章的显著性。
- **BadgeProminence**：徽章的视觉突出度。

# badgeProminence

**Instance Property**

The prominence to apply to badges associated with this environment.

## Declaration

```swift
var badgeProminence: BadgeProminence { get set }
```

## Discussion

The default is [standard](../BadgeProminence/standard.zh-Hans.md).

## Displaying a badge on a list item

- **badge(_:)**: Generates a badge for the view from an integer value.
- **badgeProminence(_:)**: Specifies the prominence of badges created by this view.
- **BadgeProminence**: The visual prominence of a badge.
