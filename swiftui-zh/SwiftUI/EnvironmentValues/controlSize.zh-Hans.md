# controlSize

**Instance Property**

The size to apply to controls within a view.

## Declaration

```swift
var controlSize: ControlSize { get set }
```

## Discussion

The default is [regular](../ControlSize/regular.zh-Hans.md).

## Sizing controls

- **controlSize(_:)**: Sets the size for controls within this view.
- **ControlSize**: The size classes, like regular or small, that you can apply to controls within a view.
