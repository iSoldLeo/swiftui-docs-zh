# 标题突出

**实例属性**

视图中应用于章节标题的突出显示。

## 声明

```swift
var headerProminence: Prominence { get set }
```

## 讨论

默认值为 [standard](../Prominence/standard.zh-Hans.md)。

## 配置标头

- **headerProminence(_:)**：设置此视图的标题突出显示。
- **Prominence**：表示视图层次结构突出度的类型。

# headerProminence

**Instance Property**

The prominence to apply to section headers within a view.

## Declaration

```swift
var headerProminence: Prominence { get set }
```

## Discussion

The default is [standard](../Prominence/standard.zh-Hans.md).

## Configuring headers

- **headerProminence(_:)**: Sets the header prominence for this view.
- **Prominence**: A type indicating the prominence of a view hierarchy.
