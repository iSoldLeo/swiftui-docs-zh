# isFocusEffectEnabled

**实例属性**

布尔值，表示与此环境相关的视图是否允许显示焦点效果。

## 声明

```swift
var isFocusEffectEnabled: Bool { get set }
```

## 讨论

默认值为 `true`。

## 配置效果

- **focusEffectDisabled(_:)**：添加控制此视图是否可以显示焦点特效（如默认焦点环或悬停特效）的条件。

# isFocusEffectEnabled

**Instance Property**

A Boolean value that indicates whether the view associated with this environment allows focus effects to be displayed.

## Declaration

```swift
var isFocusEffectEnabled: Bool { get set }
```

## Discussion

The default value is `true`.

## Configuring effects

- **focusEffectDisabled(_:)**: Adds a condition that controls whether this view can display focus effects, such as a default focus ring or hover effect.
