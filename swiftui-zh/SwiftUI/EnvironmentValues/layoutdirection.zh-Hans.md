# 布局方向

**实例属性**

与当前环境相关的布局方向。

## 声明

```swift
var layoutDirection: LayoutDirection { get set }
```

## 讨论

使用此值可确定或设置环境使用的是从左到右还是从右到左的方向。

## 设置布局方向

- **layoutDirectionBehavior(_:)**：设置该视图在不同布局方向上的行为。
- **LayoutDirectionBehavior**：描述布局方向改变时应发生的情况。
- **LayoutDirection**：SwiftUI 可以布局内容的方向。
- **LayoutRotationUnaryLayout**：SwiftUI 可以布局内容的方向。

# layoutDirection

**Instance Property**

The layout direction associated with the current environment.

## Declaration

```swift
var layoutDirection: LayoutDirection { get set }
```

## Discussion

Use this value to determine or set whether the environment uses a left-to-right or right-to-left direction.

## Setting a layout direction

- **layoutDirectionBehavior(_:)**: Sets the behavior of this view for different layout directions.
- **LayoutDirectionBehavior**: A description of what should happen when the layout direction changes.
- **LayoutDirection**: A direction in which SwiftUI can lay out content.
- **LayoutRotationUnaryLayout**
