# menuOrder

**Instance Property**

The preferred order of items for menus presented from this view.

## Declaration

```swift
var menuOrder: MenuOrder { get set }
```

## Discussion

Set this value for a view hierarchy by calling the [menuOrder(_:)](../View/menuOrder.zh-Hans.md) view modifier.

## Setting a preferred order

- **menuOrder(_:)**: Sets the preferred order of items for menus presented from this view.
- **MenuOrder**: The order in which a menu presents its content.
