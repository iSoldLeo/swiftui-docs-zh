# rename

**Instance Property**

An action that activates the standard rename interaction.

## Declaration

```swift
var rename: RenameAction? { get }
```

## Discussion

Use the [renameAction(_:)](../View/renameAction.zh-Hans.md) modifier to configure the rename action in the environment.

## Renaming a document

- **RenameButton**: A button that triggers a standard rename action.
- **renameAction(_:)**: Sets a closure to run for the rename action.
- **RenameAction**: An action that activates a standard rename interaction.
