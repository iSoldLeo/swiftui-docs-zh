# 请求年龄范围

**实例属性**

环境值，为请求年龄范围提供平台配置的操作。

## 声明

```swift

```

# requestAgeRange

**Instance Property**

An environment value that provides a platform configured action for requesting age ranges.

## Declaration

```swift
var requestAgeRange: DeclaredAgeRangeAction { get }
```

## Discussion

This environment property can be called by a [doc://com.apple.documentation/documentation/SwiftUI/Button] click or [doc://com.apple.documentation/documentation/SwiftUI/View/onAppear(perform:)]  to invoke the declared age range API.
