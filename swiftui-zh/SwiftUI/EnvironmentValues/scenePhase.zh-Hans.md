# scenePhase

**Instance Property**

The current phase of the scene.

## Declaration

```swift
var scenePhase: ScenePhase { get set }
```

## Discussion

The system sets this value to provide an indication of the operational state of a scene or collection of scenes. The exact meaning depends on where you access the value. For more information, see [ScenePhase](../ScenePhase.zh-Hans.md).

## Monitoring scene life cycle

- **ScenePhase**: An indication of a scene’s operational state.
