# 符号渲染模式

**实例属性**

当前的符号渲染模式，或`nil`，表示该模式是使用当前图像和前景样式作为参数自动选择的。

## 声明

```swift
var symbolRenderingMode: SymbolRenderingMode? { get set }
```

## 设置符号渲染模式

- **symbolRenderingMode(_:)**：设置此视图中符号图像的渲染模式。
- **SymbolRenderingMode**：符号渲染模式。
- **SymbolColorRenderingMode**：在符号图像中填充图层的方法。
- **SymbolVariableValueMode**：一种符号渲染模式：一种渲染符号图像的变量值的方法。

# symbolRenderingMode

**Instance Property**

The current symbol rendering mode, or `nil` denoting that the mode is picked automatically using the current image and foreground style as parameters.

## Declaration

```swift
var symbolRenderingMode: SymbolRenderingMode? { get set }
```

## Setting symbol rendering modes

- **symbolRenderingMode(_:)**: Sets the rendering mode for symbol images within this view.
- **SymbolRenderingMode**: A symbol rendering mode.
- **SymbolColorRenderingMode**: A method of filling a layer in a symbol image.
- **SymbolVariableValueMode**: A method of rendering the variable value of a symbol image.
