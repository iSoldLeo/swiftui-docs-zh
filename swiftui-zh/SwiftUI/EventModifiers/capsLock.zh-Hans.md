# capsLock

**类型属性**

大写锁定键。

## 声明

```swift
static let capsLock: EventModifiers
```

## 获取修饰键

- **all**：所有可能的修饰键。

- **command**：Command 键。

- **control**：Control 键。

- **numericPad**：数字小键盘上的任意键。

- **option**：Option 键。

- **shift**：Shift 键。

# capsLock

**Type Property**

The Caps Lock key.

## Declaration

```swift
static let capsLock: EventModifiers
```

## Getting modifier keys

- **all**: All possible modifier keys.
- **command**: The Command key.
- **control**: The Control key.
- **numericPad**: Any key on the numeric keypad.
- **option**: The Option key.
- **shift**: The Shift key.
