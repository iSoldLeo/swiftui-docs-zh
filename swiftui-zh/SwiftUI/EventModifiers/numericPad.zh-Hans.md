# numericPad

**类型属性**

数字小键盘上的任意键。

## 声明

```swift
static let numericPad: EventModifiers
```

## 获取修饰键

- **all**：所有可能的修饰键。

- **capsLock**：Caps Lock 键。

- **command**：Command 键。

- **control**：Control 键。

- **option**：Option 键。

- **shift**：Shift 键。

# numericPad

**Type Property**

Any key on the numeric keypad.

## Declaration

```swift
static let numericPad: EventModifiers
```

## Getting modifier keys

- **all**: All possible modifier keys.
- **capsLock**: The Caps Lock key.
- **command**: The Command key.
- **control**: The Control key.
- **option**: The Option key.
- **shift**: The Shift key.
