# Shift

**类型属性**

Shift 键。

## 声明

```swift
static let shift: EventModifiers
```

## 获取修饰键

- **all**：所有可能的修饰键。

- **capsLock**：Caps Lock 键。

- **command**：Command 键。

- **control**：Control 键。

- **numericPad**：数字小键盘上的任意键。

- **option**：Option 键。

# shift

**Type Property**

The Shift key.

## Declaration

```swift
static let shift: EventModifiers
```

## Getting modifier keys

- **all**: All possible modifier keys.
- **capsLock**: The Caps Lock key.
- **command**: The Command key.
- **control**: The Control key.
- **numericPad**: Any key on the numeric keypad.
- **option**: The Option key.
