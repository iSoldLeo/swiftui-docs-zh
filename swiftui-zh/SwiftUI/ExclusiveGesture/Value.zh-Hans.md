# ExclusiveGesture.Value

**Enumeration**

The value of an exclusive gesture that indicates which of two gestures succeeded.

## Declaration

```swift
@frozen enum Value
```

## Getting gesture values

- **ExclusiveGesture.Value.first(_:)**: The first of two gestures succeeded.
- **ExclusiveGesture.Value.second(_:)**: The second of two gestures succeeded.

- Copyable
- Equatable
- Sendable
- SendableMetatype
