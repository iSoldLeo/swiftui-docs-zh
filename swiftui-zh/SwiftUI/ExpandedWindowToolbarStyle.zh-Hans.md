# ExpandedWindowToolbarStyle

**Structure**

一种窗口工具栏样式，其标题栏区域显示在工具栏上方。

## 声明

```swift
struct ExpandedWindowToolbarStyle
```

## 概述

您也可以使用 [expanded](WindowToolbarStyle/expanded.zh-Hans.md) 来构造此样式。

## 创建窗口工具栏样式

- **init()**：创建展开的窗口工具栏样式。

## 支持的类型

- **DefaultWindowToolbarStyle**：默认的窗口工具栏样式。

- **UnifiedWindowToolbarStyle**：一种窗口工具栏样式，其工具栏和标题栏以内联方式显示。

- **UnifiedCompactWindowToolbarStyle**：一种窗口工具栏样式，类似于 [unified](WindowToolbarStyle/unified.zh-Hans.md)，但垂直尺寸更紧凑。

# ExpandedWindowToolbarStyle

**Structure**

A window toolbar style which displays its title bar area above the toolbar.

## Declaration

```swift
struct ExpandedWindowToolbarStyle
```

## Overview

You can also use [expanded](WindowToolbarStyle/expanded.zh-Hans.md) to construct this style.

## Creating the window toolbar style

- **init()**: Creates an expanded window toolbar style.

## Supporting types

- **DefaultWindowToolbarStyle**: The default window toolbar style.
- **UnifiedWindowToolbarStyle**: A window toolbar style which displays its toolbar and title bar inline.
- **UnifiedCompactWindowToolbarStyle**: A window toolbar style similar to [unified](WindowToolbarStyle/unified.zh-Hans.md), but with a more compact vertical sizing.

## Conforms To

- WindowToolbarStyle
