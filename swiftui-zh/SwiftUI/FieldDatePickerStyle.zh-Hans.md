# FieldDatePickerStyle

**Structure**

日期选择器样式，用于在可编辑字段中显示组件。

## 声明

```swift
struct FieldDatePickerStyle
```

## 概述

您也可以使用 [field](DatePickerStyle/field.zh-Hans.md) 来构建此样式。

## 创建日期选择器样式

- **init()**：创建字段日期选择器样式的实例。

## 支持的类型

- **DefaultDatePickerStyle**：日期选择器的默认样式。

- **CompactDatePickerStyle**：一种日期选择器样式，以紧凑的文本格式显示各个组件。

- **GraphicalDatePickerStyle**：一种日期选择器样式，显示交互式日历或时钟。

- **StepperFieldDatePickerStyle**：一种系统样式，在可编辑字段中显示各个组件，并带有可递增/递减所选组件的步进器。

- **WheelDatePickerStyle**：一种日期选择器样式，将每个组件显示为可滚动轮盘中的列。

## 符合以下样式

- DatePickerStyle

# FieldDatePickerStyle

**Structure**

A date picker style that displays the components in an editable field.

## Declaration

```swift
struct FieldDatePickerStyle
```

## Overview

You can also use [field](DatePickerStyle/field.zh-Hans.md) to construct this style.

## Creating the date picker style

- **init()**: Creates an instance of the field date picker style.

## Supporting types

- **DefaultDatePickerStyle**: The default style for date pickers.
- **CompactDatePickerStyle**: A date picker style that displays the components in a compact, textual format.
- **GraphicalDatePickerStyle**: A date picker style that displays an interactive calendar or clock.
- **StepperFieldDatePickerStyle**: A system style that displays the components in an editable field, with adjoining stepper that can increment/decrement the selected component.
- **WheelDatePickerStyle**: A date picker style that displays each component as columns in a scrollable wheel.

## Conforms To

- DatePickerStyle
