# FileDocumentReadConfiguration

**Structure**

The configuration for reading file contents.

## Declaration

```swift
struct FileDocumentReadConfiguration
```

## Reading the content

- **contentType**: The expected uniform type of the file contents.
- **file**: The file wrapper containing the document content.

## Reading and writing documents

- **FileDocumentWriteConfiguration**: The configuration for serializing file contents.
