# FileDocumentWriteConfiguration

**Structure**

用于序列化文件内容的配置。

## 声明

```swift
struct FileDocumentWriteConfiguration
```

## 写入内容

- **contentType**：文件内容的预期统一类型。

- **existingFile**：包含当前文档内容的文件包装器。`nil`：如果文档未保存。

## 读取和写入文档

- **FileDocumentReadConfiguration**：用于读取文件内容的配置。

# FileDocumentWriteConfiguration

**Structure**

The configuration for serializing file contents.

## Declaration

```swift
struct FileDocumentWriteConfiguration
```

## Writing the content

- **contentType**: The expected uniform type of the file contents.
- **existingFile**: The file wrapper containing the current document content. `nil` if the document is unsaved.

## Reading and writing documents

- **FileDocumentReadConfiguration**: The configuration for reading file contents.
