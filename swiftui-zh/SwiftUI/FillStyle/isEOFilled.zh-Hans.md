# isEOFilled

**实例属性**

布尔值，用于指示在渲染形状时是否使用偶数规则。

## 声明

```swift
var isEOFilled: Bool
```

## 讨论

当 `isOEFilled` 是 `false` 时，样式使用非零绕组数规则。

# isEOFilled

**Instance Property**

A Boolean value that indicates whether to use the even-odd rule when rendering a shape.

## Declaration

```swift
var isEOFilled: Bool
```

## Discussion

When `isOEFilled` is `false`, the style uses the non-zero winding number rule.

## Setting fill style properties

- **isAntialiased**: A Boolean value that indicates whether to apply antialiasing to the edges of a shape.
