# isPresented

**实例属性**

用于控制查找导航器是否显示的绑定，如果未通过 [findNavigator(isPresented:)](../View/findNavigator_isPresented.zh-Hans.md) 修饰符提供绑定，则该属性值为 nil。

## 声明

```swift
var isPresented: Binding<Bool>?
```

# isPresented

**Instance Property**

A binding controlling if the find navigator is presented, or nil if no binding has been provided via the [findNavigator(isPresented:)](../View/findNavigator_isPresented.zh-Hans.md) modifier.

## Declaration

```swift
var isPresented: Binding<Bool>?
```
