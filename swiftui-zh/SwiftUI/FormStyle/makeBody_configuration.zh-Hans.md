# makeBody(configuration:)

**Instance Method**

Creates a view that represents the body of a form.

## Declaration

```swift
@ViewBuilder @MainActor @preconcurrency func makeBody(configuration: Self.Configuration) -> Self.Body
```

## Parameters

- **configuration**: The properties of the form.

## Return Value

A view that has behavior and appearance that enables it to function as a [Form](../Form.zh-Hans.md).

## Creating custom form styles

- **FormStyle.Configuration**: The properties of a form instance.
- **Body**: A view that represents the appearance and interaction of a form.
