# GaugeStyleConfiguration.CurrentValueLabel

**Structure**

A type-erased value label of a gauge that contains the current value.

## Declaration

```swift
@MainActor @preconcurrency struct CurrentValueLabel
```

## Setting the value

- **value**: The current value of the gauge.
- **currentValueLabel**: A view that describes the current value.
- **GaugeStyleConfiguration.MarkedValueLabel**: A type-erased label describing a specific value of a gauge.
