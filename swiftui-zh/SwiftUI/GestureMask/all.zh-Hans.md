# 所有

**类型属性**

在视图及其子视图中启用添加的手势和所有其他手势。

## 声明

```swift
static let all: GestureMask
```

## 获取手势选项

- **gesture**：启用添加的手势，但禁用子视图层次结构中的所有手势。
- **subviews**：启用子视图层次结构中的所有手势，但禁用添加的手势。
- **none**：禁用子视图层次结构中的所有手势，包括添加的手势。

# all

**Type Property**

Enable both the added gesture as well as all other gestures on the view and its subviews.

## Declaration

```swift
static let all: GestureMask
```

## Getting gesture options

- **gesture**: Enable the added gesture but disable all gestures in the subview hierarchy.
- **subviews**: Enable all gestures in the subview hierarchy but disable the added gesture.
- **none**: Disable all gestures in the subview hierarchy, including the added gesture.
