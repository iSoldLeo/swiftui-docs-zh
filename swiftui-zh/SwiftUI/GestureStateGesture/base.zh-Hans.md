# base

**Instance Property**

The originating gesture.

## Declaration

```swift
var base: Base
```

## Creating an in-progress gesture

- **init(base:state:body:)**: Creates a new gesture that’s the result of an ongoing gesture.
- **state**: A value that changes as the user performs the gesture.
