# 状态

**实例属性**

用户执行手势时会改变的值。

## 声明

```swift
var state: GestureState<State>
```

## 创建一个进行中的手势

- **init(base:state:body:)**：创建一个新手势，该手势是正在进行的手势的结果。
- **base**：初始手势。

# state

**Instance Property**

A value that changes as the user performs the gesture.

## Declaration

```swift
var state: GestureState<State>
```

## Creating an in-progress gesture

- **init(base:state:body:)**: Creates a new gesture that’s the result of an ongoing gesture.
- **base**: The originating gesture.
