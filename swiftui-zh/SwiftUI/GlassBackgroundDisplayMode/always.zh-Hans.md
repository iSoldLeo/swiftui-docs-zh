# GlassBackgroundDisplayMode.always

**Case**

始终显示 Glass 材质。

## 声明

```swift
case always
```

## 获取模式

- **GlassBackgroundDisplayMode.implicit**：仅当视图尚未包含在 Glass 材质中时才显示 Glass 材质。

- **GlassBackgroundDisplayMode.never**：从不显示 Glass 材质。

# GlassBackgroundDisplayMode.always

**Case**

Always display the glass material.

## Declaration

```swift
case always
```

## Getting the mode

- **GlassBackgroundDisplayMode.implicit**: Display the glass material only when the view isn’t already contained in glass.
- **GlassBackgroundDisplayMode.never**: Never display the glass material.
