# GlassBackgroundDisplayMode.never

**Case**

始终不显示 Glass 材质。

## 声明

```swift
case never
```

## 获取模式

- **GlassBackgroundDisplayMode.always**：始终显示 Glass 材质。

- **GlassBackgroundDisplayMode.implicit**：仅当视图尚未包含在 Glass 材质中时才显示 Glass 材质。

# GlassBackgroundDisplayMode.never

**Case**

Never display the glass material.

## Declaration

```swift
case never
```

## Getting the mode

- **GlassBackgroundDisplayMode.always**: Always display the glass material.
- **GlassBackgroundDisplayMode.implicit**: Display the glass material only when the view isn’t already contained in glass.
