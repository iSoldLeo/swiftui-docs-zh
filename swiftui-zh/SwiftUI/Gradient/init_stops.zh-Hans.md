# init(stops:)

**Initializer**

使用颜色止点数组创建渐变效果。

```swift
init(stops: [Gradient.Stop])
```

## 从停止点创建梯度

- **stops**：色站数组。
- **Gradient.Stop**：渐变中的一个色站。

# init(stops:)

**Initializer**

Creates a gradient from an array of color stops.

## Declaration

```swift
init(stops: [Gradient.Stop])
```

## Creating a gradient from stops

- **stops**: The array of color stops.
- **Gradient.Stop**: One color stop in the gradient.
