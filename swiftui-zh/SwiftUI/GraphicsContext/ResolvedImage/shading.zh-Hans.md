# 阴影

**实例属性**

用于填充图像的可选阴影。

## 声明

```swift
var shading: GraphicsContext.Shading?
```

## 讨论

对于模板图像，该属性的值默认为 [foreground](../Shading/foreground.zh-Hans.md)，否则为 `nil`。

## 获取图像属性

- **size**：图像的大小。
- **baseline**：图像顶部到基线的距离。

# shading

**Instance Property**

An optional shading to fill the image with.

## Declaration

```swift
var shading: GraphicsContext.Shading?
```

## Discussion

The value of this property defaults to [foreground](../Shading/foreground.zh-Hans.md) for template images, and to `nil` otherwise.

## Getting the image properties

- **size**: The size of the image.
- **baseline**: The distance from the top of the image to its baseline.
