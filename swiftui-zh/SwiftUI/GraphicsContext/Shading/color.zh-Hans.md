# color(_:)

**类型方法**

返回一个填充颜色的阴影实例。

## 声明

```swift
static func color(_ color: Color) -> GraphicsContext.Shading
```

## 参数

- **color**：定义阴影颜色的 [Color](../../Color.zh-Hans.md) 实例。

## 返回值

一个填充了颜色的阴影实例。

### 颜色

- **color(_:red:green:blue:opacity:)**：返回在给定颜色空间中填充颜色的阴影实例。
- **color(_:white:opacity:)**：返回一个在给定颜色空间中填充单色的着色实例。

# color(_:)

**Type Method**

Returns a shading instance that fills with a color.

## Declaration

```swift
static func color(_ color: Color) -> GraphicsContext.Shading
```

## Parameters

- **color**: A [Color](../../Color.zh-Hans.md) instance that defines the color of the shading.

## Return Value

A shading instance filled with a color.

## Colors

- **color(_:red:green:blue:opacity:)**: Returns a shading instance that fills with a color in the given color space.
- **color(_:white:opacity:)**: Returns a shading instance that fills with a monochrome color in the given color space.
