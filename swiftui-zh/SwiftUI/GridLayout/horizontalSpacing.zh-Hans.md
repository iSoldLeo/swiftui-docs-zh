# horizontalSpacing

**实例属性**

相邻子视图之间的水平距离。

## 声明

```swift
var horizontalSpacing: CGFloat?
```

## 说明

将此值设置为 `nil` 可使用子视图之间的默认水平距离。

## 获取网格属性

- **alignment**：子视图的对齐方式。

- **verticalSpacing**：相邻子视图之间的垂直距离。

# horizontalSpacing

**Instance Property**

The horizontal distance between adjacent subviews.

## Declaration

```swift
var horizontalSpacing: CGFloat?
```

## Discussion

Set this value to `nil` to use default horizonal distances between subviews.

## Getting the grid’s properties

- **alignment**: The alignment of subviews.
- **verticalSpacing**: The vertical distance between adjacent subviews.
