# verticalSpacing

**实例属性**

相邻子视图之间的垂直距离。

## 声明

```swift
var verticalSpacing: CGFloat?
```

## 说明

将此值设置为 `nil` 可使用子视图之间的默认垂直距离。

## 获取网格属性

- **alignment**：子视图的对齐方式。

- **horizontalSpacing**：相邻子视图之间的水平距离。

# verticalSpacing

**Instance Property**

The vertical distance between adjacent subviews.

## Declaration

```swift
var verticalSpacing: CGFloat?
```

## Discussion

Set this value to `nil` to use default vertical distances between subviews.

## Getting the grid’s properties

- **alignment**: The alignment of subviews.
- **horizontalSpacing**: The horizontal distance between adjacent subviews.
