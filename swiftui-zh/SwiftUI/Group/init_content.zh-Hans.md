# init(content:)

**Initializer**

创建一个实例，通过按顺序组合传入的结果生成器中指定的所有 Rotor 内容来生成 Rotor 内容。

### 声明

```swift
init(@AccessibilityRotorContentBuilder content: () -> Content)
```

## 参数

- **content**：为组群生成 Rotor 内容的结果生成器。

# init(content:)

**Initializer**

Creates an instance that generates Rotor content by combining, in order, all the Rotor content specified in the passed-in result builder.

## Declaration

```swift
init(@AccessibilityRotorContentBuilder content: () -> Content)
```

## Parameters

- **content**: The result builder that generates Rotor content for the group.
