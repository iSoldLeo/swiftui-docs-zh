# pinch

**Type Property**

Activation that requires a pinched hand.

## Declaration

```swift
static let pinch: HandActivationBehavior
```

## Getting the behaviors

- **automatic**: The default activation behavior, including direct touch, direct pinch, and indirect pinch.
