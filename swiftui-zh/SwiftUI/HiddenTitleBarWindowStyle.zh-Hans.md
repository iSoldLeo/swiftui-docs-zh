# 隐藏标题栏窗口样式

**Structure**

一种窗口样式，可同时隐藏窗口的标题和标题栏区域的后部，从而显示更多的窗口内容。

## 声明

```swift
struct HiddenTitleBarWindowStyle
```

## 概览

您也可以使用 [hiddenTitleBar](WindowStyle/hiddenTitleBar.zh-Hans.md) 构建这种样式。

## 创建窗口样式

- **init()**：创建隐藏标题栏窗口样式。

## 支持类型

- **DefaultWindowStyle**：默认窗口样式。
- **PlainWindowStyle**：普通窗口样式。
- **TitleBarWindowStyle**：默认窗口样式：显示窗口标题栏部分的窗口样式。
- **VolumetricWindowStyle**：一种窗口样式，用于创建三维体积窗口。

## 符合

# HiddenTitleBarWindowStyle

**Structure**

A window style which hides both the window’s title and the backing of the titlebar area, allowing more of the window’s content to show.

## Declaration

```swift
struct HiddenTitleBarWindowStyle
```

## Overview

You can also use [hiddenTitleBar](WindowStyle/hiddenTitleBar.zh-Hans.md) to construct this style.

## Creating the window style

- **init()**: Creates a hidden title bar window style.

## Supporting types

- **DefaultWindowStyle**: The default window style.
- **PlainWindowStyle**: The plain window style.
- **TitleBarWindowStyle**: A window style which displays the title bar section of the window.
- **VolumetricWindowStyle**: A window style that creates a 3D volumetric window.

## Conforms To

- WindowStyle
