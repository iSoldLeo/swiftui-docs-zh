# HorizontalDirection

**Enumeration**

A direction on the horizontal axis.

## Declaration

```swift
@frozen enum HorizontalDirection
```

## Structures

- **HorizontalDirection.Set**: An efficient set of horizontal directions.

## Enumeration Cases

- **HorizontalDirection.leading**: The leading direction.
- **HorizontalDirection.trailing**: The trailing direction.

## Supporting types

- **VerticalDirection**: A direction on the horizontal axis.
- **FrameResizePosition**: The position along the perimeter of a rectangular frame (its edges and corners) from which it’s resized.
- **FrameResizeDirection**: The direction in which a rectangular frame can be resized.
