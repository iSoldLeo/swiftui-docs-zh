
- **leading**：一个仅包含前导水平方向的集合。

# HorizontalDirection.Set

**Structure**

An efficient set of horizontal directions.

## Declaration

```swift
@frozen struct Set
```

## Initializers

- **init(_:)**: Creates a set of directions containing only the specified direction.

## Type Properties

- **all**: A set containing the leading and trailing horizontal directions.
- **leading**: A set containing only the leading horizontal direction.
- **trailing**: A set containing only the trailing horizontal direction.

## Conforms To

- BitwiseCopyable
- Copyable
- Equatable
- ExpressibleByArrayLiteral
- Hashable
- OptionSet
- RawRepresentable
- Sendable
- SendableMetatype
- SetAlgebra
