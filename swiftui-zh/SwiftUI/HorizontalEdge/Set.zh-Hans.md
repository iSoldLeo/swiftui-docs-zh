# HorizontalEdge.Set

**Structure**

An efficient set of horizontal edges.

## Declaration

```swift
@frozen struct Set
```

## Getting edge sets

- **all**: A set containing the leading and trailing horizontal edges.
- **leading**: A set containing only the leading horizontal edge.
- **trailing**: A set containing only the trailing horizontal edge.

## Creating an edge set

- **init(_:)**: Creates a set of edges containing only the specified horizontal edge.
