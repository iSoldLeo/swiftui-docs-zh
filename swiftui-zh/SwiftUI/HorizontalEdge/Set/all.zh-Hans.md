# all

**Type Property**

A set containing the leading and trailing horizontal edges.

## Declaration

```swift
static let all: HorizontalEdge.Set
```

## Getting edge sets

- **leading**: A set containing only the leading horizontal edge.
- **trailing**: A set containing only the trailing horizontal edge.
