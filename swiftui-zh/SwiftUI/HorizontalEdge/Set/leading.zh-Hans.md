**类型属性**

```swift

```

# leading

**Type Property**

A set containing only the leading horizontal edge.

## Declaration

```swift
static let leading: HorizontalEdge.Set
```

## Getting edge sets

- **all**: A set containing the leading and trailing horizontal edges.
- **trailing**: A set containing only the trailing horizontal edge.
