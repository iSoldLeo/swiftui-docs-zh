# 尾随

**类型属性**

仅包含水平尾边的集合。

## 声明

```swift
static let trailing: HorizontalEdge.Set
```

## 获取边缘集

- **all**：包含水平前缘和水平后缘的集合。
- **leading**：仅包含前水平边的集合。

# trailing

**Type Property**

A set containing only the trailing horizontal edge.

## Declaration

```swift
static let trailing: HorizontalEdge.Set
```

## Getting edge sets

- **all**: A set containing the leading and trailing horizontal edges.
- **leading**: A set containing only the leading horizontal edge.
