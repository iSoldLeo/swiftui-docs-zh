# offset(_:)

**实例方法**

将视图水平和垂直偏移指定的偏移量（由 offset 参数指定）。

## 声明

```swift
func offset(_ offset: CGSize) -> some HoverEffectContent

```

## 参数

- **offset**：视图的偏移距离。

## 返回值

一个将视图偏移 `offset` 的效果。

# offset(_:)

**Instance Method**

Offsets the view by the horizontal and vertical amount specified in the offset parameter.

## Declaration

```swift
func offset(_ offset: CGSize) -> some HoverEffectContent

```

## Parameters

- **offset**: The distance to offset the view.

## Return Value

An effect that offsets the view by `offset`.
