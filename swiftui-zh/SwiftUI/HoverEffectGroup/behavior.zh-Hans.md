# HoverEffectGroup.Behavior

**Structure**

描述组中某个效果的行为。

## 声明

```swift
struct Behavior
```

## 类型属性

- **activatesGroup**：使匹配组中的其他效果变为活动状态。

- **followsGroup**：跟随匹配组的当前阶段。

- **ignoresGroup**：忽略匹配组的当前阶段。

- **preservesGroup**：保持组的当前阶段。

# HoverEffectGroup.Behavior

**Structure**

Describes the behavior of an effect in a group.

## Declaration

```swift
struct Behavior
```

## Type Properties

- **activatesGroup**: Causes other effects in the match group to become active.
- **followsGroup**: Follows the current phase of the match group.
- **ignoresGroup**: Ignores the current phase of the match group.
- **preservesGroup**: Preserves the current phase of the group.

## Conforms To

- Equatable
- Sendable
- SendableMetatype
