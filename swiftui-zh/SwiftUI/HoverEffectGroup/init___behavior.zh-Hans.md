# init(_:behavior:)

**Initializer**

从 `Namespace.ID` 创建一个新的 [HoverEffectGroup](../HoverEffectGroup.zh-Hans.md)。

## 声明

```swift
init(_ namespace: Namespace.ID, behavior: HoverEffectGroup.Behavior = .activatesGroup)
```

## 参数

- **namespace**：标识组的命名空间。

- **behavior**：该效果相对于组中其他效果的行为方式。

## 返回值

一个新的 HoverEffectGroup

# init(_:behavior:)

**Initializer**

Creates a new [HoverEffectGroup](../HoverEffectGroup.zh-Hans.md) from a `Namespace.ID`.

## Declaration

```swift
init(_ namespace: Namespace.ID, behavior: HoverEffectGroup.Behavior = .activatesGroup)
```

## Parameters

- **namespace**: The namespace that identifies the group.
- **behavior**: How the effect will behave relative to other effects in the group.

## Return Value

A new HoverEffectGroup
