# IconOnlyLabelStyle

**Structure**

仅显示标签图标的标签样式。

## 声明

```swift
struct IconOnlyLabelStyle
```

## 概述

您也可以使用 [iconOnly](LabelStyle/iconOnly.zh-Hans.md) 来构造此样式。

## 创建标签样式

- **init()**：创建仅显示图标的标签样式。

## 支持的类型

- **DefaultLabelStyle**：当前上下文中的默认标签样式。

- **TitleAndIconLabelStyle**：一种标签样式，使用系统标准布局同时显示标签标题和图标。

- **TitleOnlyLabelStyle**：一种标签样式，仅显示标签标题。

## 符合以下规范

- LabelStyle

# IconOnlyLabelStyle

**Structure**

A label style that only displays the icon of the label.

## Declaration

```swift
struct IconOnlyLabelStyle
```

## Overview

You can also use [iconOnly](LabelStyle/iconOnly.zh-Hans.md) to construct this style.

## Creating the label style

- **init()**: Creates an icon-only label style.

## Supporting types

- **DefaultLabelStyle**: The default label style in the current context.
- **TitleAndIconLabelStyle**: A label style that shows both the title and icon of the label using a system-standard layout.
- **TitleOnlyLabelStyle**: A label style that only displays the title of the label.

## Conforms To

- LabelStyle
