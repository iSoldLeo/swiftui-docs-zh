# 高

**类型属性**

允许图像内容使用不受限制的扩展范围。

## 声明

```swift
static let high: Image.DynamicRange
```

## 获取动态范围值

- **standard**：将图像内容的动态范围限制在标准范围内。
- **constrainedHigh**：允许图像内容使用某些扩展范围。这适合将 HDR 内容放在 SDR 内容旁边。

# high

**Type Property**

Allow image content to use an unrestricted extended range.

## Declaration

```swift
static let high: Image.DynamicRange
```

## Getting dynamic range values

- **standard**: Restrict the image content dynamic range to the standard range.
- **constrainedHigh**: Allow image content to use some extended range. This is appropriate for placing HDR content next to SDR content.
