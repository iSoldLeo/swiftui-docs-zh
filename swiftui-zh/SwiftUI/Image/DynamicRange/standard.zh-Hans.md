# 标准

**类型属性**

将图像内容的动态范围限制在标准范围内。

## 声明

```swift
static let standard: Image.DynamicRange
```

## 获取动态范围值

- **high**：允许图像内容使用不受限制的扩展范围。
- **constrainedHigh**：允许图像内容使用某些扩展范围。适合将 HDR 内容放在 SDR 内容旁边。

# standard

**Type Property**

Restrict the image content dynamic range to the standard range.

## Declaration

```swift
static let standard: Image.DynamicRange
```

## Getting dynamic range values

- **high**: Allow image content to use an unrestricted extended range.
- **constrainedHigh**: Allow image content to use some extended range. This is appropriate for placing HDR content next to SDR content.
