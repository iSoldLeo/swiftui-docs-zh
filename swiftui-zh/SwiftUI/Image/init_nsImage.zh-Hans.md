# init(nsImage:)

**Initializer**

从 AppKit 图像实例创建 SwiftUI 图像。

## 声明

```swift
nonisolated init(nsImage: NSImage)
```

## 参数

- **nsImage**：要使用 SwiftUI [Image](../Image.zh-Hans.md).实例包装的 AppKit 图像。

## 从另一个图像创建图像

- **init(uiImage:)**：从 UIKit 图像实例创建 SwiftUI 图像。

# init(nsImage:)

**Initializer**

Creates a SwiftUI image from an AppKit image instance.

## Declaration

```swift
nonisolated init(nsImage: NSImage)
```

## Parameters

- **nsImage**: The AppKit image to wrap with a SwiftUI [Image](../Image.zh-Hans.md). instance.

## Creating an image from another image

- **init(uiImage:)**: Creates a SwiftUI image from a UIKit image instance.
