# init(uiImage:)

**Initializer**

从 UIKit 图像实例创建 SwiftUI 图像。

## 声明

```swift
init(uiImage: UIImage)
```

## 参数

- **uiImage**：要使用 SwiftUI [Image](../Image.zh-Hans.md) 实例封装的 UIKit 图像。

## 从另一个图像创建一个图像

- **init(nsImage:)**：从 AppKit 图像实例创建 SwiftUI 图像。

# init(uiImage:)

**Initializer**

Creates a SwiftUI image from a UIKit image instance.

## Declaration

```swift
init(uiImage: UIImage)
```

## Parameters

- **uiImage**: The UIKit image to wrap with a SwiftUI [Image](../Image.zh-Hans.md) instance.

## Creating an image from another image

- **init(nsImage:)**: Creates a SwiftUI image from an AppKit image instance.
