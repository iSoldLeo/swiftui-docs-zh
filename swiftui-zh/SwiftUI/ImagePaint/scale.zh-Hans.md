# scale

**Instance Property**

A scale factor applied to the image while being drawn.

## Declaration

```swift
var scale: CGFloat
```

## Configuring the image paint style

- **image**: The image to be drawn.
- **sourceRect**: A unit-space rectangle defining how much of the source image to draw.
