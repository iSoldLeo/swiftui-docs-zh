# automatic

**Type Property**

A behavior that matches the system default behavior.

## Declaration

```swift
static var automatic: ImmersiveEnvironmentBehavior { get }
```

## Discussion

Use this value with the [immersiveEnvironmentBehavior(_:)](../scene/immersiveEnvironmentBehavior.zh-Hans.md) scene modifier to define that the system follows a default behavior regarding the system’s immersive environment when your app opens a scene.
