# coexist

**Type Property**

A behavior that keeps the system’s immersive environment as is when opening a scene.

## Declaration

```swift
static var coexist: ImmersiveEnvironmentBehavior { get }
```

## Discussion

Use this value with the [immersiveEnvironmentBehavior(_:)](../scene/immersiveEnvironmentBehavior.zh-Hans.md) scene modifier to define that the system should allow the system’s immersive environment to co-exist with the scene.
