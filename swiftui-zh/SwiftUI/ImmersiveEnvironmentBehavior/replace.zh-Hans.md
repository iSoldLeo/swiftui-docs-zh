# 替换

**类型属性**

一种行为，用于将当前打开的沉浸式环境替换为新场景。

## 声明

```swift
static var replace: ImmersiveEnvironmentBehavior { get }
```

## 说明

将此值与 [immersiveEnvironmentBehavior(_:)](../scene/immersiveEnvironmentBehavior.zh-Hans.md) 场景修饰符一起使用，以定义系统在打开场景之前应隐藏系统当前的沉浸式环境。

# replace

**Type Property**

A behavior that replaces any currently opened immersive environment with the new scene.

## Declaration

```swift
static var replace: ImmersiveEnvironmentBehavior { get }
```

## Discussion

Use this value with the [immersiveEnvironmentBehavior(_:)](../scene/immersiveEnvironmentBehavior.zh-Hans.md) scene modifier to define that the system should hide the system’s immersive environment before opening the scene.
