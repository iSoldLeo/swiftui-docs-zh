# ImmersiveSpaceViewContent

**Structure**

使用 SwiftUI 视图层级结构作为内容的沉浸式空间内容。

## 声明

```swift

```

## 概述

您不能直接创建此类型。SwiftUI 会在您使用基于视图的内容构造 [ImmersiveSpace](ImmersiveSpace.zh-Hans.md) 时创建它。

## 支持的类型

- **ImmersiveSpaceContent**：可用作沉浸式空间内容的类型。

# ImmersiveSpaceViewContent

**Structure**

Immersive space content that uses a SwiftUI view hierarchy as the content.

## Declaration

```swift
struct ImmersiveSpaceViewContent<Content> where Content : View
```

## Overview

You don’t create this type directly. SwiftUI creates it when you construct an [ImmersiveSpace](ImmersiveSpace.zh-Hans.md) with view-based content.

## Supporting types

- **ImmersiveSpaceContent**: A type that you can use as the content of an immersive space.

## Conforms To

- ImmersiveSpaceContent
