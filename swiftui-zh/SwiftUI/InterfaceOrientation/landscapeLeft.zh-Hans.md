# 左侧景观

**类型属性**

设备处于横向模式，设备顶部在左侧。

## 声明

```swift
static let landscapeLeft: InterfaceOrientation
```

## 获取方向

- **portrait**：设备处于纵向模式，设备顶部在上。
- **portraitUpsideDown**：设备处于纵向模式，但上下颠倒。
- **landscapeRight**：设备处于横向模式，设备顶部在右侧。

# landscapeLeft

**Type Property**

The device is in landscape mode, with the top of the device on the left.

## Declaration

```swift
static let landscapeLeft: InterfaceOrientation
```

## Getting an orientation

- **portrait**: The device is in portrait mode, with the top of the device on top.
- **portraitUpsideDown**: The device is in portrait mode, but is upside down.
- **landscapeRight**: The device is in landscape mode, with the top of the device on the right.
