# 肖像

**类型属性**

设备处于纵向模式，设备顶部位于上方。

## 声明

```swift
static let portrait: InterfaceOrientation
```

## 获取方向

- **portraitUpsideDown**：设备处于纵向模式，但上下颠倒。
- **landscapeLeft**：设备处于横向模式，设备顶部在左侧。
- **landscapeRight**：设备处于横向模式，设备顶部在右侧。

# portrait

**Type Property**

The device is in portrait mode, with the top of the device on top.

## Declaration

```swift
static let portrait: InterfaceOrientation
```

## Getting an orientation

- **portraitUpsideDown**: The device is in portrait mode, but is upside down.
- **landscapeLeft**: The device is in landscape mode, with the top of the device on the left.
- **landscapeRight**: The device is in landscape mode, with the top of the device on the right.
