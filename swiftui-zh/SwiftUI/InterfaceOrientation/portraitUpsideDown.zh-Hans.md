# portraitUpsideDown

**类型属性**

设备处于纵向模式，但上下颠倒。

## 声明

```swift
static let portraitUpsideDown: InterfaceOrientation
```

## 获取方向

- **portrait**：设备处于纵向模式，设备顶部在上。
- **landscapeLeft**：设备处于横向模式，设备顶部在左侧。
- **landscapeRight**：设备处于横向模式，设备顶部在右侧。

# portraitUpsideDown

**Type Property**

The device is in portrait mode, but is upside down.

## Declaration

```swift
static let portraitUpsideDown: InterfaceOrientation
```

## Getting an orientation

- **portrait**: The device is in portrait mode, with the top of the device on top.
- **landscapeLeft**: The device is in landscape mode, with the top of the device on the left.
- **landscapeRight**: The device is in landscape mode, with the top of the device on the right.
