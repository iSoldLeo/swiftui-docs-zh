# all

**类型属性**

一个匹配所有按键阶段的值。

## 声明

```swift
static let all: KeyPress.Phases
```

## 获取按键阶段

- **down**：用户按下某个键。

- **up**：用户松开某个键。

- **repeat**：用户按住某个键以触发一系列重复事件。

# all

**Type Property**

A value that matches all key press phases.

## Declaration

```swift
static let all: KeyPress.Phases
```

## Getting the phases

- **down**: The user pressed down on a key.
- **up**: The user released a key.
- **repeat**: The user held a key down to issue a sequence of repeating events.
