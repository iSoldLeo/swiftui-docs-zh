# down

**类型属性**

用户按下某个键。

## 声明

```swift
static let down: KeyPress.Phases
```

## 获取按键阶段

- **up**：用户松开某个键。

- **repeat**：用户按住某个键，触发一系列重复事件。

- **all**：包含所有按键阶段的值。

# down

**Type Property**

The user pressed down on a key.

## Declaration

```swift
static let down: KeyPress.Phases
```

## Getting the phases

- **up**: The user released a key.
- **repeat**: The user held a key down to issue a sequence of repeating events.
- **all**: A value that matches all key press phases.
