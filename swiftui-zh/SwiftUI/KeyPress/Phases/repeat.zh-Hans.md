# repeat

**类型属性**

用户按住某个键以触发一系列重复事件。

## 声明

```swift
static let `repeat`: KeyPress.Phases
```

## 获取按键阶段

- **down**：用户按下某个键。

- **up**：用户松开某个键。

- **all**：包含所有按键阶段的值。

# repeat

**Type Property**

The user held a key down to issue a sequence of repeating events.

## Declaration

```swift
static let `repeat`: KeyPress.Phases
```

## Getting the phases

- **down**: The user pressed down on a key.
- **up**: The user released a key.
- **all**: A value that matches all key press phases.
