# 字符

**实例属性**

按下按键时生成的字符，假设没有使用任何修饰键。

## 声明

```swift
let characters: String
```

## 获取按键信息

- **key**：按下按键的等效键值。

- **modifiers**：用户除了按下的按键之外还按住的修饰键集合。

# characters

**Instance Property**

The characters generated by the pressed key as if no modifier key applies.

## Declaration

```swift
let characters: String
```

## Getting the keypress

- **key**: The key equivalent value for the pressed key.
- **modifiers**: The set of modifier keys the user held in addition to the pressed key.
