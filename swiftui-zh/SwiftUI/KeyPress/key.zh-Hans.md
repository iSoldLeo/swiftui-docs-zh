# key

**实例属性**

按下的键对应的键值。

## 声明

```swift
let key: KeyEquivalent
```

## 获取按键信息

- **characters**：按下的键生成的字符，假设没有使用任何修饰键。

- **modifiers**：用户除了按下的键之外还按住的修饰键集合。

# key

**Instance Property**

The key equivalent value for the pressed key.

## Declaration

```swift
let key: KeyEquivalent
```

## Getting the keypress

- **characters**: The characters generated by the pressed key as if no modifier key applies.
- **modifiers**: The set of modifier keys the user held in addition to the pressed key.
