# modifiers

**实例属性**

用户除了按下的键之外还按住的修饰键集合。

## 声明

```swift
let modifiers: EventModifiers
```

## 获取按键

- **key**：按下的键的等效值。

- **characters**：按下的键在没有使用任何修饰键的情况下生成的字符。

# modifiers

**Instance Property**

The set of modifier keys the user held in addition to the pressed key.

## Declaration

```swift
let modifiers: EventModifiers
```

## Getting the keypress

- **key**: The key equivalent value for the pressed key.
- **characters**: The characters generated by the pressed key as if no modifier key applies.
