# 自动

**类型属性**

将快捷方式重新映射为国际通用快捷方式，并根据需要从右向左镜像。

## 声明

```swift
static let automatic: KeyboardShortcut.Localization
```

## 讨论

这是默认配置。

## 获取本地化策略

- **custom**：不使用自动快捷方式重映射。
- **withoutMirroring**：不要镜像快捷方式。

# automatic

**Type Property**

Remap shortcuts to their international counterparts, mirrored for right-to-left usage if appropriate.

## Declaration

```swift
static let automatic: KeyboardShortcut.Localization
```

## Discussion

This is the default configuration.

## Getting localization strategies

- **custom**: Don’t use automatic shortcut remapping.
- **withoutMirroring**: Don’t mirror shortcuts.
