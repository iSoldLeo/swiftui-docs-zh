# 本地化

**实例属性**

应用于此快捷方式的本地化策略。

## 声明

```swift
var localization: KeyboardShortcut.Localization
```

## 创建本地化快捷方式

- **init(_:modifiers:localization:)**：使用给定的等效键和修改键集创建新的快捷键。
- **KeyboardShortcut.Localization**：键盘快捷方式如何参与自动本地化的选项。

# localization

**Instance Property**

The localization strategy to apply to this shortcut.

## Declaration

```swift
var localization: KeyboardShortcut.Localization
```

## Creating a localized shortcut

- **init(_:modifiers:localization:)**: Creates a new keyboard shortcut with the given key equivalent and set of modifier keys.
- **KeyboardShortcut.Localization**: Options for how a keyboard shortcut participates in automatic localization.
