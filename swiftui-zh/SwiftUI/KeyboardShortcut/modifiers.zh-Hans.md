# 修改器

**实例属性**

用户连同等效键一起按下以激活快捷键的修改键。

## 声明

```swift
var modifiers: EventModifiers
```

## 创建快捷方式

- **init(_:modifiers:)**：使用给定的等效键和修改键集创建新的快捷键。
- **key**：用户连同任何指定的修改键一起按下以激活快捷键的等效键。

# modifiers

**Instance Property**

The modifier keys that the user presses in conjunction with a key equivalent to activate the shortcut.

## Declaration

```swift
var modifiers: EventModifiers
```

## Creating a shortcut

- **init(_:modifiers:)**: Creates a new keyboard shortcut with the given key equivalent and set of modifier keys.
- **key**: The key equivalent that the user presses in conjunction with any specified modifier keys to activate the shortcut.
