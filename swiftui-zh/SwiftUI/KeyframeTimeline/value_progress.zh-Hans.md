# value(progress:)

**Instance Method**

Returns the interpolated value at the given progress in the range zero to one.

## Declaration

```swift
func value(progress: Double) -> Value
```

## Getting an interpolated value

- **value(time:)**: Returns the interpolated value at the given time.
