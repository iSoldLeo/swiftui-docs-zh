# 主体

**关联类型**

表示此类型主体的关键帧类型。

## 声明

```swift
associatedtype Body : Keyframes
```

## 讨论

创建自定义关键帧类型时，Swift 会根据您对所需属性 [body-swift.property](body-swift_property.zh-Hans.md) 的实现来推断此类型。

## 创建关键帧

- **body**：构成关键帧的内容组合。

- **Value**：此关键帧类型所动画的值类型。

# Body

**Associated Type**

The type of keyframes representing the body of this type.

## Declaration

```swift
associatedtype Body : Keyframes
```

## Discussion

When you create a custom keyframes type, Swift infers this type from your implementation of the required [body-swift.property](body-swift_property.zh-Hans.md) property.

## Creating a keyframe

- **body**: The composition of content that comprise the keyframes.
- **Value**: The type of value animated by this keyframes type
