# init(_:)

**Initializer**

创建代表家庭活动应用程序的标签。

## 声明

```swift
init(_ applicationToken: ApplicationToken)
```

## 创建标签

- **init(_:image:)**：创建带有图标图像和由本地化字符串生成的标题的标签。
- **init(_:systemImage:)**：创建带有系统图标图像和由本地化字符串生成的标题的标签。
- **init(title:icon:)**：创建带有自定义标题和图标的标签。
- **init(_:image:)**：创建带有图标图像和由本地化字符串生成的标题的标签。

# init(_:)

**Initializer**

Creates a label representing a family activity application.

## Declaration

```swift
init(_ applicationToken: ApplicationToken)
```

## Creating a label

- **init(_:image:)**: Creates a label with an icon image and a title generated from a localized string.
- **init(_:systemImage:)**: Creates a label with a system icon image and a title generated from a localized string.
- **init(title:icon:)**: Creates a label with a custom title and icon.
- **init(_:image:)**: Creates a label with an icon image and a title generated from a localized string.
