# init(_:image:)

**Initializer**

创建带有图标图像的标签，标题由本地化字符串生成。

### 声明

```swift
nonisolated init(_ titleKey: LocalizedStringKey, image name: String)
```

## 参数

- **titleKey**：由本地化字符串生成的标题。

## 创建标签

- **init(_:systemImage:)**：创建带有系统图标图像和由本地化字符串生成的标题的标签。
- **init(title:icon:)**：创建带有自定义标题和图标的标签。
- **init(_:)**：创建代表家庭活动应用程序的标签。

# init(_:image:)

**Initializer**

Creates a label with an icon image and a title generated from a localized string.

## Declaration

```swift
nonisolated init(_ titleKey: LocalizedStringKey, image name: String)
```

## Parameters

- **titleKey**: A title generated from a localized string.

## Creating a label

- **init(_:systemImage:)**: Creates a label with a system icon image and a title generated from a localized string.
- **init(title:icon:)**: Creates a label with a custom title and icon.
- **init(_:)**: Creates a label representing a family activity application.
