# 标签样式配置

**Structure**

标签的属性。

## 声明

```swift
struct LabelStyleConfiguration
```

## 设置图标

- **icon**：标签项目的符号表示。
- **LabelStyleConfiguration.Icon**：标签的类型化图标视图。

## 设置标题

- **title**：对标签项目的描述。
- **LabelStyleConfiguration.Title**：标签的类型化标题视图。

## 显示文本的视图样式

- **labelStyle(_:)**：设置该视图中标签的样式。
- **LabelStyle**：为视图中的所有标签应用自定义外观的类型。
- **textFieldStyle(_:)**：设置该视图中文本字段的样式。
- **TextFieldStyle**：文本字段的外观和交互规范。
- **textEditorStyle(_:)**：设置该视图中文本编辑器的样式。
- **TextEditorStyle**：文本编辑器的外观和交互规范。
- **TextEditorStyleConfiguration**：文本编辑器的属性。

# LabelStyleConfiguration

**Structure**

The properties of a label.

## Declaration

```swift
struct LabelStyleConfiguration
```

## Setting the icon

- **icon**: A symbolic representation of the labeled item.
- **LabelStyleConfiguration.Icon**: A type-erased icon view of a label.

## Setting the title

- **title**: A description of the labeled item.
- **LabelStyleConfiguration.Title**: A type-erased title view of a label.

## Styling views that display text

- **labelStyle(_:)**: Sets the style for labels within this view.
- **LabelStyle**: A type that applies a custom appearance to all labels within a view.
- **textFieldStyle(_:)**: Sets the style for text fields within this view.
- **TextFieldStyle**: A specification for the appearance and interaction of a text field.
- **textEditorStyle(_:)**: Sets the style for text editors within this view.
- **TextEditorStyle**: A specification for the appearance and interaction of a text editor.
- **TextEditorStyleConfiguration**: The properties of a text editor.
