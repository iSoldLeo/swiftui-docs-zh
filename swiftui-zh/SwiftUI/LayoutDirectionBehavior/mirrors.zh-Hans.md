# 镜像

**类型属性**

当布局方向为从右到左时镜像的行为。

## 声明

```swift
static var mirrors: LayoutDirectionBehavior { get }
```

## 获取行为

- **LayoutDirectionBehavior.fixed**：当布局方向发生变化时无法镜像的行为。
- **LayoutDirectionBehavior.mirrors(in:)**：当布局方向具有指定值时可进行镜像的行为。

# mirrors

**Type Property**

A behavior that mirrors when the layout direction is right-to-left.

## Declaration

```swift
static var mirrors: LayoutDirectionBehavior { get }
```

## Getting behaviors

- **LayoutDirectionBehavior.fixed**: A behavior that doesn’t mirror when the layout direction changes.
- **LayoutDirectionBehavior.mirrors(in:)**: A behavior that mirrors when the layout direction has the specified value.
