# 值

**关联类型**

键值的类型。

## 声明

```swift
associatedtype Value
```

## 说明

Swift 通常会根据你对 [defaultValue](defaultValue.zh-Hans.md) 属性的实现来推断此类型，因此你无需显式定义它。

## 提供默认值

- **defaultValue**：键的默认值。

# Value

**Associated Type**

The type of the key’s value.

## Declaration

```swift
associatedtype Value
```

## Discussion

Swift typically infers this type from your implementation of the [defaultValue](defaultValue.zh-Hans.md) property, so you don’t have to define it explicitly.

## Providing a default value

- **defaultValue**: The default value of the key.
