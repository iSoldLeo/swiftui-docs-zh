# init(stops:startPoint:endPoint:)

**Initializer**

Creates a linear gradient from a collection of color stops.

## Declaration

```swift
init(stops: [Gradient.Stop], startPoint: UnitPoint, endPoint: UnitPoint)
```

## Creating a linear gradient

- **init(gradient:startPoint:endPoint:)**: Creates a linear gradient from a base gradient.
- **init(colors:startPoint:endPoint:)**: Creates a linear gradient from a collection of colors.
