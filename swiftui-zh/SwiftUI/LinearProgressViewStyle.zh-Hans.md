# LinearProgressViewStyle

**Structure**

A progress view that visually indicates its progress using a horizontal bar.

## Declaration

```swift
struct LinearProgressViewStyle
```

## Overview

Use [linear](ProgressViewStyle/linear.zh-Hans.md) to construct this style.

## Creating the progress view style

- **init()**: Creates a linear progress view style.

## Deprecated initializers

- **init(tint:)**: Creates a linear progress view style with a tint color.

## Supporting types

- **DefaultProgressViewStyle**: The default progress view style in the current context of the view being styled.
- **CircularProgressViewStyle**: A progress view that uses a circular gauge to indicate the partial completion of an activity.

## Conforms To

- ProgressViewStyle
