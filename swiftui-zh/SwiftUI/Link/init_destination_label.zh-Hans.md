# init(destination:label:)

**Initializer**

创建一个控件，包含一个 URL 和一个标签，用于导航到指定的 URL。

## 声明

```swift
@MainActor @preconcurrency init(destination: URL, @ViewBuilder label: () -> Label)
```

## 参数

- **destination**：链接的 URL。

- **label**：描述 URL 目标位置的视图。

## 创建链接

- **init(_:destination:)**：创建一个控件，包含一个 URL 和一个标题键，用于导航到指定的 URL。

# init(destination:label:)

**Initializer**

Creates a control, consisting of a URL and a label, used to navigate to the given URL.

## Declaration

```swift
@MainActor @preconcurrency init(destination: URL, @ViewBuilder label: () -> Label)
```

## Parameters

- **destination**: The URL for the link.
- **label**: A view that describes the destination of URL.

## Creating a link

- **init(_:destination:)**: Creates a control, consisting of a URL and a title key, used to navigate to a URL.
