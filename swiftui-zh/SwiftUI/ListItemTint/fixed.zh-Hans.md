# fixed(_:)

**类型方法**

显式着色颜色。

## 声明

```swift
static func fixed(_ tint: Color) -> ListItemTint
```

## 参数

- **tint**：用于着色内容的颜色。

## 讨论

系统不会覆盖此着色效果。

## 获取列表项着色选项

- **monochrome**：标准灰度着色效果。

- **preferred(_:)**：系统可以覆盖的显式着色颜色。

# fixed(_:)

**Type Method**

An explicit tint color.

## Declaration

```swift
static func fixed(_ tint: Color) -> ListItemTint
```

## Parameters

- **tint**: The color to use to tint the content.

## Discussion

The system doesn’t override this tint effect.

## Getting list item tint options

- **monochrome**: A standard grayscale tint effect.
- **preferred(_:)**: An explicit tint color that the system can override.
