# monochrome

**Type Property**

A standard grayscale tint effect.

## Declaration

```swift
static let monochrome: ListItemTint
```

## Discussion

The system doesn’t override monochrome tints.

## Getting list item tint options

- **fixed(_:)**: An explicit tint color.
- **preferred(_:)**: An explicit tint color that the system can override.
