# custom(_:)

**类型方法**

创建自定义间距值。

## 声明

```swift
static func custom(_ spacing: CGFloat) -> ListSectionSpacing
```

## 参数

- **spacing**：要使用的间距大小。

## 获取节间距

- **default**：节之间的默认间距

- **compact**：节之间的紧凑间距

# custom(_:)

**Type Method**

Creates a custom spacing value.

## Declaration

```swift
static func custom(_ spacing: CGFloat) -> ListSectionSpacing
```

## Parameters

- **spacing**: The amount of spacing to use.

## Getting section spacing

- **default**: The default spacing between sections
- **compact**: Compact spacing between sections
