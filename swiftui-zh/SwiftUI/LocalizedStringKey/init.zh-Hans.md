# init(_:)

**Initializer**

根据给定的字符串值创建本地化字符串键。

## 声明

```swift
init(_ value: String)
```

## 参数

- **value**：用作本地化键的字符串。

## 根据字面值创建键

- **init(stringLiteral:)**：根据给定的字符串字面值创建本地化字符串键。

# init(_:)

**Initializer**

Creates a localized string key from the given string value.

## Declaration

```swift
init(_ value: String)
```

## Parameters

- **value**: The string to use as a localization key.

## Creating a key from a literal value

- **init(stringLiteral:)**: Creates a localized string key from the given string literal.
