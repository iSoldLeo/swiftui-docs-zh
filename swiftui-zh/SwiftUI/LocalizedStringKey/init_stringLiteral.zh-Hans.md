# init(stringLiteral:)

**Initializer**

Creates a localized string key from the given string literal.

## Declaration

```swift
init(stringLiteral value: String)
```

## Parameters

- **value**: The string literal to use as a localization key.

## Creating a key from a literal value

- **init(_:)**: Creates a localized string key from the given string value.
