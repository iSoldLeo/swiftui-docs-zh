# init(minimumScaleDelta:)

**Initializer**

以给定的最小 delta 值创建放大手势。

## 声明

```swift
init(minimumScaleDelta: CGFloat = 0.01)
```

## 参数

- **minimumScaleDelta**：手势开始前所需的最小刻度偏差。

## 创建手势

- **minimumScaleDelta**：手势开始前所需的最小偏移量。

# init(minimumScaleDelta:)

**Initializer**

Creates a magnify gesture with a given minimum delta for the gesture to start.

## Declaration

```swift
init(minimumScaleDelta: CGFloat = 0.01)
```

## Parameters

- **minimumScaleDelta**: The minimum scale delta required before the gesture starts.

## Creating the gesture

- **minimumScaleDelta**: The minimum required delta before the gesture starts.
