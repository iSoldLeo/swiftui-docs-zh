# Manipulable.Event.Phase

**Enumeration**

Describes the phase of a manipulation gesture.

## Declaration

```swift
enum Phase
```

## Enumeration Cases

- **Manipulable.Event.Phase.active(_:)**: A person is manipulating the view.
- **Manipulable.Event.Phase.ended(_:)**: A person successfully ended manipulating the view.
- **Manipulable.Event.Phase.failed**: A person started manipulating the view but failed to successfully end the manipulation gesture.
