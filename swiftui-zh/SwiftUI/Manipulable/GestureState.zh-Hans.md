# Manipulable.GestureState

**Structure**

Describes the state of a manipulation gesture.

## Declaration

```swift
struct GestureState
```

## Overview





## Initializers

- **init(transform:)**: Creates a new manipulation gesture state with the given transform.

## Instance Properties

- **isActive**: The Boolean value that indicates whether a manipulation gesture is currently active.
- **transform**: The current 3D affine transform applied by an active manipulation gesture.

## Conforms To

- Sendable
- SendableMetatype
