# Manipulable.InputDevice.Kind

**Enumeration**

Describes the kind of an input device.

## Declaration

```swift
enum Kind
```

## Enumeration Cases

- **Manipulable.InputDevice.Kind.directPinch**: A pinching hand in close proximity to content.
- **Manipulable.InputDevice.Kind.indirectPinch**: An indirectly targeted pinching hand.
- **Manipulable.InputDevice.Kind.pointer**: A click-based, indirect input device.
