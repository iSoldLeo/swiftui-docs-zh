# 位置

**类型属性**

视图的位置，以窗口坐标为单位。

## 声明

```swift
static let position: MatchedGeometryProperties
```

## 匹配属性

- **frame**：同时具有 `position` 和 `size` 属性。
- **size**：视图的大小（以本地坐标表示）。

# position

**Type Property**

The view’s position, in window coordinates.

## Declaration

```swift
static let position: MatchedGeometryProperties
```

## Matching properties

- **frame**: Both the `position` and `size` properties.
- **size**: The view’s size, in local coordinates.
