# 大小

**类型属性**

视图的大小，以本地坐标表示。

## 声明

```swift
static let size: MatchedGeometryProperties
```

## 匹配属性

- **frame**：同时具有 `position` 和 `size` 属性。
- **position**：以窗口坐标表示的视图位置。

# size

**Type Property**

The view’s size, in local coordinates.

## Declaration

```swift
static let size: MatchedGeometryProperties
```

## Matching properties

- **frame**: Both the `position` and `size` properties.
- **position**: The view’s position, in window coordinates.
