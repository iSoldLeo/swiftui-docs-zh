# 已禁用

**类型属性**

执行操作后，绝不关闭已显示的菜单。

## 声明

```swift
static let disabled: MenuActionDismissBehavior
```

## 获取关闭行为

- **automatic**：使用适合当前上下文的关闭行为。

- **enabled**：执行操作后，始终关闭已显示的菜单。

# disabled

**Type Property**

Never dismiss the presented menu after performing an action.

## Declaration

```swift
static let disabled: MenuActionDismissBehavior
```

## Getting dismiss behaviors

- **automatic**: Use the a dismissal behavior that’s appropriate for the given context.
- **enabled**: Always dismiss the presented menu after performing an action.
