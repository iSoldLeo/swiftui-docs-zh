# automatic

**类型属性**

默认菜单栏额外样式。

## 声明

```swift
static var automatic: AutomaticMenuBarExtraStyle { get }
```

## 获取菜单栏额外样式

- **menu**：一种菜单栏额外样式，其内容渲染为一个可从菜单栏图标下拉的菜单。

- **window**：一种菜单栏额外样式，其内容渲染为一个类似弹出框的窗口。

# automatic

**Type Property**

The default menu bar extra style.

## Declaration

```swift
static var automatic: AutomaticMenuBarExtraStyle { get }
```

## Getting menu bar extra styles

- **menu**: A menu bar extra style that renders its contents as a menu that pulls down from the icon in the menu bar.
- **window**: A menu bar extra style that renders its contents in a popover-like window.
