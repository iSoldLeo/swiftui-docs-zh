# 菜单

**类型属性**

菜单栏额外样式，将其内容渲染为可从菜单栏图标下拉的菜单。

## 声明

```swift
static var menu: PullDownMenuBarExtraStyle { get }
```

## 获取菜单栏额外样式

- **automatic**：默认菜单栏额外样式。

- **window**：将其内容渲染为类似弹出窗口的菜单栏额外样式。

# menu

**Type Property**

A menu bar extra style that renders its contents as a menu that pulls down from the icon in the menu bar.

## Declaration

```swift
static var menu: PullDownMenuBarExtraStyle { get }
```

## Getting menu bar extra styles

- **automatic**: The default menu bar extra style.
- **window**: A menu bar extra style that renders its contents in a popover-like window.
