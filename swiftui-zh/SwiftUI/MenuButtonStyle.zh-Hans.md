# MenuButtonStyle

**Protocol**

菜单按钮外观和交互的自定义规范。

### 声明

```swift
protocol MenuButtonStyle
```

## 支持类型

- **BorderlessButtonMenuButtonStyle**：菜单按钮样式，表现为无边框按钮，没有视觉修饰。
- **BorderlessPullDownMenuButtonStyle**：表现为无边框下拉按钮的菜单按钮样式。
- **DefaultMenuButtonStyle**：默认菜单按钮样式。
- **PullDownMenuButtonStyle**：默认菜单按钮样式：显示为下拉按钮的菜单按钮样式。

## 菜单按钮样式

- **menuButtonStyle(_:)**：设置该视图中菜单按钮的样式。

## 符合类型

- 无边框按钮菜单按钮样式
- 无边框下拉菜单按钮样式
- 默认菜单按钮样式
- 下拉菜单按钮样式

# MenuButtonStyle

**Protocol**

A custom specification for the appearance and interaction of a menu button.

## Declaration

```swift
protocol MenuButtonStyle
```

## Supporting types

- **BorderlessButtonMenuButtonStyle**: A menu button style which manifests as a borderless button with no visual embelishments.
- **BorderlessPullDownMenuButtonStyle**: A menu button style which manifests as a borderless pull-down button.
- **DefaultMenuButtonStyle**: The default menu button style.
- **PullDownMenuButtonStyle**: A menu button style which manifests as a pull-down button.

## Styling a menu button

- **menuButtonStyle(_:)**: Sets the style for menu buttons within this view.

## Conforming Types

- BorderlessButtonMenuButtonStyle
- BorderlessPullDownMenuButtonStyle
- DefaultMenuButtonStyle
- PullDownMenuButtonStyle
