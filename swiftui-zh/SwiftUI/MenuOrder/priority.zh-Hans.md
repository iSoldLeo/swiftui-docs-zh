# priority

**Type Property**

Keep the first items closest to user’s interaction point.

## Declaration

```swift
static let priority: MenuOrder
```

## Getting menu orders

- **automatic**: The ordering of the menu chosen by the system for the current context.
- **fixed**: Order items from top to bottom.
