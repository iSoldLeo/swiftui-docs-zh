# 内容

**实例属性**

修改器转换为新视图或新视图修改器的内容。

## 声明

```swift
var content: Content
```

## 创建修改后的内容视图

- **init(content:modifier:)**：定义产生新视图或视图修改器所需的内容和修改器的结构。
- **modifier**：视图修改器。

# content

**Instance Property**

The content that the modifier transforms into a new view or new view modifier.

## Declaration

```swift
var content: Content
```

## Creating a modified content view

- **init(content:modifier:)**: A structure that defines the content and modifier needed to produce a new view or view modifier.
- **modifier**: The view modifier.
