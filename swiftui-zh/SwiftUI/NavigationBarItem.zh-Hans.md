- **NavigationBarItem.TitleDisplayMode**：用于显示导航栏标题的样式。

- **navigationBarBackButtonHidden(_:)**：隐藏视图的导航栏返回按钮。

# NavigationBarItem

**Structure**

A configuration for a navigation bar that represents a view at the top of a navigation stack.

## Declaration

```swift
struct NavigationBarItem
```

## Overview

Use one of the [TitleDisplayMode](NavigationBarItem/TitleDisplayMode.zh-Hans.md) values to configure a navigation bar title’s display mode with the [navigationBarTitleDisplayMode(_:)](View/navigationBarTitleDisplayMode.zh-Hans.md) view modifier.

## Setting a title display mode

- **NavigationBarItem.TitleDisplayMode**: A style for displaying the title of a navigation bar.

## Configuring the navigation bar

- **navigationBarBackButtonHidden(_:)**: Hides the navigation bar back button for the view.
- **navigationBarTitleDisplayMode(_:)**: Configures the title display mode for this view.

## Conforms To

- Sendable
- SendableMetatype
