# NavigationBarItem.TitleDisplayMode.large

**Case**

## 声明

```swift
case large
```

## 获取标题显示模式

- **NavigationBarItem.TitleDisplayMode.automatic**：继承上一个导航项目的显示模式。
- **NavigationBarItem.TitleDisplayMode.inline**：在导航栏的标准范围内显示标题。

# NavigationBarItem.TitleDisplayMode.large

**Case**

Display a large title within an expanded navigation bar.

## Declaration

```swift
case large
```

## Getting title display modes

- **NavigationBarItem.TitleDisplayMode.automatic**: Inherit the display mode from the previous navigation item.
- **NavigationBarItem.TitleDisplayMode.inline**: Display the title within the standard bounds of the navigation bar.
