# init(root:)

**Initializer**

Creates a navigation stack that manages its own navigation state.

## Declaration

```swift
@MainActor @preconcurrency init(@ViewBuilder root: () -> Root) where Data == NavigationPath
```

## Parameters

- **root**: The view to display when the stack is empty.
