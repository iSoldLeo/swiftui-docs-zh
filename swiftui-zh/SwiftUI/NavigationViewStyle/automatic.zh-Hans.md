# automatic

**Type 属性**

当前视图上下文中的默认导航视图样式。

## 声明

```swift
static var automatic: DefaultNavigationViewStyle { get }
```

## 获取内置导航视图样式

- **columns**：由一系列列视图表示的导航视图样式。

- **stack**：由视图堆栈表示的导航视图样式，每次只显示一个顶层视图。

# automatic

**Type Property**

The default navigation view style in the current context of the view being styled.

## Declaration

```swift
static var automatic: DefaultNavigationViewStyle { get }
```

## Getting built-in navigation view styles

- **columns**: A navigation view style represented by a series of views in columns.
- **stack**: A navigation view style represented by a view stack that only shows a single top view at a time.
