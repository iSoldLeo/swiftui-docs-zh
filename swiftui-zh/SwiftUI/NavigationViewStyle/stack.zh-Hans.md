# stack

**类型属性**

一种导航视图样式，由视图堆栈表示，每次只显示一个顶层视图。

## 声明

```swift
static var stack: StackNavigationViewStyle { get }
```

## 获取内置导航视图样式

- **automatic**：当前视图上下文中的默认导航视图样式。

- **columns**：一种导航视图样式，由一系列列视图表示。

# stack

**Type Property**

A navigation view style represented by a view stack that only shows a single top view at a time.

## Declaration

```swift
static var stack: StackNavigationViewStyle { get }
```

## Getting built-in navigation view styles

- **automatic**: The default navigation view style in the current context of the view being styled.
- **columns**: A navigation view style represented by a series of views in columns.
