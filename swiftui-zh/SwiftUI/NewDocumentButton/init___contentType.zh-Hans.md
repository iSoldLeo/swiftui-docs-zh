# init(_:contentType:)

**Initializer**

创建并打开新文档。

## 声明

```swift
init(_ label: Text? = nil, contentType: UTType? = nil)
```

## 参数

- **label**：用于按钮的标签。

# init(_:contentType:)

**Initializer**

Creates and opens new documents.

## Declaration

```swift
init(_ label: Text? = nil, contentType: UTType? = nil)
```

## Parameters

- **label**: A label to use for the button.
- **contentType**: An optional content type of the document to create.
