# init(initialValue:)

**Initializer**

创建具有初始值的观察对象。

## 声明

```swift
@MainActor @preconcurrency init(initialValue: ObjectType)
```

## 参数

- **initialValue**：初始值。

## 讨论

此初始化函数与 [init(wrappedValue:)](init_wrappedValue.zh-Hans.md) 初始化函数具有相同的行为。更多信息请参见该初始化程序。

## 创建观察对象

- **init(wrappedValue:)**：创建一个具有初始包装值的观察对象。

# init(initialValue:)

**Initializer**

Creates an observed object with an initial value.

## Declaration

```swift
@MainActor @preconcurrency init(initialValue: ObjectType)
```

## Parameters

- **initialValue**: An initial value.

## Discussion

This initializer has the same behavior as the [init(wrappedValue:)](init_wrappedValue.zh-Hans.md) initializer. See that initializer for more information.

## Creating an observed object

- **init(wrappedValue:)**: Creates an observed object with an initial wrapped value.
