# OpenImmersiveSpaceAction.Result.error

**Case**

系统无法打开沉浸式空间。

## 声明

```swift
case error
```

## 获取结果

- **OpenImmersiveSpaceAction.Result.opened**：沉浸式空间打开。
- **OpenImmersiveSpaceAction.Result.userCancelled**：沉浸式空间没有打开，因为用户取消了操作。

# OpenImmersiveSpaceAction.Result.error

**Case**

The system was unable to open the immersive space.

## Declaration

```swift
case error
```

## Getting the result

- **OpenImmersiveSpaceAction.Result.opened**: The immersive space opened.
- **OpenImmersiveSpaceAction.Result.userCancelled**: The immersive space didn’t open because the user cancelled the operation.
