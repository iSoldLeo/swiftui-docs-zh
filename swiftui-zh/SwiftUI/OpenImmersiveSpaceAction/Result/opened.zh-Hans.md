# OpenImmersiveSpaceAction.Result.opened

**Case**

## 声明

```swift
case opened
```

## 获取结果

- **OpenImmersiveSpaceAction.Result.userCancelled**：由于用户取消了操作，沉浸式空间没有打开。
- **OpenImmersiveSpaceAction.Result.error**：系统无法打开沉浸式空间。

# OpenImmersiveSpaceAction.Result.opened

**Case**

The immersive space opened.

## Declaration

```swift
case opened
```

## Getting the result

- **OpenImmersiveSpaceAction.Result.userCancelled**: The immersive space didn’t open because the user cancelled the operation.
- **OpenImmersiveSpaceAction.Result.error**: The system was unable to open the immersive space.
