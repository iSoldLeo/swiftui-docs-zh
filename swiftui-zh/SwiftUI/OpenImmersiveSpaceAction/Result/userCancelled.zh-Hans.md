# OpenImmersiveSpaceAction.Result.userCancelled

**Case**

沉浸式空间没有打开，因为用户取消了操作。

## 声明

```swift
case userCancelled
```

## 获取结果

- **OpenImmersiveSpaceAction.Result.opened**：沉浸式空间打开。
- **OpenImmersiveSpaceAction.Result.error**：系统无法打开沉浸式空间。

# OpenImmersiveSpaceAction.Result.userCancelled

**Case**

The immersive space didn’t open because the user cancelled the operation.

## Declaration

```swift
case userCancelled
```

## Getting the result

- **OpenImmersiveSpaceAction.Result.opened**: The immersive space opened.
- **OpenImmersiveSpaceAction.Result.error**: The system was unable to open the immersive space.
