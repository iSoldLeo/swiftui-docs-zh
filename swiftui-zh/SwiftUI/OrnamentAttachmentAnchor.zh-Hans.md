# OrnamentAttachmentAnchor

**Structure**

An attachment anchor for an ornament.

## Declaration

```swift
struct OrnamentAttachmentAnchor
```

## Getting an anchor

- **scene(_:)**: The anchor point for the ornament expressed as a unit point relative to the scene.

## Type Methods

- **parent(_:)**: The anchor point for the ornament expressed as a 3D unit point relative to its parent.

## Creating an ornament

- **ornament(visibility:attachmentAnchor:contentAlignment:ornament:)**: Presents an ornament.

## Conforms To

- Sendable
- SendableMetatype
