# PageIndexViewStyle

**Structure**

一种索引视图样式，用于将页面索引视图置于内容之上。

```swift

```

- **init(backgroundDisplayMode:)**：创建页面索引视图样式。

# PageIndexViewStyle

**Structure**

An index view style that places a page index view over its content.

## Declaration

```swift
struct PageIndexViewStyle
```

## Overview

You can also use [page](IndexViewStyle/page.zh-Hans.md) to construct this style.

## Creating the control group style

- **init(backgroundDisplayMode:)**: Creates a page index view style.
- **PageIndexViewStyle.BackgroundDisplayMode**: The background style for the page index view.

## Conforms To

- IndexViewStyle
