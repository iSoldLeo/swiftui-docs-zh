# PageIndexViewStyle.BackgroundDisplayMode

**Structure**

页面索引视图的背景样式。

## 声明

```swift
struct BackgroundDisplayMode
```

## 获取显示模式

- **automatic**：背景将使用平台的默认值。

- **always**：背景始终显示在页面索引视图的后面。

- **interactive**：背景仅在与索引视图交互时显示。

- **never**：背景永远不会显示在页面索引视图的后面。

## 创建控制组样式

- **init(backgroundDisplayMode:)**：创建页面索引视图样式。

# PageIndexViewStyle.BackgroundDisplayMode

**Structure**

The background style for the page index view.

## Declaration

```swift
struct BackgroundDisplayMode
```

## Getting the display modes

- **automatic**: Background will use the default for the platform.
- **always**: Background is always displayed behind the page index view.
- **interactive**: Background is only shown while the index view is interacted with.
- **never**: Background is never displayed behind the page index view.

## Creating the control group style

- **init(backgroundDisplayMode:)**: Creates a page index view style.

## Conforms To

- Sendable
- SendableMetatype
