# automatic

**类型属性**

背景将使用平台的默认值。

## 声明

```swift
static let automatic: PageIndexViewStyle.BackgroundDisplayMode
```

## 获取显示模式

- **always**：背景始终显示在页面索引视图的后面。

- **interactive**：背景仅在与索引视图交互时显示。

- **never**：背景始终不显示在页面索引视图的后面。

# automatic

**Type Property**

Background will use the default for the platform.

## Declaration

```swift
static let automatic: PageIndexViewStyle.BackgroundDisplayMode
```

## Getting the display modes

- **always**: Background is always displayed behind the page index view.
- **interactive**: Background is only shown while the index view is interacted with.
- **never**: Background is never displayed behind the page index view.
