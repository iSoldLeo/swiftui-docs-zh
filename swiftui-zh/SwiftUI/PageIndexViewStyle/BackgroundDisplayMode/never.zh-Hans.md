# never

**类型属性**

页面索引视图的背景永远不会显示。

## 声明

```swift
static let never: PageIndexViewStyle.BackgroundDisplayMode
```

## 获取显示模式

- **automatic**：背景将使用平台的默认设置。

- **always**：背景始终显示在页面索引视图的后面。

- **interactive**：背景仅在与索引视图交互时显示。

# never

**Type Property**

Background is never displayed behind the page index view.

## Declaration

```swift
static let never: PageIndexViewStyle.BackgroundDisplayMode
```

## Getting the display modes

- **automatic**: Background will use the default for the platform.
- **always**: Background is always displayed behind the page index view.
- **interactive**: Background is only shown while the index view is interacted with.
