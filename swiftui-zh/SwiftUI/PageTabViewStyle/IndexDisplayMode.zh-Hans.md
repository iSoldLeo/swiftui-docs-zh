# PageTabViewStyle.IndexDisplayMode

**Structure**

用于显示页面索引视图的样式

## 声明

```swift
struct IndexDisplayMode
```

## 获取模式

- **always**：无论页数多少，始终显示索引视图
- **automatic**：当有多个页面时显示索引视图
- **never**：从不显示索引视图

## 创建页面选项卡视图样式

- **init(indexDisplayMode:)**：创建新的`PageTabViewStyle`，采用索引显示模式

## 符合

- 可发送
- 可发送元数据类型

# PageTabViewStyle.IndexDisplayMode

**Structure**

A style for displaying the page index view

## Declaration

```swift
struct IndexDisplayMode
```

## Getting the modes

- **always**: Always display an index view regardless of page count
- **automatic**: Displays an index view when there are more than one page
- **never**: Never display an index view

## Creating a page tab view style

- **init(indexDisplayMode:)**: Creates a new `PageTabViewStyle` with an index display mode

## Conforms To

- Sendable
- SendableMetatype
