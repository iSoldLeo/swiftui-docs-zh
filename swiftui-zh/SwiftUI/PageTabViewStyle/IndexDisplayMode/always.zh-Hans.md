# 始终

**类型属性**

无论页数多少，始终显示索引视图

## 声明

```swift
static let always: PageTabViewStyle.IndexDisplayMode
```

## 获取模式

- **automatic**：当有多个页面时显示索引视图
- **never**：从不显示索引视图

# always

**Type Property**

Always display an index view regardless of page count

## Declaration

```swift
static let always: PageTabViewStyle.IndexDisplayMode
```

## Getting the modes

- **automatic**: Displays an index view when there are more than one page
- **never**: Never display an index view
