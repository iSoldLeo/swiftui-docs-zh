# 自动

**类型属性**

当有多个页面时显示索引视图

## 声明

```swift
static let automatic: PageTabViewStyle.IndexDisplayMode
```

## 获取模式

- **always**：无论页数多少，始终显示索引视图
- **never**：从不显示索引视图

# automatic

**Type Property**

Displays an index view when there are more than one page

## Declaration

```swift
static let automatic: PageTabViewStyle.IndexDisplayMode
```

## Getting the modes

- **always**: Always display an index view regardless of page count
- **never**: Never display an index view
