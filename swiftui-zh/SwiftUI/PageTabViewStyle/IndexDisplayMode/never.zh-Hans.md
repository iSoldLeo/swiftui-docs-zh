# 从不

**类型属性**

从不显示索引视图

## 声明

```swift
static let never: PageTabViewStyle.IndexDisplayMode
```

## 获取模式

- **always**：无论页数多少，始终显示索引视图
- **automatic**：当有一个以上页面时显示索引视图

# never

**Type Property**

Never display an index view

## Declaration

```swift
static let never: PageTabViewStyle.IndexDisplayMode
```

## Getting the modes

- **always**: Always display an index view regardless of page count
- **automatic**: Displays an index view when there are more than one page
