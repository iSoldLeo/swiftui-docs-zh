# azimuth

**实例属性**

表示 Apple Pencil 悬停时的方位角。

## 声明

```swift
let azimuth: Angle
```

## 讨论

当 Apple Pencil 指向视图的 x 轴正方向时，此值为 `.zero`。

# azimuth

**Instance Property**

A value that represents the azimuth angle of a hovering Apple Pencil.

## Declaration

```swift
let azimuth: Angle
```

## Discussion

This value is `.zero` when the Apple Pencil points along the view’s positive x-axis.
