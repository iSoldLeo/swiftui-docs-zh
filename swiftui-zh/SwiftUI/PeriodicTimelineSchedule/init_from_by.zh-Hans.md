# init(from:by:)

**Initializer**

## 声明

```swift
init(from startDate: Date, by interval: TimeInterval)
```

## 参数

- **startDate**：序列的起始日期。

- **interval**：连续序列条目之间的时间间隔。

## 说明

# init(from:by:)

**Initializer**

Creates a periodic update schedule.

## Declaration

```swift
init(from startDate: Date, by interval: TimeInterval)
```

## Parameters

- **startDate**: The date on which to start the sequence.
- **interval**: The time interval between successive sequence entries.

## Discussion

Use the [entries(from:mode:)](entries_from_mode.zh-Hans.md) method to get the sequence of dates.
