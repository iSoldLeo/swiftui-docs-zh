# sectionFooters

**Type Property**

The footer view of each `Section` will be pinned.

## Declaration

```swift
static let sectionFooters: PinnedScrollableViews
```

## Getting scrollable view types

- **sectionHeaders**: The header view of each `Section` will be pinned.
