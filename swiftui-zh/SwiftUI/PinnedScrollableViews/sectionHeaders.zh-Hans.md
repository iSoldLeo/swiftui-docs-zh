# sectionHeaders

**Type Property**

The header view of each `Section` will be pinned.

## Declaration

```swift
static let sectionHeaders: PinnedScrollableViews
```

## Getting scrollable view types

- **sectionFooters**: The footer view of each `Section` will be pinned.
