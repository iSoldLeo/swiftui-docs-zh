**类型属性**

# columnResize

**Type Property**

The pointer style for resizing a column, or vertical division, in either direction.

## Declaration

```swift
static let columnResize: PointerStyle
```

## Discussion

You may apply this pointer style to a single view or a view hierarchy using the [pointerStyle(_:)](../View/pointerStyle.zh-Hans.md) modifier.
