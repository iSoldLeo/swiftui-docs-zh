**类型属性**

# rowResize

**Type Property**

The pointer style for resizing a row, or horizontal division, in either direction.

## Declaration

```swift
static let rowResize: PointerStyle
```

## Discussion

You may apply this pointer style to a single view or a view hierarchy using the [pointerStyle(_:)](../View/pointerStyle.zh-Hans.md) modifier.
