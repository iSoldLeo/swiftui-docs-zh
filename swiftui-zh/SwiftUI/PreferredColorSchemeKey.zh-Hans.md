# PreferredColorSchemeKey

**Structure**

A key for specifying the preferred color scheme.

## Declaration

```swift
struct PreferredColorSchemeKey
```

## Overview

Don’t use this key directly. Instead, set a preferred color scheme for a view using the [preferredColorScheme(_:)](View/preferredColorScheme.zh-Hans.md) view modifier. Get the current color scheme for a view by accessing the [colorScheme](EnvironmentValues/colorScheme.zh-Hans.md) value.

## Conforms To

- PreferenceKey
