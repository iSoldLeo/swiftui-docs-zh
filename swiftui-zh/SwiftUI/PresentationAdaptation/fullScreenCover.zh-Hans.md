# 全屏封面

**类型属性**

在调整尺寸类别时，首选全屏封面外观。

## 声明

```swift
static var fullScreenCover: PresentationAdaptation { get }
```

## 获取适应战略

- **automatic**：使用默认的演示适配。
- **none**：如果可能，不要对尺寸类别进行适配。
- **popover**：为尺寸类别适配时，首选弹出式外观。
- **sheet**：在适应尺寸类别时，首选工作表外观。

# fullScreenCover

**Type Property**

Prefer a full-screen-cover appearance when adapting for size classes.

## Declaration

```swift
static var fullScreenCover: PresentationAdaptation { get }
```

## Getting adaptation strategies

- **automatic**: Use the default presentation adaptation.
- **none**: Don’t adapt for the size class, if possible.
- **popover**: Prefer a popover appearance when adapting for size classes.
- **sheet**: Prefer a sheet appearance when adapting for size classes.
