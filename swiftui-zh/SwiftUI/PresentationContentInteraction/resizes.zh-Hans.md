# 调整大小

**类型属性**

一种行为，用于在轻扫时优先调整演示文稿的大小，而不是滚动演示文稿的内容。

### 声明

```swift
static var resizes: PresentationContentInteraction { get }
```

## 获取交互行为

- **automatic**：演示文稿的默认轻扫行为。
- **scrolls**：在轻扫时优先滚动演示文稿内容而不是调整演示文稿大小的行为。

# resizes

**Type Property**

A behavior that prioritizes resizing a presentation when swiping, rather than scrolling the content of the presentation.

## Declaration

```swift
static var resizes: PresentationContentInteraction { get }
```

## Getting interaction behaviors

- **automatic**: The default swipe behavior for the presentation.
- **scrolls**: A behavior that prioritizes scrolling the content of a presentation when swiping, rather than resizing the presentation.
