# 页面

**类型属性**

页面大小大致相当于一张纸的大小，适合信息性或排版性内容。

## 声明

```swift
static var page: PagePresentationSizing { get }
```

## 获取内置的演示大小

- **automatic**：默认的演示大小，适合平台。

- **fitted**：演示大小取决于内容的理想大小。

- **form**：适用于表单，宽度略小于`.page`

# page

**Type Property**

The size is roughly the size of a page of paper, appropriate for informational or compositional content.

## Declaration

```swift
static var page: PagePresentationSizing { get }
```

## Getting built-in presentation size

- **automatic**: The default presentation sizing, appropriate for the platform.
- **fitted**: The presentation sizing is dictated by the ideal size of the content
- **form**: The size is appropriate for forms and slightly less wide than`.page`
