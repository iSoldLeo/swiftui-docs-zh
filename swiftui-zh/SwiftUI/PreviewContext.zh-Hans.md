# PreviewContext

**Protocol**

A context type for use with a preview.

## Declaration

```swift
protocol PreviewContext
```

## Accessing a preview context

- **subscript(_:)**: Returns the context’s value for a key, or a the key’s default value if the context doesn’t define a value for the key.

## Setting a context

- **previewContext(_:)**: Declares a context for the preview.
- **PreviewContextKey**: A key type for a preview context.
