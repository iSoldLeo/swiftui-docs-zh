# PreviewContextKey

**Protocol**

A key type for a preview context.

## Declaration

```swift
protocol PreviewContextKey
```

## Overview

The default value is `nil`.

## Setting a default

- **defaultValue**: The default value of the key.
- **Value**: The type of the value returned by the key.

## Setting a context

- **previewContext(_:)**: Declares a context for the preview.
- **PreviewContext**: A context type for use with a preview.
