# Previews

**Associated Type**

The type to preview.

## Declaration

```swift
associatedtype Previews : View
```

## Discussion

When you create a preview, Swift infers this type from your implementation of the required [previews-swift.type.property](previews-swift_type_property.zh-Hans.md) property.

## Creating a preview

- **previews**: A collection of views to preview.
