# height

**实例属性**

建议的垂直高度，以磅为单位。

## 声明

```swift
var height: CGFloat?
```

## 说明

`nil` 的值表示未指定高度建议，视图会将其解释为应使用其理想高度。

## 获取建议的高度

- **width**：建议的水平高度，以磅为单位。

# height

**Instance Property**

The proposed vertical size measured in points.

## Declaration

```swift
var height: CGFloat?
```

## Discussion

A value of `nil` represents an unspecified height proposal, which a view interprets to mean that it should use its ideal height.

## Getting the proposal’s dimensions

- **width**: The proposed horizontal size measured in points.
