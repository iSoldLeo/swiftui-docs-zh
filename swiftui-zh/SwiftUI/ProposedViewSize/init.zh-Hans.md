# init(_:)

**Initializer**

根据指定尺寸创建新的建议尺寸。

## 声明

```swift
init(_ size: CGSize)
```

## 参数

- **size**：以磅为单位测量的建议尺寸。

## 创建自定义尺寸建议

- **init(width:height:)**：使用指定的宽度和高度创建新的建议尺寸。

# init(_:)

**Initializer**

Creates a new proposed size from a specified size.

## Declaration

```swift
init(_ size: CGSize)
```

## Parameters

- **size**: A proposed size with dimensions measured in points.

## Creating a custom size proposal

- **init(width:height:)**: Creates a new proposed size using the specified width and height.
