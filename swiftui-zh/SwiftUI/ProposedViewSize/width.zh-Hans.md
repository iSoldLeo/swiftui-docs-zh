# width

**实例属性**

建议的水平尺寸，以磅为单位。

## 声明

```swift
var width: CGFloat?
```

## 说明

`nil` 的值表示未指定宽度建议，视图会将其解释为应使用其理想宽度。

## 获取建议的尺寸

- **height**：建议的垂直尺寸，以磅为单位。

# width

**Instance Property**

The proposed horizontal size measured in points.

## Declaration

```swift
var width: CGFloat?
```

## Discussion

A value of `nil` represents an unspecified width proposal, which a view interprets to mean that it should use its ideal width.

## Getting the proposal’s dimensions

- **height**: The proposed vertical size measured in points.
