**Structure**

菜单栏附加样式，可将其内容渲染为从菜单栏图标向下拉动的菜单。

## 声明

```swift
struct PullDownMenuBarExtraStyle
```

## 概览

使用 [menu](MenuBarExtraStyle/menu.zh-Hans.md) 构建此样式。

## 创建菜单栏额外样式

- **init()**：创建下拉菜单栏额外样式。

## 支持类型

- **AutomaticMenuBarExtraStyle**：默认菜单栏额外样式。您也可以使用 [automatic](MenuBarExtraStyle/automatic.zh-Hans.md) 构建此样式。
- **WindowMenuBarExtraStyle**：菜单栏额外样式，可在弹出窗口中显示其内容。

# PullDownMenuBarExtraStyle

**Structure**

A menu bar extra style that renders its contents as a menu that pulls down from the icon in the menu bar.

## Declaration

```swift
struct PullDownMenuBarExtraStyle
```

## Overview

Use [menu](MenuBarExtraStyle/menu.zh-Hans.md) to construct this style.

## Creating the menu bar extra style

- **init()**: Creates a pull down menu bar extra style.

## Supporting types

- **AutomaticMenuBarExtraStyle**: The default menu bar extra style. You can also use [automatic](MenuBarExtraStyle/automatic.zh-Hans.md) to construct this style.
- **WindowMenuBarExtraStyle**: A menu bar extra style that renders its contents in a popover-like window.

## Conforms To

- MenuBarExtraStyle
