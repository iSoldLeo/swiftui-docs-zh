# init(colors:center:startRadius:endRadius:)

**Initializer**

# init(colors:center:startRadius:endRadius:)

**Initializer**

Creates a radial gradient from a collection of colors.

## Declaration

```swift
init(colors: [Color], center: UnitPoint, startRadius: CGFloat, endRadius: CGFloat)
```

## Creating a radial gradient

- **init(gradient:center:startRadius:endRadius:)**: Creates a radial gradient from a base gradient.
- **init(stops:center:startRadius:endRadius:)**: Creates a radial gradient from a collection of color stops.
