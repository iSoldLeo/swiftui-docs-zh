# init(stops:center:startRadius:endRadius:)

**Initializer**

Creates a radial gradient from a collection of color stops.

## Declaration

```swift
init(stops: [Gradient.Stop], center: UnitPoint, startRadius: CGFloat, endRadius: CGFloat)
```

## Creating a radial gradient

- **init(gradient:center:startRadius:endRadius:)**: Creates a radial gradient from a base gradient.
- **init(colors:center:startRadius:endRadius:)**: Creates a radial gradient from a collection of colors.
