# 底层领导

**实例属性**

底部引导角的半径。

## 声明

```swift
var bottomLeading: CGFloat { get set }
```

## 获取特定角的值

- **topLeading**：顶角的半径。
- **topTrailing**：倾斜顶角的半径。
- **bottomTrailing**：尾部底角的半径。

# bottomLeading

**Instance Property**

The radius of the bottom-leading corner.

## Declaration

```swift
var bottomLeading: CGFloat { get set }
```

## Getting values for specific corners

- **topLeading**: The radius of the top-leading corner.
- **topTrailing**: The radius of the top-trailing corner.
- **bottomTrailing**: The radius of the bottom-trailing corner.
