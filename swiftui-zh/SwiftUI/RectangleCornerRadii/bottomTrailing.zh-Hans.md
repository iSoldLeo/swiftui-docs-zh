# BottomTrailing

**实例属性**

底部拐角的半径。

## 声明

```swift
var bottomTrailing: CGFloat { get set }
```

## 获取特定角的值

- **topLeading**：顶角的半径。
- **topTrailing**：倾斜顶角的半径。
- **bottomLeading**：下引角的半径。

# bottomTrailing

**Instance Property**

The radius of the bottom-trailing corner.

## Declaration

```swift
var bottomTrailing: CGFloat { get set }
```

## Getting values for specific corners

- **topLeading**: The radius of the top-leading corner.
- **topTrailing**: The radius of the top-trailing corner.
- **bottomLeading**: The radius of the bottom-leading corner.
