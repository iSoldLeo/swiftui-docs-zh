# topTrailing

**实例属性**

顶部倾斜角的半径。

## 声明

```swift
var topTrailing: CGFloat { get set }
```

## 获取特定角的值

- **topLeading**：顶角的半径。
- **bottomLeading**：下导角半径。
- **bottomTrailing**：倾斜底角的半径。

# topTrailing

**Instance Property**

The radius of the top-trailing corner.

## Declaration

```swift
var topTrailing: CGFloat { get set }
```

## Getting values for specific corners

- **topLeading**: The radius of the top-leading corner.
- **bottomLeading**: The radius of the bottom-leading corner.
- **bottomTrailing**: The radius of the bottom-trailing corner.
