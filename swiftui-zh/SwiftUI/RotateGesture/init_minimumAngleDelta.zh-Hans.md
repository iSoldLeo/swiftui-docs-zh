# init(minimumAngleDelta:)

**Initializer**

创建旋转手势，并设定手势开始时的最小三角角。

## 声明

```swift
init(minimumAngleDelta: Angle = .degrees(1))
```

## 参数

- **minimumAngleDelta**：手势开始前所需的最小偏差。默认值为一度角。

## 创建手势

- **minimumAngleDelta**：手势成功前所需的最小偏差。

# init(minimumAngleDelta:)

**Initializer**

Creates a rotation gesture with a minimum delta for the gesture to start.

## Declaration

```swift
init(minimumAngleDelta: Angle = .degrees(1))
```

## Parameters

- **minimumAngleDelta**: The minimum delta required before the gesture starts. The default value is a one-degree angle.

## Creating the gesture

- **minimumAngleDelta**: The minimum delta required before the gesture succeeds.
