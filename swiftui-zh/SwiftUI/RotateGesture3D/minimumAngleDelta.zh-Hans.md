# minimumAngleDelta

**实例属性**

手势激活前的最小角度增量。

## 声明

```swift
var minimumAngleDelta: Angle
```

## 创建手势

- **init(constrainedToAxis:minimumAngleDelta:)**：创建一个旋转手势，并指定手势开始的最小角度增量和用于约束旋转测量的轴。

- **constrainedAxis**：旋转围绕的轴。

# minimumAngleDelta

**Instance Property**

The minimum angle delta before the gesture becomes active.

## Declaration

```swift
var minimumAngleDelta: Angle
```

## Creating the gesture

- **init(constrainedToAxis:minimumAngleDelta:)**: Creates a rotation gesture with a minimum delta for the gesture to start and axis to constrain measurement of rotation.
- **constrainedAxis**: An axis around which the rotation is constrained.
