# init(topLeading:topTrailing:bottomLeading:bottomTrailing:)

**Initializer**

用每个角的样式创建角样式。

## 声明

```swift
init(topLeading: Edge.Corner.Style = 0, topTrailing: Edge.Corner.Style = 0, bottomLeading: Edge.Corner.Style = 0, bottomTrailing: Edge.Corner.Style = 0)
```

## 参数

- **topLeading**：顶角样式。
- **topTrailing**：顶部尾角样式。
- **bottomLeading**：下引角样式。
- **bottomTrailing**：底部拖尾角样式。

# init(topLeading:topTrailing:bottomLeading:bottomTrailing:)

**Initializer**

Create corner styles with per-corner styles.

## Declaration

```swift
init(topLeading: Edge.Corner.Style = 0, topTrailing: Edge.Corner.Style = 0, bottomLeading: Edge.Corner.Style = 0, bottomTrailing: Edge.Corner.Style = 0)
```

## Parameters

- **topLeading**: Top leading corner style.
- **topTrailing**: Top trailing corner style.
- **bottomLeading**: Bottom leading corner style.
- **bottomTrailing**: Bottom trailing corner style.
