# 容器

**类型属性**

由设备和用户界面内的容器（包括顶部和底部栏等元素）定义的安全区域。

## 声明

```swift
static let container: SafeAreaRegions
```

## 获取安全区域

- **all**：所有安全区域。
- **keyboard**：与视图内容上显示的任何软件键盘的当前范围相匹配的安全区域。

# container

**Type Property**

The safe area defined by the device and containers within the user interface, including elements such as top and bottom bars.

## Declaration

```swift
static let container: SafeAreaRegions
```

## Getting safe area regions

- **all**: All safe area regions.
- **keyboard**: The safe area matching the current extent of any software keyboard displayed over the view content.
