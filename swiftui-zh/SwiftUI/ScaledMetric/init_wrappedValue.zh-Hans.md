# init(wrappedValue:)

**Initializer**

使用默认缩放比例，创建一个具有未缩放值的缩放指标。

## 声明

```swift
init(wrappedValue: Value)
```

## 创建指标

- **init(wrappedValue:relativeTo:)**：创建一个具有未缩放值和用于缩放的文本样式的缩放指标。

# init(wrappedValue:)

**Initializer**

Creates the scaled metric with an unscaled value using the default scaling.

## Declaration

```swift
init(wrappedValue: Value)
```

## Creating the metric

- **init(wrappedValue:relativeTo:)**: Creates the scaled metric with an unscaled value and a text style to scale relative to.
