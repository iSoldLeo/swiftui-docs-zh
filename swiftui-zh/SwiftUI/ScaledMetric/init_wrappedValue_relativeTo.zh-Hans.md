# init(wrappedValue:relativeTo:)

**Initializer**

创建一个缩放后的指标，该指标具有一个未缩放的值和一个用于缩放的文本样式。

## 声明

```swift
init(wrappedValue: Value, relativeTo textStyle: Font.TextStyle)
```

## 创建指标

- **init(wrappedValue:)**：使用默认缩放比例创建一个具有未缩放值的缩放后的指标。

# init(wrappedValue:relativeTo:)

**Initializer**

Creates the scaled metric with an unscaled value and a text style to scale relative to.

## Declaration

```swift
init(wrappedValue: Value, relativeTo textStyle: Font.TextStyle)
```

## Creating the metric

- **init(wrappedValue:)**: Creates the scaled metric with an unscaled value using the default scaling.
