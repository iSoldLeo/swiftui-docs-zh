# 始终

**类型属性**

滚动视图总是跳动。

## 声明

```swift
static var always: ScrollBounceBehavior { get }
```

## 讨论

无论内容的大小如何，滚动视图总是沿着指定的轴跳动。

## 弹跳行为

- **automatic**：自动行为。
- **basedOnSize**：自动行为：当滚动视图的内容大到需要滚动时，就会弹起。

# always

**Type Property**

The scrollable view always bounces.

## Declaration

```swift
static var always: ScrollBounceBehavior { get }
```

## Discussion

The scrollable view always bounces along the specified axis, regardless of the size of the content.

## Bounce behaviors

- **automatic**: The automatic behavior.
- **basedOnSize**: The scrollable view bounces when its content is large enough to require scrolling.
