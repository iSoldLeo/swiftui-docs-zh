# 滚动内容偏移调整行为（ScrollContentOffsetAdjustmentBehavior

**Structure**

一种类型，用于定义滚动视图的各种内容偏移调整行为。

## 声明

```swift
struct ScrollContentOffsetAdjustmentBehavior
```

## 类型属性

- **automatic**：自动行为。
- **disabled**：禁用行为。

## 管理内容可见性

- **scrollContentBackground(_:)**：指定该视图中可滚动视图的背景可见性。
- **scrollClipDisabled(_:)**：设置滚动视图是否将其内容剪切到其边界。

# ScrollContentOffsetAdjustmentBehavior

**Structure**

A type that defines the different kinds of content offset adjusting behaviors a scroll view can have.

## Declaration

```swift
struct ScrollContentOffsetAdjustmentBehavior
```

## Type Properties

- **automatic**: The automatic behavior.
- **disabled**: The disabled behavior.

## Managing content visibility

- **scrollContentBackground(_:)**: Specifies the visibility of the background for scrollable views within this view.
- **scrollClipDisabled(_:)**: Sets whether a scroll view clips its content to its bounds.
