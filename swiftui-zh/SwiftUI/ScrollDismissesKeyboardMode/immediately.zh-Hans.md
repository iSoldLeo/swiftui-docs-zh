# immediate

**类型属性**

滚动开始时立即关闭键盘。

## 声明

```swift
static var immediately: ScrollDismissesKeyboardMode { get }
```

## 获取模式

- **automatic**：根据上下文自动确定模式。

- **interactively**：允许用户在滚动操作过程中交互式地关闭键盘。

- **never**：滚动时键盘不会自动关闭。

# immediately

**Type Property**

Dismiss the keyboard as soon as scrolling starts.

## Declaration

```swift
static var immediately: ScrollDismissesKeyboardMode { get }
```

## Getting modes

- **automatic**: Determine the mode automatically based on the surrounding context.
- **interactively**: Enable people to interactively dismiss the keyboard as part of the scroll operation.
- **never**: Never dismiss the keyboard automatically as a result of scrolling.
