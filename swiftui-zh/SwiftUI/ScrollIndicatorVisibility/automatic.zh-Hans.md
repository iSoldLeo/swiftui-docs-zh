# automatic

**类型属性**

滚动指示器的可见性取决于接受可见性配置的组件的策略。

## 声明

```swift
static var automatic: ScrollIndicatorVisibility { get }
```

## 获取可见性

- **hidden**：隐藏滚动指示器。

- **never**：滚动指示器始终不可见。

- **visible**：显示滚动指示器。

# automatic

**Type Property**

Scroll indicator visibility depends on the policies of the component accepting the visibility configuration.

## Declaration

```swift
static var automatic: ScrollIndicatorVisibility { get }
```

## Getting visibilties

- **hidden**: Hide the scroll indicators.
- **never**: Scroll indicators should never be visible.
- **visible**: Show the scroll indicators.
