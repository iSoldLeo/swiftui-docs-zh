# 轴

**实例属性**

滚动视图的可滚动轴。

## 声明

```swift
var axes: Axis.Set { get set }
```

## 讨论

默认值为 [vertical](../Axis/vertical.zh-Hans.md)。

## 配置滚动视图

- **content**：滚动视图的内容。
- **showsIndicators**：表示滚动视图是否以适合平台的方式显示内容偏移的可滚动部分的值。

# axes

**Instance Property**

The scrollable axes of the scroll view.

## Declaration

```swift
var axes: Axis.Set { get set }
```

## Discussion

The default value is [vertical](../Axis/vertical.zh-Hans.md).

## Configuring a scroll view

- **content**: The scroll view’s content.
- **showsIndicators**: A value that indicates whether the scroll view displays the scrollable component of the content offset, in a way that’s suitable for the platform.
