# 显示指标

**实例属性**

表示滚动视图是否以适合平台的方式显示内容偏移的可滚动组件的值。

## 声明

```swift
var showsIndicators: Bool { get set }
```

## 讨论

默认值为 `true`。

## 配置滚动视图

- **content**：滚动视图的内容。
- **axes**：滚动视图的可滚动轴。

# showsIndicators

**Instance Property**

A value that indicates whether the scroll view displays the scrollable component of the content offset, in a way that’s suitable for the platform.

## Declaration

```swift
var showsIndicators: Bool { get set }
```

## Discussion

The default is `true`.

## Configuring a scroll view

- **content**: The scroll view’s content.
- **axes**: The scrollable axes of the scroll view.
