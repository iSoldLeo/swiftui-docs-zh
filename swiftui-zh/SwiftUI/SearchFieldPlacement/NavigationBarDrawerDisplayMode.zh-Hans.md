**Structure**

一种模式，用于确定何时显示导航栏中的搜索框。


- **automatic**：启用滚动时隐藏搜索框的功能。

# SearchFieldPlacement.NavigationBarDrawerDisplayMode

**Structure**

A mode that determines when to display a search field that appears in a navigation bar.

## Declaration

```swift
struct NavigationBarDrawerDisplayMode
```

## Getting display modes

- **always**: Always display the search field regardless of the scroll activity.
- **automatic**: Enable hiding the search field in response to scrolling.

## Conforms To

- Sendable
- SendableMetatype
