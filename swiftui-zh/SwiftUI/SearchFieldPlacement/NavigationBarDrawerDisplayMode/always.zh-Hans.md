# always

**类型属性**

无论滚动操作如何，始终显示搜索框。

## 声明

```swift
static let always: SearchFieldPlacement.NavigationBarDrawerDisplayMode
```

## 获取显示模式

- **automatic**：启用滚动时隐藏搜索框的功能。

# always

**Type Property**

Always display the search field regardless of the scroll activity.

## Declaration

```swift
static let always: SearchFieldPlacement.NavigationBarDrawerDisplayMode
```

## Getting display modes

- **automatic**: Enable hiding the search field in response to scrolling.
