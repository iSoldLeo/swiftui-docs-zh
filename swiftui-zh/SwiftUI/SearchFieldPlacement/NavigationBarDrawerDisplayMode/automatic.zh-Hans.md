# automatic

**Type 属性**

启用滚动时隐藏搜索框的功能。

## 声明

```swift
static let automatic: SearchFieldPlacement.NavigationBarDrawerDisplayMode
```

## 获取显示模式

- **always**：无论滚动情况如何，始终显示搜索框。

# automatic

**Type Property**

Enable hiding the search field in response to scrolling.

## Declaration

```swift
static let automatic: SearchFieldPlacement.NavigationBarDrawerDisplayMode
```

## Getting display modes

- **always**: Always display the search field regardless of the scroll activity.
