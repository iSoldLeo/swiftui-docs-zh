# SearchPresentationToolbarBehavior

**Structure**

定义工具栏在显示搜索时的行为的类型。

## 声明

```swift
struct SearchPresentationToolbarBehavior
```

## 概览

将此类型与 [searchPresentationToolbarBehavior(_:)](View/searchPresentationToolbarBehavior.zh-Hans.md) 结合使用

## 获取工具栏行为

- **automatic**：自动行为。
- **avoidHidingContent**：避免隐藏内容行为。

## 在搜索过程中显示工具栏内容

- **searchPresentationToolbarBehavior(_:)**：为该视图中的任何可搜索修改器配置搜索工具栏显示行为。

# SearchPresentationToolbarBehavior

**Structure**

A type that defines how the toolbar behaves when presenting search.

## Declaration

```swift
struct SearchPresentationToolbarBehavior
```

## Overview

Use this type in combination with the [searchPresentationToolbarBehavior(_:)](View/searchPresentationToolbarBehavior.zh-Hans.md)

## Getting toolbar behaviors

- **automatic**: The automatic behavior.
- **avoidHidingContent**: The avoid hiding content behavior.

## Displaying toolbar content during search

- **searchPresentationToolbarBehavior(_:)**: Configures the search toolbar presentation behavior for any searchable modifiers within this view.
