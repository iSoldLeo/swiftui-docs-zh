# onSearchPresentation

**类型属性**

此激活方式会在显示搜索框后显示搜索范围，并在取消搜索后隐藏搜索范围。

## 声明

```swift
static var onSearchPresentation: SearchScopeActivation { get }
```

## 获取搜索范围激活类型

- **automatic**：自动激活搜索范围栏。

- **onTextEntry**：此激活方式会在开始在搜索框中输入内容时显示搜索范围，并在取消搜索后隐藏搜索范围。

# onSearchPresentation

**Type Property**

An activation where the system shows search scopes after presenting search and hides search scopes after search cancellation.

## Declaration

```swift
static var onSearchPresentation: SearchScopeActivation { get }
```

## Getting search scope activiation types

- **automatic**: The automatic activation of the scope bar.
- **onTextEntry**: An activation where the system shows search scopes when typing begins in the search field and hides search scopes after search cancellation.
