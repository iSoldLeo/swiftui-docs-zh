# content

**类型属性**

搜索建议会显示在应用的主要内容中。

## 声明

```swift
static var content: SearchSuggestionsPlacement { get }
```

## 获取位置

- **automatic**：搜索建议会根据周围上下文自动显示。

- **menu**：搜索建议会显示在附加到搜索框的菜单中。

# content

**Type Property**

Search suggestions render in the main content of the app.

## Declaration

```swift
static var content: SearchSuggestionsPlacement { get }
```

## Getting placements

- **automatic**: Search suggestions render automatically based on the surrounding context.
- **menu**: Search suggestions render inside of a menu attached to the search field.
