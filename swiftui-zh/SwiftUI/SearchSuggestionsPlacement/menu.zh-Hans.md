# 菜单

**类型属性**

搜索建议会显示在附加到搜索框的菜单中。

## 声明

```swift
static var menu: SearchSuggestionsPlacement { get }
```

## 获取位置

- **automatic**：搜索建议会根据周围环境自动显示。

- **content**：搜索建议会显示在应用的主要内容中。

# menu

**Type Property**

Search suggestions render inside of a menu attached to the search field.

## Declaration

```swift
static var menu: SearchSuggestionsPlacement { get }
```

## Getting placements

- **automatic**: Search suggestions render automatically based on the surrounding context.
- **content**: Search suggestions render in the main content of the app.
