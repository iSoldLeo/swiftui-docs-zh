# SearchUnavailableContent

**Structure**

表示静态占位符搜索视图主体的结构。

## 声明

```swift
struct SearchUnavailableContent
```

## 概览

您不会直接创建此类型。SwiftUI 会在您创建搜索[ContentUnavailableView](ContentUnavailableView.zh-Hans.md) 时创建它。

## 获取内容类型

- **SearchUnavailableContent.Actions**：表示静态`ContentUnavailableView.search` 视图操作的视图。
- **SearchUnavailableContent.Description**：表示静态`ContentUnavailableView.search` 视图描述的视图。
- **SearchUnavailableContent.Label**：表示静态占位符搜索视图标签的视图。

# SearchUnavailableContent

**Structure**

A structure that represents the body of a static placeholder search view.

## Declaration

```swift
struct SearchUnavailableContent
```

## Overview

You don’t create this type directly. SwiftUI creates it when you build a search[ContentUnavailableView](ContentUnavailableView.zh-Hans.md).

## Getting content types

- **SearchUnavailableContent.Actions**: A view that represents the actions of a static `ContentUnavailableView.search` view.
- **SearchUnavailableContent.Description**: A view that represents the description of a static `ContentUnavailableView.search` view.
- **SearchUnavailableContent.Label**: A view that represents the label of a static placeholder search view.
