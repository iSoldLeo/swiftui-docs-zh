# SearchUnavailableContent.Description

**Structure**

A view that represents the description of a static `ContentUnavailableView.search` view.

## Declaration

```swift
@MainActor @preconcurrency struct Description
```

## Overview

You don’t create this type directly. SwiftUI creates it when you build a search``ContentUnavailableView`.

## Getting content types

- **SearchUnavailableContent.Actions**: A view that represents the actions of a static `ContentUnavailableView.search` view.
- **SearchUnavailableContent.Label**: A view that represents the label of a static placeholder search view.
