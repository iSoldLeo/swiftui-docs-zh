# 软

**类型属性**

表示软性或柔性 UI 对象之间发生碰撞。

## 声明

```swift
static let soft: SensoryFeedback.Flexibility
```

## 获取灵活性值

- **rigid**：表示硬或不灵活的用户界面对象之间发生碰撞。
- **solid**：表示中等柔性的实体 UI 对象之间发生碰撞。

# soft

**Type Property**

Indicates a collision between soft or flexible UI objects.

## Declaration

```swift
static let soft: SensoryFeedback.Flexibility
```

## Getting flexibility values

- **rigid**: Indicates a collision between hard or inflexible UI objects.
- **solid**: Indicates a collision between solid UI objects of medium flexibility.
