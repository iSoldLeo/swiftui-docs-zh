
## 类型属性

- **button**：表示按下按钮（向下触摸）。
- **buttonIconOnly**：表示按下了无文字的图标按钮（向下触摸）。
- **slider**：表示已按下滑块的拇指（向下触摸）。
- **tab**：表示已按下激活选项卡的控件（向下触摸）。
- **toggle**：表示已按下切换按钮（向下触摸）。

# SensoryFeedback.PressFeedback

**Structure**

Feedback that can be played in response to a press (touch down) on a control.

## Declaration

```swift
struct PressFeedback
```

## Type Properties

- **button**: Indicates that a button has been pressed (touch down).
- **buttonIconOnly**: Indicates that an icon-only button, without text, has been pressed (touch down).
- **slider**: Indicates that a slider’s thumb has been pressed (touch down).
- **tab**: Indicates that a control to activate a tab has been pressed (touch down).
- **toggle**: Indicates that a toggle has been pressed (touch down).
