# buttonIconOnly

**Type Property**

Indicates that an icon-only button, without text, has been pressed (touch down).

## Declaration

```swift
static let buttonIconOnly: SensoryFeedback.PressFeedback
```

## Discussion

Only plays feedback on visionOS.
