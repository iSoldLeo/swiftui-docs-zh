# 滑块

**类型属性**

表示滑块的拇指已按下（向下触摸）。

## 声明

```swift
static let slider: SensoryFeedback.PressFeedback
```

## 讨论

滑块控件也应与[release(_:)](../release.zh-Hans.md)一起播放。

只在 visionOS 上播放反馈。

# slider

**Type Property**

Indicates that a slider’s thumb has been pressed (touch down).

## Declaration

```swift
static let slider: SensoryFeedback.PressFeedback
```

## Discussion

Slider controls should also play [release(_:)](../release.zh-Hans.md) with [slider](../ReleaseFeedback/slider.zh-Hans.md).

Only plays feedback on visionOS.
