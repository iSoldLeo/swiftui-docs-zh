# 切换

**类型属性**

表示切换按钮已被按下（向下触摸）。

## 声明

```swift
static let toggle: SensoryFeedback.PressFeedback
```

## 讨论

切换控件还应与[on](../SelectionFeedback/on.zh-Hans.md) 和[off](../SelectionFeedback/off.zh-Hans.md) 一起播放[selection(_:)](../selection.zh-Hans.md)。

仅在 visionOS 上播放反馈。

# toggle

**Type Property**

Indicates that a toggle has been pressed (touch down).

## Declaration

```swift
static let toggle: SensoryFeedback.PressFeedback
```

## Discussion

Toggle controls should also play [selection(_:)](../selection.zh-Hans.md) with [on](../SelectionFeedback/on.zh-Hans.md) and [off](../SelectionFeedback/off.zh-Hans.md).

Only plays feedback on visionOS.
