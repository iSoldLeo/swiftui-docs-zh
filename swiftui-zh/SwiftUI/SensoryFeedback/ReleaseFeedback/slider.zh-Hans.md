# 滑块

**类型属性**

表示滑块的拇指已松开（向上触摸）。

## 声明

```swift
static let slider: SensoryFeedback.ReleaseFeedback
```

## 讨论

滑块控件也应与[press(_:)](../press.zh-Hans.md)一起播放。

只在 visionOS 上播放反馈。

# slider

**Type Property**

Indicates that a slider’s thumb has been released (touch up).

## Declaration

```swift
static let slider: SensoryFeedback.ReleaseFeedback
```

## Discussion

Slider controls should also play [press(_:)](../press.zh-Hans.md) with [slider](../PressFeedback/slider.zh-Hans.md).

Only plays feedback on visionOS.
