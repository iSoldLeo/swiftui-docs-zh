# light

**类型属性**

表示小型或轻量级用户界面对象之间发生碰撞。

## 声明

```swift
static let light: SensoryFeedback.Weight
```

## 获取灵活性值

- **medium**：表示中等大小或中等重量的 UI 对象之间发生碰撞。
- **heavy**：表示大型或重量级 UI 对象之间发生碰撞。

# light

**Type Property**

Indicates a collision between small or lightweight UI objects.

## Declaration

```swift
static let light: SensoryFeedback.Weight
```

## Getting flexibility values

- **medium**: Indicates a collision between medium-sized or medium-weight UI objects.
- **heavy**: Indicates a collision between large or heavyweight UI objects.
