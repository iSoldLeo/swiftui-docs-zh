# 库

**实例属性**

存储函数的着色器库。

## 声明

```swift
var library: ShaderLibrary
```

## 配置函数

- **name**：库中着色器函数的名称。

- **dynamicallyCall(withArguments:)**：通过将提供的参数值应用于引用的函数来返回一个新的着色器。

# library

**Instance Property**

The shader library storing the function.

## Declaration

```swift
var library: ShaderLibrary
```

## Configuring a function

- **name**: The name of the shader function in the library.
- **dynamicallyCall(withArguments:)**: Returns a new shader by applying the provided argument values to the referenced function.
