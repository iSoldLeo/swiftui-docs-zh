# name

**实例属性**

库中着色器函数的名称。

## 声明

```swift
var name: String
```

## 配置函数

- **library**：存储该函数的着色器库。

- **dynamicallyCall(withArguments:)**：通过将提供的参数值应用于引用的函数来返回一个新的着色器。

# name

**Instance Property**

The name of the shader function in the library.

## Declaration

```swift
var name: String
```

## Configuring a function

- **library**: The shader library storing the function.
- **dynamicallyCall(withArguments:)**: Returns a new shader by applying the provided argument values to the referenced function.
