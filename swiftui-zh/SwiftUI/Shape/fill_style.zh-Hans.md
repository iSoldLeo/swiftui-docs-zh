# fill(style:)

**实例方法**

用前景色填充此形状。

## 声明

```swift
nonisolated func fill(style: FillStyle = FillStyle()) -> some View

```

## 参数

- **style**：决定填充渲染方式的样式选项。

## 返回值

填充前景色的形状。

## 填充一个形状

- **fill(_:style:)**：用颜色或渐变填充该形状。

# fill(style:)

**Instance Method**

Fills this shape with the foreground color.

## Declaration

```swift
nonisolated func fill(style: FillStyle = FillStyle()) -> some View

```

## Parameters

- **style**: The style options that determine how the fill renders.

## Return Value

A shape filled with the foreground color.

## Filling a shape

- **fill(_:style:)**: Fills this shape with a color or gradient.
