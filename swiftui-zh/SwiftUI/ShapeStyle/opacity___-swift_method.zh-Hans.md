# opacity(_:)

**实例方法**

返回一个基于 `self` 的新样式，该样式在绘制时会乘以指定的透明度。

## 声明

```swift
func opacity(_ opacity: Double) -> some ShapeStyle

```

## 修改形状样式

- **blendMode(_:)**：返回一个基于 `self` 的新样式，该样式在绘制时会应用指定的混合模式。

- **shadow(_:)**：将指定的阴影效果应用于形状样式。

# opacity(_:)

**Instance Method**

Returns a new style based on `self` that multiplies by the specified opacity when drawing.

## Declaration

```swift
func opacity(_ opacity: Double) -> some ShapeStyle

```

## Modifying a shape style

- **blendMode(_:)**: Returns a new style based on `self` that applies the specified blend mode when drawing.
- **shadow(_:)**: Applies the specified shadow effect to the shape style.
