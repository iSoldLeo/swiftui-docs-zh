# ultraThinMaterial

**类型属性**

一种半透明材质。

## 声明

```swift
static var ultraThinMaterial: Material { get }
```

## 材质

- **thinMaterial**：一种半透明程度高于不透明程度的材质。

- **regularMaterial**：一种略微半透明的材质。

- **thickMaterial**：一种不透明程度高于半透明程度的材质。

- **ultraThickMaterial**：一种半透明程度高于不透明程度的材质。

- **bar**：一种与系统工具栏风格相匹配的材质。

# ultraThinMaterial

**Type Property**

A mostly translucent material.

## Declaration

```swift
static var ultraThinMaterial: Material { get }
```

## Materials

- **thinMaterial**: A material that’s more translucent than opaque.
- **regularMaterial**: A material that’s somewhat translucent.
- **thickMaterial**: A material that’s more opaque than translucent.
- **ultraThickMaterial**: A mostly opaque material.
- **bar**: A material matching the style of system toolbars.
