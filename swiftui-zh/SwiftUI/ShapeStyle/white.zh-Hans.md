# 白色

**类型属性**

适用于 UI 元素的白色。

## 声明

```swift
static var white: Color { get }
```

## 系统颜色

- **black**：适用于 UI 元素的黑色。

- **blue**：适用于 UI 元素的上下文相关的蓝色。

- **brown**：适用于 UI 元素的上下文相关的棕色。

- **clear**：适用于 UI 元素的透明色。

- **cyan**：适用于用户界面元素的上下文相关青色。

- **gray**：适用于用户界面元素的上下文相关灰色。

- **green**：适用于用户界面元素的上下文相关绿色。

- **indigo**：适用于用户界面元素的上下文相关靛蓝色。

- **mint**：适用于用户界面元素的上下文相关薄荷绿。

- **orange**：适用于用户界面元素的上下文相关橙色。

- **pink**：适用于用户界面元素的上下文相关粉色。

- **purple**：适用于用户界面元素的上下文相关紫色。

- **red**：一种上下文相关的红色，适用于用户界面元素。

- **teal**：一种上下文相关的青色，适用于用户界面元素。

- **yellow**：一种上下文相关的黄色，适用于用户界面元素。
