# init(_:)

**Initializer**

创建预览表示。

## 声明

```swift
init(_ title: Text)
```

## 参数

- **title**：要在预览中显示的标题。

## 创建预览

- **init(_:image:)**：创建预览表示。
- **init(_:icon:)**：创建预览表示。
- **init(_:image:icon:)**：创建预览表示。

# init(_:)

**Initializer**

Creates a preview representation.

## Declaration

```swift
init(_ title: Text)
```

## Parameters

- **title**: A title to show in a preview.

## Creating a preview

- **init(_:image:)**: Creates a preview representation.
- **init(_:icon:)**: Creates a preview representation.
- **init(_:image:icon:)**: Creates a preview representation.
