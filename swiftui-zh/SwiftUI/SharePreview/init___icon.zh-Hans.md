# init(_:icon:)

**Initializer**

创建预览表示。

## 声明

```swift
init(_ title: Text, icon: Icon)
```

## 参数

- **title**：要在预览中显示的标题。
- **icon**：在预览中显示的图标。

## 创建预览

- **init(_:)**：创建预览表示。
- **init(_:image:)**：创建预览表示。
- **init(_:image:icon:)**：创建预览表示。

# init(_:icon:)

**Initializer**

Creates a preview representation.

## Declaration

```swift
init(_ title: Text, icon: Icon)
```

## Parameters

- **title**: A title to show in a preview.
- **icon**: An icon to show in a preview.

## Creating a preview

- **init(_:)**: Creates a preview representation.
- **init(_:image:)**: Creates a preview representation.
- **init(_:image:icon:)**: Creates a preview representation.
