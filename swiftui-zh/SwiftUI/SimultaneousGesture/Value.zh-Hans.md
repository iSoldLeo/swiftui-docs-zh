# SimultaneousGesture.Value

**Structure**

The value of a simultaneous gesture that indicates which of its two gestures receives events.

## Declaration

```swift
@frozen struct Value
```

## Getting gesture values

- **first**: The value of the first gesture.
- **second**: The value of the second gesture.

## Conforms To

- Copyable
- Equatable
- Hashable
- Sendable
- SendableMetatype
