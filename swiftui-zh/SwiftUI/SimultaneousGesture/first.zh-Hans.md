# 第一次

**实例属性**

可同时进行的两个手势中的第一个。

## 声明

```swift
var first: First
```

## 创建手势

- **init(_:_:)**：创建一个包含两个手势的手势，这两个手势可以相互独立地接收更新或成功。
- **second**：两个手势中可以同时出现的第二个手势。

# first

**Instance Property**

The first of two gestures that can happen simultaneously.

## Declaration

```swift
var first: First
```

## Creating the gesture

- **init(_:_:)**: Creates a gesture with two gestures that can receive updates or succeed independently of each other.
- **second**: The second of two gestures that can happen simultaneously.
