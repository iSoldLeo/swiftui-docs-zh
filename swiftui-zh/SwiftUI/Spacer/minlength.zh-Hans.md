# 最小长度

**实例属性**

该垫片沿一个或多个膨胀轴可收缩到的最小长度。

## 声明

```swift
var minLength: CGFloat?
```

## 讨论

如果 `nil`，则使用系统默认的视图间距。

## 创建间距

- **init(minLength:)**

# minLength

**Instance Property**

The minimum length this spacer can be shrunk to, along the axis or axes of expansion.

## Declaration

```swift
var minLength: CGFloat?
```

## Discussion

If `nil`, the system default spacing between views is used.

## Creating a spacer

- **init(minLength:)**
