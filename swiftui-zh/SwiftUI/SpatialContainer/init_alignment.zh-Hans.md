# init(alignment:)

**Initializer**

Creates a spatial container layout with the specified 3D alignment.

## Declaration

```swift
init(alignment: Alignment3D = .center)
```

## Parameters

- **alignment**: The 3D alignment guide for aligning the subviews in this container.
