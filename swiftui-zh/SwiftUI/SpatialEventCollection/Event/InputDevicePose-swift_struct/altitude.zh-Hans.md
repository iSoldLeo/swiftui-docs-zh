# 高度

**实例属性**

高度角。

## 声明

```swift
var altitude: Angle
```

## 讨论

角度为 0 表示设备与内容平行，90 度表示设备与内容表面垂直。

## 获取事件类型

- **azimuth**：方位角。
- **pose3D**：输入设备的 3D 姿态。

# altitude

**Instance Property**

The altitude angle.

## Declaration

```swift
var altitude: Angle
```

## Discussion

An angle of zero indicates that the device is parallel to the content, while 90 degrees indicates that it is normal to the content surface.

## Getting the event type

- **azimuth**: The azimuth angle.
- **pose3D**: The 3D pose of the input device.
