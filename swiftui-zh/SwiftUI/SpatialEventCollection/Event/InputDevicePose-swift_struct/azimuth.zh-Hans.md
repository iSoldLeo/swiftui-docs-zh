# azimuth

**Instance Property**

The azimuth angle.

## Declaration

```swift
var azimuth: Angle
```

## Discussion

An angle of zero points along the content’s positive x-axis.

## Getting the event type

- **altitude**: The altitude angle.
- **pose3D**: The 3D pose of the input device.
