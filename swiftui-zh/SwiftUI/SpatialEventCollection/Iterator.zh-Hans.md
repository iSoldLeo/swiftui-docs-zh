# SpatialEventCollection.Iterator

**Structure**

An iterator over all events in the collection.

## Declaration

```swift
struct Iterator
```

## Getting the next event

- **next()**: The next `Event` in the sequence, if one exists.

## Iterating over events in the collection

- **makeIterator()**: Makes an iterator over all events in the collection.

## Conforms To

- IteratorProtocol
