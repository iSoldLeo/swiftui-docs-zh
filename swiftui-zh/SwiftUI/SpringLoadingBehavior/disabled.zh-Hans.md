# 已禁用

**类型属性**

将禁用适用视图的 Spring Loading 交互。

## 声明

```swift
static let disabled: SpringLoadingBehavior
```

## 获取行为

- **automatic**：自动 Spring Loading 行为。

- **enabled**：将启用适用视图的 Spring Loading 交互。

# disabled

**Type Property**

Spring loaded interactions will be disabled for applicable views.

## Declaration

```swift
static let disabled: SpringLoadingBehavior
```

## Getting the behaviors

- **automatic**: The automatic spring loading behavior.
- **enabled**: Spring loaded interactions will be enabled for applicable views.
