# enabled

**类型属性**

将为适用的视图启用 Spring Loading 交互。

## 声明

```swift
static let enabled: SpringLoadingBehavior
```

## 获取行为

- **automatic**：自动 Spring Loading 行为。

- **disabled**：将为适用的视图禁用 Spring Loading 交互。

# enabled

**Type Property**

Spring loaded interactions will be enabled for applicable views.

## Declaration

```swift
static let enabled: SpringLoadingBehavior
```

## Getting the behaviors

- **automatic**: The automatic spring loading behavior.
- **disabled**: Spring loaded interactions will be disabled for applicable views.
