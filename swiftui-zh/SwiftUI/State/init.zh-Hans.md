# init()

**Initializer**

创建无初始值的状态属性。

## 声明

```swift
init()
```

## 讨论

这个初始化函数的行为类似于输入`nil` 的 [init(wrappedValue:)](init_wrappedValue.zh-Hans.md) 初始化函数。更多信息，请参见该初始化程序。

## 创建状态

- **init(wrappedValue:)**：创建一个存储初始包装值的状态属性。
- **init(initialValue:)**：创建存储初始值的状态属性。

# init()

**Initializer**

Creates a state property without an initial value.

## Declaration

```swift
init()
```

## Discussion

This initializer behaves like the [init(wrappedValue:)](init_wrappedValue.zh-Hans.md) initializer with an input of `nil`. See that initializer for more information.

## Creating a state

- **init(wrappedValue:)**: Creates a state property that stores an initial wrapped value.
- **init(initialValue:)**: Creates a state property that stores an initial value.
