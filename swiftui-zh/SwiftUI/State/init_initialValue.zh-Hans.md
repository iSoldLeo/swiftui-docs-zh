# init(initialValue:)

**Initializer**

创建一个存储初始值的状态属性。

### 声明

```swift
init(initialValue value: Value)
```

## 参数

- **value**：要存储在状态属性中的初始值。

## 讨论

该初始化程序与 [init(wrappedValue:)](init_wrappedValue.zh-Hans.md) 初始化程序具有相同的行为。更多信息请参见该初始化程序。

## 创建状态

- **init(wrappedValue:)**：创建一个存储初始包装值的状态属性。
- **init()**：创建没有初始值的状态属性。

# init(initialValue:)

**Initializer**

Creates a state property that stores an initial value.

## Declaration

```swift
init(initialValue value: Value)
```

## Parameters

- **value**: An initial value to store in the state property.

## Discussion

This initializer has the same behavior as the [init(wrappedValue:)](init_wrappedValue.zh-Hans.md) initializer. See that initializer for more information.

## Creating a state

- **init(wrappedValue:)**: Creates a state property that stores an initial wrapped value.
- **init()**: Creates a state property without an initial value.
