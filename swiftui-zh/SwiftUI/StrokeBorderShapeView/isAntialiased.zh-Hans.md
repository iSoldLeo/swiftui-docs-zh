# isAntialiased

**实例属性**

是否要对该形状进行反锯齿绘制。

## 声明

```swift
var isAntialiased: Bool { get set }
```

## 获取形状视图属性

- **background**：该视图下方显示的背景。
- **shape**：该类型绘制的形状，并为其他绘制操作提供形状。
- **strokeStyle**：绘制该视图的形状时使用的描边样式。
- **style**：用于描边此视图形状的边框的样式。

# isAntialiased

**Instance Property**

Whether this shape should be drawn antialiased.

## Declaration

```swift
var isAntialiased: Bool { get set }
```

## Getting shape view properties

- **background**: The background shown beneath this view.
- **shape**: The shape that this type draws and provides for other drawing operations.
- **strokeStyle**: The stroke style used when stroking this view’s shape.
- **style**: The style that strokes the border of this view’s shape.
