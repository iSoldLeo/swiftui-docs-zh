# dash

**实例属性**

用于绘制虚线的已绘制线段和未绘制线段的长度。

## 声明

```swift
var dash: [CGFloat]
```

## 设置描边样式属性

- **lineWidth**：描边路径的宽度。

- **lineCap**：线条的端点样式。

- **lineJoin**：线条的连接类型。

- **miterLimit**：用于确定在连接处是否使用斜角而不是斜接的阈值。

- **dashPhase**：虚线从虚线图案的哪个位置开始。

# dash

**Instance Property**

The lengths of painted and unpainted segments used to make a dashed line.

## Declaration

```swift
var dash: [CGFloat]
```

## Setting stroke style properties

- **lineWidth**: The width of the stroked path.
- **lineCap**: The endpoint style of a line.
- **lineJoin**: The join type of a line.
- **miterLimit**: A threshold used to determine whether to use a bevel instead of a miter at a join.
- **dashPhase**: How far into the dash pattern the line starts.
