# Subview.ContainerSizingOptions

**Enumeration**

Options on how all subviews should be sized when in a container.

## Declaration

```swift
enum ContainerSizingOptions
```

## Overview



## Enumeration Cases

- **Subview.ContainerSizingOptions.uniform(axis:)**: Subviews will share the same size.
- **Subview.ContainerSizingOptions.variable**: Subviews will be sized individually.

- Sendable
- SendableMetatype
