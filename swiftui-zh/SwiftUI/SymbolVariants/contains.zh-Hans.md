# contains(_:)

**实例方法**

返回一个布尔值，指示当前变体是否包含指定的变体。

## 声明

```swift
func contains(_ other: SymbolVariants) -> Bool
```

## 参数

- **other**：要在此变体中查找的变体。

## 返回值

`true`：如果此变体包含 `other`；否则，返回 `false`。

# contains(_:)

**Instance Method**

Returns a Boolean value that indicates whether the current variant contains the specified variant.

## Declaration

```swift
func contains(_ other: SymbolVariants) -> Bool
```

## Parameters

- **other**: A variant to look for in this variant.

## Return Value

`true` if this variant contains `other`; otherwise, `false`.
