# SystemFormatStyle.Timer

**Structure**

系统计时器格式样式。

## 声明

```swift
struct Timer
```

## Initializers

- **init(countingDownIn:showsHours:maxFieldCount:maxPrecision:)**：创建从给定时间间隔开始倒计时的计时器格式样式。
- **init(countingUpIn:showsHours:maxFieldCount:maxPrecision:)**：创建向上计数到给定间隔的定时器格式样式。

- 离散格式样式
- 可编码
- 可等价
- 格式样式

# SystemFormatStyle.Timer

**Structure**

The system timer format style.

## Declaration

```swift
struct Timer
```

## Initializers

- **init(countingDownIn:showsHours:maxFieldCount:maxPrecision:)**: Create a timer format style that counts down from the given interval.
- **init(countingUpIn:showsHours:maxFieldCount:maxPrecision:)**: Create a timer format style that counts up to the given interval.

## Conforms To

- Copyable
- Decodable
- DiscreteFormatStyle
- Encodable
- Equatable
- FormatStyle
- Hashable
- Sendable
- SendableMetatype
