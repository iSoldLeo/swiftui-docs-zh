# topBar

**类型属性**

标签页视图的顶部栏。

## 声明

```swift
static let topBar: TabBarPlacement
```

## 讨论

标签页栏可以在 iPad 和 macOS 上显示为 [tabBarOnly](../TabViewStyle/tabBarOnly.zh-Hans.md) 的顶部栏，也可以在 iPad 上显示为 [sidebarAdaptable](../TabViewStyle/sidebarAdaptable.zh-Hans.md) 的顶部栏。

# topBar

**Type Property**

Top bar of a tab view.

## Declaration

```swift
static let topBar: TabBarPlacement
```

## Discussion

A tab bar can display as the top bar in [tabBarOnly](../TabViewStyle/tabBarOnly.zh-Hans.md) on iPad and macOS, or [sidebarAdaptable](../TabViewStyle/sidebarAdaptable.zh-Hans.md) on iPad.
