# TabCustomizationBehavior

**Structure**

可自定义标签页内容的自定义行为。

## 声明

```swift
struct TabCustomizationBehavior
```

## 概述

此类型需与 [customizationBehavior(_:for:)](TabContent/customizationBehavior___for.zh-Hans.md) 修饰符结合使用。

## 类型属性

- **automatic**：自动自定义行为。

- **disabled**：此自定义行为不可用。

- **reorderable**：可重新排序的自定义行为。

## 启用标签页自定义

- **tabViewCustomization(_:)**：指定要应用于标签页视图侧边栏显示的自定义设置。

- **TabViewCustomization**：用户对可自适应侧边栏标签页视图所做的自定义设置。

# TabCustomizationBehavior

**Structure**

The customization behavior of customizable tab view content.

## Declaration

```swift
struct TabCustomizationBehavior
```

## Overview

Use this type in conjunction with the [customizationBehavior(_:for:)](TabContent/customizationBehavior___for.zh-Hans.md) modifier.

## Type Properties

- **automatic**: The automatic customization behavior.
- **disabled**: The customization behavior isn’t available.
- **reorderable**: The reorderable customization behavior.

## Enabling tab customization

- **tabViewCustomization(_:)**: Specifies the customizations to apply to the sidebar representation of the tab view.
- **TabViewCustomization**: The customizations a person makes to an adaptable sidebar tab view.
