# sidebarOnly

**Type Property**

The sidebar tab placement location.

## Declaration

```swift
static let sidebarOnly: TabPlacement
```

## Discussion

On visionOS in the [sidebarAdaptable](../TabViewStyle/sidebarAdaptable.zh-Hans.md) style, the ornament tab bar is considered part of the sidebar.
