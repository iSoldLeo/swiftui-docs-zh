# 标签搜索激活

**Structure**

配置搜索选项卡中搜索的激活行为。

## 声明

```swift
struct TabSearchActivation
```

## 概览

您可以使用[TabView](TabView.zh-Hans.md)修饰符为特定[tabViewSearchActivation(_:)](View/tabViewSearchActivation.zh-Hans.md) 配置此功能。

### 类型属性

- **automatic**：默认搜索激活行为。
- **searchTabSelection**：将搜索选项卡的选择链接到搜索激活。

# TabSearchActivation

**Structure**

Configures the activation behavior of search in the search tab.

## Declaration

```swift
struct TabSearchActivation
```

## Overview

You can configure this for a particular [TabView](TabView.zh-Hans.md) by using the [tabViewSearchActivation(_:)](View/tabViewSearchActivation.zh-Hans.md) modifier.

## Type Properties

- **automatic**: The default search activation behavior.
- **searchTabSelection**: Links the search tab’s selection to search activation.

- Hashable
