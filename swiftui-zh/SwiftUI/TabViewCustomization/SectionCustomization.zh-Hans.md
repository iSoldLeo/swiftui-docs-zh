# TabViewCustomization.SectionCustomization

**Structure**

The customizations a user has made to a [TabSection](../TabSection.zh-Hans.md).

## Declaration

```swift
struct SectionCustomization
```

## Instance Properties

- **tabOrder**: The order customization for the tabs in a section, identified by customization identifier.

## Instance Methods

- **resetTabOrder()**: Resets the order of the tabs in the section back to the default.
