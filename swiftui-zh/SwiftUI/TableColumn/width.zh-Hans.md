# width()

**实例方法**

设置列的宽度。

## 声明

```swift
nonisolated func width() -> TableColumn<RowValue, Sort, Content, Label>
```

## 设置列宽

- **width(_:)**：创建固定宽度的表格列，用户不能调整其大小。
- **width(min:ideal:max:)**：根据提供的约束创建可调整大小的表列。

# width()

**Instance Method**

Sets the column’s width.

## Declaration

```swift
nonisolated func width() -> TableColumn<RowValue, Sort, Content, Label>
```

## Setting the column width

- **width(_:)**: Creates a fixed width table column that isn’t user resizable.
- **width(min:ideal:max:)**: Creates a resizable table column with the provided constraints.
