# resetOrder()

**实例方法**

将列顺序重置为默认值，同时保留自定义的可见性和大小。

## 声明

```swift
mutating func resetOrder()
```

## 说明

绑定到此状态的表格将按照其列构建器描述的顺序排列列。

## 管理自定义设置

- **subscript(visibility:)**：由其标识符标识的列的可见性。

# resetOrder()

**Instance Method**

Resets the column order back to the default, preserving the customized visibility and size.

## Declaration

```swift
mutating func resetOrder()
```

## Discussion

Tables that are bound to this state will order their columns as described by their column builder.

## Managing the customization

- **subscript(visibility:)**: The visibility of the column identified by its identifier.
