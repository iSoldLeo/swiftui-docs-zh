# 可见性

**类型属性**

允许用户隐藏或显示列的行为。

## 声明

```swift
static let visibility: TableColumnCustomizationBehavior
```

## 获取自定义行为

- **all**：所有自定义行为。

- **reorder**：允许用户重新排序列的行为。

- **resize**：允许用户调整列宽的行为。

# visibility

**Type Property**

A behavior that allows the column to be hidden or revealed by the user.

## Declaration

```swift
static let visibility: TableColumnCustomizationBehavior
```

## Getting the customization behavior

- **all**: All customization behaviors.
- **reorder**: A behavior that allows the column to be reordered by the user.
- **resize**: A behavior that allows the column to be resized by the user.
