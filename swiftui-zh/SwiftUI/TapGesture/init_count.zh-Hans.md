# init(count:)

**Initializer**

创建一个需要指定点击次数的点击手势。

## 声明

```swift
init(count: Int = 1)
```

## 参数

- **count**：完成点击手势所需的点击次数。

## 创建点击手势

- **count**：所需的点击事件数。

# init(count:)

**Initializer**

Creates a tap gesture with the number of required taps.

## Declaration

```swift
init(count: Int = 1)
```

## Parameters

- **count**: The required number of taps to complete the tap gesture.

## Creating a tap gesture

- **count**: The required number of tap events.
