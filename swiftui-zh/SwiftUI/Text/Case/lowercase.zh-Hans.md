# Text.Case.lowercase

**Case**

Displays text in all lowercase characters.

## Declaration

```swift
case lowercase
```

## Discussion

For example, “Hello” would be displayed as “hello”.



## Getting text cases

- **Text.Case.uppercase**: Displays text in all uppercase characters.
