# Text.Case.uppercase

**Case**

Displays text in all uppercase characters.

## Declaration

```swift
case uppercase
```

## Discussion

For example, “Hello” would be displayed as “HELLO”.



## Getting text cases

- **Text.Case.lowercase**: Displays text in all lowercase characters.
