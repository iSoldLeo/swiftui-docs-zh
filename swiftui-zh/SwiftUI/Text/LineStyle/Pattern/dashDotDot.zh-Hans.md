# dashDotDot

**类型属性**

绘制一条由短划线和两个点交替组成的线条。

## 声明

```swift
static let dashDotDot: Text.LineStyle.Pattern
```

## 获取线条样式图案

- **solid**：绘制一条实线。

- **dot**：绘制一条点线。

- **dash**：绘制一条短划线。

- **dashDot**

# dashDotDot

**Type Property**

Draw a line of alternating dashes and two dots.

## Declaration

```swift
static let dashDotDot: Text.LineStyle.Pattern
```

## Getting line style patterns

- **solid**: Draw a solid line.
- **dot**: Draw a line of dots.
- **dash**: Draw a line of dashes.
- **dashDot**
