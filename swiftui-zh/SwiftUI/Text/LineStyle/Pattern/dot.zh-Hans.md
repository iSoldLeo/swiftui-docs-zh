# dot

**类型属性**

绘制一条点线。

## 声明

```swift
static let dot: Text.LineStyle.Pattern
```

## 获取线条样式图案

- **solid**：绘制一条实线。

- **dash**：绘制一条虚线。

- **dashDot**

- **dashDotDot**：绘制一条由虚线和两个点交替组成的线条。

# dot

**Type Property**

Draw a line of dots.

## Declaration

```swift
static let dot: Text.LineStyle.Pattern
```

## Getting line style patterns

- **solid**: Draw a solid line.
- **dash**: Draw a line of dashes.
- **dashDot**
- **dashDotDot**: Draw a line of alternating dashes and two dots.
