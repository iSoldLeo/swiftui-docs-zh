# solid

**Type Property**

Draw a solid line.

## Declaration

```swift
static let solid: Text.LineStyle.Pattern
```

## Getting line style patterns

- **dot**: Draw a line of dots.
- **dash**: Draw a line of dashes.
- **dashDot**
- **dashDotDot**: Draw a line of alternating dashes and two dots.
