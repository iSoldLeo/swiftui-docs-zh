# fontDesign(_:)

**实例方法**

设置文本的字体样式。

## 声明

```swift
nonisolated func fontDesign(_ design: Font.Design?) -> Text
```

## 参数

- **design**：可用的字体样式之一。

## 返回值

使用指定字体样式的文本。

## 选择字体

- **font(_:)**：设置视图中文本的默认字体。

- **fontWeight(_:)**：设置文本的字重。

- **fontWidth(_:)**：设置文本的字宽。

# fontDesign(_:)

**Instance Method**

Sets the font design of the text.

## Declaration

```swift
nonisolated func fontDesign(_ design: Font.Design?) -> Text
```

## Parameters

- **design**: One of the available font designs.

## Return Value

Text that uses the font design you specify.

## Choosing a font

- **font(_:)**: Sets the default font for text in the view.
- **fontWeight(_:)**: Sets the font weight of the text.
- **fontWidth(_:)**: Sets the font width of the text.
