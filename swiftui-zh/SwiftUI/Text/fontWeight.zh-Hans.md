# fontWeight(_:)

**实例方法**

设置文本的字体粗细。

## 声明

```swift
nonisolated func fontWeight(_ weight: Font.Weight?) -> Text
```

## 参数

- **weight**：可用的字体粗细之一。

## 返回值

使用指定字体粗细显示的文本。

## 选择字体

- **font(_:)**：设置视图中文本的默认字体。

- **fontDesign(_:)**：设置文本的字体样式。

- **fontWidth(_:)**：设置文本的字体宽度。

# fontWeight(_:)

**Instance Method**

Sets the font weight of the text.

## Declaration

```swift
nonisolated func fontWeight(_ weight: Font.Weight?) -> Text
```

## Parameters

- **weight**: One of the available font weights.

## Return Value

Text that uses the font weight you specify.

## Choosing a font

- **font(_:)**: Sets the default font for text in the view.
- **fontDesign(_:)**: Sets the font design of the text.
- **fontWidth(_:)**: Sets the font width of the text.
