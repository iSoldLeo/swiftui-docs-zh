# fontWidth(_:)

**实例方法**

设置文本的字体宽度。

## 声明

```swift
nonisolated func fontWidth(_ width: Font.Width?) -> Text
```

## 参数

- **width**：可用的字体宽度之一。

## 返回值

使用您指定的字体宽度显示的文本（如果可用）。

## 选择字体

- **font(_:)**：设置视图中文本的默认字体。

- **fontWeight(_:)**：设置文本的字体粗细。

- **fontDesign(_:)**：设置文本的字体样式。

# fontWidth(_:)

**Instance Method**

Sets the font width of the text.

## Declaration

```swift
nonisolated func fontWidth(_ width: Font.Width?) -> Text
```

## Parameters

- **width**: One of the available font widths.

## Return Value

Text that uses the font width you specify, if available.

## Choosing a font

- **font(_:)**: Sets the default font for text in the view.
- **fontWeight(_:)**: Sets the font weight of the text.
- **fontDesign(_:)**: Sets the font design of the text.
