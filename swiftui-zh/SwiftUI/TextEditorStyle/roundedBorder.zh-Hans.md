# roundedBorder

**类型属性**

带有系统定义的圆角边框的文本编辑器样式。

## 声明

```swift
@MainActor @preconcurrency static var roundedBorder: RoundedBorderTextEditorStyle { get }
```

## 获取内置样式

- **automatic**：基于文本编辑器上下文的默认文本编辑器样式。

- **plain**：无装饰的文本编辑器样式。

# roundedBorder

**Type Property**

A text editor style with a system-defined rounded border.

## Declaration

```swift
@MainActor @preconcurrency static var roundedBorder: RoundedBorderTextEditorStyle { get }
```

## Getting built-in styles

- **automatic**: The default text editor style, based on the text editor’s context.
- **plain**: A text editor style with no decoration.
