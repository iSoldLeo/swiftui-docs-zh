# 自动

**类型属性**

基于文本字段上下文的默认文本字段样式。

## 声明

```swift
static var automatic: DefaultTextFieldStyle { get }
```

## 讨论

默认样式是根据当前平台和文本字段在视图层次结构中的上下文推荐的样式。

## 获取内置文本字段样式

- **plain**：无装饰的文本字段样式。
- **roundedBorder**：带有系统定义的圆形边框的文本字段样式。
- **squareBorder**：具有系统定义的方形边框的文本字段样式。

# automatic

**Type Property**

The default text field style, based on the text field’s context.

## Declaration

```swift
static var automatic: DefaultTextFieldStyle { get }
```

## Discussion

The default style represents the recommended style based on the current platform and the text field’s context within the view hierarchy.

## Getting built-in text field styles

- **plain**: A text field style with no decoration.
- **roundedBorder**: A text field style with a system-defined rounded border.
- **squareBorder**: A text field style with a system-defined square border.
