# 句子

**类型属性**

定义自动大写行为，将每个句子中的第一个字母大写。

## 声明

```swift
static var sentences: TextInputAutocapitalization { get }
```

## 获取自动大写选项

- **characters**：定义自动大写行为，将每个字母大写。
- **words**：定义自动大写行为，该行为将大写每个单词的第一个字母。
- **never**：定义不大写任何内容的自动大写行为。

# sentences

**Type Property**

Defines an autocapitalizing behavior that will capitalize the first letter in every sentence.

## Declaration

```swift
static var sentences: TextInputAutocapitalization { get }
```

## Getting autocapitalization options

- **characters**: Defines an autocapitalizing behavior that will capitalize every letter.
- **words**: Defines an autocapitalizing behavior that will capitalize the first letter of every word.
- **never**: Defines an autocapitalizing behavior that will not capitalize anything.
