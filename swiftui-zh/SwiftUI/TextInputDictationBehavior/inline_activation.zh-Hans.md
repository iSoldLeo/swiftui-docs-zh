# inline(activation:)

**类型方法**

在搜索栏中添加语音输入麦克风。

## 声明

```swift
static func inline(activation: TextInputDictationActivation) -> TextInputDictationBehavior
```

## 获取行为值

- **automatic**：平台适用的默认文本输入语音输入行为。

- **preventDictation**：禁止在搜索栏中使用语音输入麦克风。

# inline(activation:)

**Type Method**

Adds a dictation microphone in the search bar.

## Declaration

```swift
static func inline(activation: TextInputDictationActivation) -> TextInputDictationBehavior
```

## Getting behavior values

- **automatic**: A platform-appropriate default text input dictation behavior.
- **preventDictation**: Prevents the search bar from having a dictation microphone.
