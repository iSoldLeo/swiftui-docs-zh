# preventDictation

**类型属性**

阻止搜索栏启用语音输入麦克风。

## 声明

```swift
static let preventDictation: TextInputDictationBehavior
```

## 获取行为值

- **automatic**：平台适用的默认文本输入语音输入行为。

- **inline(activation:)**：在搜索栏中添加语音输入麦克风。

# preventDictation

**Type Property**

Prevents the search bar from having a dictation microphone.

## Declaration

```swift
static let preventDictation: TextInputDictationBehavior
```

## Getting behavior values

- **automatic**: A platform-appropriate default text input dictation behavior.
- **inline(activation:)**: Adds a dictation microphone in the search bar.
