# TextInputFormattingControlPlacement.Set

**Structure**

A set of system text formatting controls.

## Declaration

```swift
struct Set
```

## Type Properties

- **accessoryBar**: Text formatting controls placed beneath the window toolbar.
- **all**
- **contextMenu**: Text formatting controls placed in the context menu, and the edit menu on iOS.
- **default**
- **fontPanel**: Text formatting controls for choosing fonts that appear in a dedicated window on macOS.
- **inputAssistant**: Expanded text formatting controls which appear in the shortcuts bar on iPadOS or in the bottom ornament on visionOS.
