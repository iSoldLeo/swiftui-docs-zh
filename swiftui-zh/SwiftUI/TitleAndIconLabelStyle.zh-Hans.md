# TitleAndIconLabelStyle

**Structure**

A label style that shows both the title and icon of the label using a system-standard layout.

## Declaration

```swift
struct TitleAndIconLabelStyle
```

## Overview

You can also use [titleAndIcon](LabelStyle/titleAndIcon.zh-Hans.md) to construct this style.

## Creating the label style

- **init()**: Creates a label style that shows both the title and icon of the label using a system-standard layout.

## Supporting types

- **DefaultLabelStyle**: The default label style in the current context.
- **IconOnlyLabelStyle**: A label style that only displays the icon of the label.
- **TitleOnlyLabelStyle**: A label style that only displays the title of the label.

## Conforms To

- LabelStyle
