# TitleOnlyLabelStyle

**Structure**

仅显示标签标题的标签样式。

## 声明

```swift
struct TitleOnlyLabelStyle
```

## 概述

您也可以使用 [titleOnly](LabelStyle/titleOnly.zh-Hans.md) 来构建此样式。

## 创建标签样式

- **init()**：创建仅显示标题的标签样式。

## 支持的类型

- **DefaultLabelStyle**：当前上下文中的默认标签样式。

- **IconOnlyLabelStyle**：仅显示标签图标的标签样式。 - **TitleAndIconLabelStyle**：一种标签样式，使用系统标准布局同时显示标签的标题和图标。

# TitleOnlyLabelStyle

**Structure**

A label style that only displays the title of the label.

## Declaration

```swift
struct TitleOnlyLabelStyle
```

## Overview

You can also use [titleOnly](LabelStyle/titleOnly.zh-Hans.md) to construct this style.

## Creating the label style

- **init()**: Creates a title-only label style.

## Supporting types

- **DefaultLabelStyle**: The default label style in the current context.
- **IconOnlyLabelStyle**: A label style that only displays the icon of the label.
- **TitleAndIconLabelStyle**: A label style that shows both the title and icon of the label using a system-standard layout.

## Conforms To

- LabelStyle
