# ToggleStyleConfiguration.Label

**Structure**

A type-erased label of a toggle.

## Declaration

```swift
@MainActor @preconcurrency struct Label
```

## Overview

SwiftUI provides a value of this type — which is a [View](../View.zh-Hans.md) type — as the [label-swift.property](label-swift_property.zh-Hans.md) to your custom toggle style implementation. Use the label to help define the appearance of the toggle.

## Getting the label view

- **label**: A view that describes the effect of switching the toggle between states.

## Conforms To

- View
