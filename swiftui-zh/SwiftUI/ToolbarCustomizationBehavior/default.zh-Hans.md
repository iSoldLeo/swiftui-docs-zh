# default

**类型属性**

默认的自定义行为。

## 声明

```swift
static var `default`: ToolbarCustomizationBehavior { get }
```

## 讨论

具有此行为的项目默认显示在工具栏中，用户可以将其移动或从工具栏中移除。

## 获取自定义行为

- **disabled**：已禁用的自定义行为。

- **reorderable**：可重新排序的自定义行为。

# default

**Type Property**

The default customization behavior.

## Declaration

```swift
static var `default`: ToolbarCustomizationBehavior { get }
```

## Discussion

Items with this behavior start in the toolbar and can be moved or removed from the toolbar by the user.

## Getting customization behaviors

- **disabled**: The disabled customization behavior.
- **reorderable**: The reorderable customization behavior.
