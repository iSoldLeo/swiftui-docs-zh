# navigationBarTrailing

**Type Property**

Places the item in the trailing edge of the navigation bar.

## Declaration

```swift
static let navigationBarTrailing: ToolbarItemPlacement
```

## Deprecated symbols

- **init(id:)**: Creates a custom accessory bar item placement.
- **navigationBarLeading**: Places the item in the leading edge of the navigation bar.
