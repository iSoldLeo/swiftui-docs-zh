# automatic

**Type Property**

The automatic role.

## Declaration

```swift
static var automatic: ToolbarRole { get }
```

## Discussion

In iOS, tvOS, and watchOS this resolves to the [navigationStack](navigationStack.zh-Hans.md) role. In macOS, this resolves to the [editor](editor.zh-Hans.md) role.
