**Type Property**

The navigationStack role.

## Declaration

```swift
static var navigationStack: ToolbarRole { get }
```

## Discussion

Use this role for content that can be pushed and popped.

## Behavior-specific roles

- **browser**: The browser role.
- **editor**: The editor role.
