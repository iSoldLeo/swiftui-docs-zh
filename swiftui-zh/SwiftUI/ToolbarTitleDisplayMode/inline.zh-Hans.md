# 内联

**类型属性**

内联模式。

## 声明

```swift
static var inline: ToolbarTitleDisplayMode { get }
```

## 讨论

在 iOS、tvOS 和 watchOS 中，该模式会在工具栏中间以较小的尺寸显示标题。这在 macOS 中没有影响。

## 获取显示模式

- **automatic**：自动模式。
- **inlineLarge**：内联大屏模式。
- **large**：大型模式。

# inline

**Type Property**

The inline mode.

## Declaration

```swift
static var inline: ToolbarTitleDisplayMode { get }
```

## Discussion

In iOS, tvOS, and watchOS this mode displays the title with a smaller size in the middle of the toolbar. This has no effect in macOS.

## Getting display modes

- **automatic**: The automatic mode.
- **inlineLarge**: The inline large mode.
- **large**: The large mode.
