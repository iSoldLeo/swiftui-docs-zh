# 大

**类型属性**

大号模式。

## 声明

```swift
static var large: ToolbarTitleDisplayMode { get }
```

## 讨论

在 iOS 和 watchOS 中，当可滚动内容滚动到顶部时，工具栏标题会显示在导航栏内容的下方，当内容滚动时，标题会过渡到工具栏的中央。

## 获取显示模式

- **automatic**：自动模式。
- **inline**：内联模式。
- **inlineLarge**：内联大屏模式。

# large

**Type Property**

The large mode.

## Declaration

```swift
static var large: ToolbarTitleDisplayMode { get }
```

## Discussion

In iOS, and watchOS, this displays the toolbar title below the content of the navigation bar when scrollable content is scrolled to the top and transitions to the center of the toolbar as content is scrolled.

## Getting display modes

- **automatic**: The automatic mode.
- **inline**: The inline mode.
- **inlineLarge**: The inline large mode.
