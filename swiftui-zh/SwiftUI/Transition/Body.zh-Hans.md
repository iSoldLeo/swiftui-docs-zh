# Body

**Associated Type**

The type of view representing the body.

## Declaration

```swift
associatedtype Body : View
```

## Creating a custom transition

- **body(content:phase:)**: Gets the current body of the caller.
- **Transition.Content**: The content view type passed to `body()`.
