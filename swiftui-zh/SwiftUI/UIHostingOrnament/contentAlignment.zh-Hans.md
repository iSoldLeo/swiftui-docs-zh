# contentAlignment

**Instance Property**

The alignment in the ornament used to position it.

## Declaration

```swift
var contentAlignment: Alignment { get set }
```

## Setting the alignment

- **sceneAnchor**: The anchor point for aligning the ornament’s content (based on the `contentAlignment`) with the scene.
