# UIOrnament

**Class**

The abstract base class that represents an ornament.

## Declaration

```swift
class UIOrnament
```

## Overview

You don’t create an instance of this class directly. Instead use [UIHostingOrnament](UIHostingOrnament.zh-Hans.md) when you want to add ornaments to a UIKit view controller.

## Hosting an ornament in UIKit

- **UIHostingOrnament**: A model that represents an ornament suitable for being hosted in UIKit.
