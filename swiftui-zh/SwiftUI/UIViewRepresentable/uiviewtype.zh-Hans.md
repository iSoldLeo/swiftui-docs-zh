# UIViewType

**关联类型**

要显示的视图类型。

## 声明

```swift
associatedtype UIViewType : UIView
```

## 创建和更新视图

- **makeUIView(context:)**：创建视图对象并配置其初始状态。
- **updateUIView(_:context:)**：使用来自 SwiftUI 的新信息更新指定视图的状态。
- **UIViewRepresentable.Context**：使用来自 SwiftUI 的新信息更新指定视图的状态。
