# init(cornerRadii:style:)

**Initializer**

新建一个四角不平的圆角矩形。

## 声明

```swift
init(cornerRadii: RectangleCornerRadii, style: RoundedCornerStyle = .continuous)
```

## 参数

- **cornerRadii**：每个角的半径。
- **style**：形状绘制的角的样式。

## 创建不平整的圆角矩形

- **init(topLeadingRadius:bottomLeadingRadius:bottomTrailingRadius:topTrailingRadius:style:)**：新建一个四角不平整的圆角矩形。

# init(cornerRadii:style:)

**Initializer**

Creates a new rounded rectangle shape with uneven corners.

## Declaration

```swift
init(cornerRadii: RectangleCornerRadii, style: RoundedCornerStyle = .continuous)
```

## Parameters

- **cornerRadii**: The radii of each corner.
- **style**: The style of corners drawn by the shape.

## Creating an uneven rounded rectangle

- **init(topLeadingRadius:bottomLeadingRadius:bottomTrailingRadius:topTrailingRadius:style:)**: Creates a new rounded rectangle shape with uneven corners.
