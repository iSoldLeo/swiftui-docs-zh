# UnifiedCompactWindowToolbarStyle

**Structure**

一种类似于 [unified](WindowToolbarStyle/unified.zh-Hans.md) 的窗口工具栏样式，但垂直尺寸更紧凑。

## 声明

```swift
struct UnifiedCompactWindowToolbarStyle
```

## 概述

您也可以使用 [unifiedCompact](WindowToolbarStyle/unifiedCompact.zh-Hans.md) 或 [unifiedCompact(showsTitle:)](WindowToolbarStyle/unifiedCompact_showsTitle.zh-Hans.md) 来构建此样式。

## 创建窗口工具栏样式

- **init()**：创建统一的紧凑型窗口工具栏样式。

- **init(showsTitle:)**：创建统一的紧凑型窗口工具栏样式。

## 支持的类型

- **DefaultWindowToolbarStyle**：默认窗口工具栏样式。

- **ExpandedWindowToolbarStyle**：标题栏区域显示在工具栏上方的窗口工具栏样式。

- **UnifiedWindowToolbarStyle**：工具栏和标题栏内联显示的窗口工具栏样式。

# UnifiedCompactWindowToolbarStyle

**Structure**

A window toolbar style similar to [unified](WindowToolbarStyle/unified.zh-Hans.md), but with a more compact vertical sizing.

## Declaration

```swift
struct UnifiedCompactWindowToolbarStyle
```

## Overview

You can also use [unifiedCompact](WindowToolbarStyle/unifiedCompact.zh-Hans.md) or [unifiedCompact(showsTitle:)](WindowToolbarStyle/unifiedCompact_showsTitle.zh-Hans.md) to construct this style.

## Creating the window toolbar style

- **init()**: Creates a unified compact window toolbar style.
- **init(showsTitle:)**: Creates a unified compact window toolbar style.

## Supporting types

- **DefaultWindowToolbarStyle**: The default window toolbar style.
- **ExpandedWindowToolbarStyle**: A window toolbar style which displays its title bar area above the toolbar.
- **UnifiedWindowToolbarStyle**: A window toolbar style which displays its toolbar and title bar inline.

## Conforms To

- WindowToolbarStyle
