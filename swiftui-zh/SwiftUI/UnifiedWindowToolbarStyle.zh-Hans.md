# 统一窗口工具栏样式

**Structure**

一种窗口工具栏样式，它以内联方式显示工具栏和标题栏。

### 声明

```swift
struct UnifiedWindowToolbarStyle
```

## 概览

您也可以使用 [unified](WindowToolbarStyle/unified.zh-Hans.md) 或 [unified(showsTitle:)](WindowToolbarStyle/unified_showsTitle.zh-Hans.md) 构建此样式。

## 创建窗口工具栏样式

- **init()**：创建统一的窗口工具栏样式。
- **init(showsTitle:)**：创建统一的窗口工具栏样式。

## 支持类型

- **DefaultWindowToolbarStyle**：默认窗口工具栏样式。
- **ExpandedWindowToolbarStyle**：在工具栏上方显示标题栏区域的窗口工具栏样式。
- **UnifiedCompactWindowToolbarStyle**：与[unified](WindowToolbarStyle/unified.zh-Hans.md) 相似的窗口工具栏样式，但垂直尺寸更紧凑。

## 符合

- WindowToolbarStyle

# UnifiedWindowToolbarStyle

**Structure**

A window toolbar style which displays its toolbar and title bar inline.

## Declaration

```swift
struct UnifiedWindowToolbarStyle
```

## Overview

You can also use [unified](WindowToolbarStyle/unified.zh-Hans.md) or [unified(showsTitle:)](WindowToolbarStyle/unified_showsTitle.zh-Hans.md) to construct this style.

## Creating the window toolbar style

- **init()**: Creates a unified window toolbar style.
- **init(showsTitle:)**: Creates a unified window toolbar style.

## Supporting types

- **DefaultWindowToolbarStyle**: The default window toolbar style.
- **ExpandedWindowToolbarStyle**: A window toolbar style which displays its title bar area above the toolbar.
- **UnifiedCompactWindowToolbarStyle**: A window toolbar style similar to [unified](WindowToolbarStyle/unified.zh-Hans.md), but with a more compact vertical sizing.

## Conforms To

- WindowToolbarStyle
