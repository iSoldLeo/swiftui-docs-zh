# value(at:)

**实例方法**

```swift

```

# value(at:)

**Instance Method**

Returns the output value (y component) of the curve at the given time.

## Declaration

```swift
func value(at progress: Double) -> Double
```

## Return Value

The output value (y component) of the curve at the given progress.

## Getting curve characteristics

- **velocity(at:)**: Returns the rate of change (first derivative) of the output value of the curve at the given time.
