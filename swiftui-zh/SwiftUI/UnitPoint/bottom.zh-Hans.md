# 底部

**类型属性**

视图底部边缘水平居中的点。

## 声明

```swift
static let bottom: UnitPoint
```

## 讨论

该点位于[bottom](../Alignment/bottom.zh-Hans.md)对齐时水平和垂直对齐导线相交的位置。

## 获取底部点

- **bottomLeading**：位于视图底部前角的点。
- **bottomTrailing**：位于视图底部尾角的点。

# bottom

**Type Property**

A point that’s centered horizontally on the bottom edge of a view.

## Declaration

```swift
static let bottom: UnitPoint
```

## Discussion

This point occupies the position where the horizontal and vertical alignment guides intersect for [bottom](../Alignment/bottom.zh-Hans.md) alignment.

## Getting bottom points

- **bottomLeading**: A point that’s in the bottom, leading corner of a view.
- **bottomTrailing**: A point that’s in the bottom, trailing corner of a view.
