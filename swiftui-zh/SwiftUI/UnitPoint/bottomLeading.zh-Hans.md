# 底层领导

**类型属性**

位于视图底部前角的点。

## 声明

```swift
static let bottomLeading: UnitPoint
```

## 讨论

该点位于[bottomLeading](../Alignment/bottomLeading.zh-Hans.md)对齐方式的水平和垂直对齐导线相交的位置。在从左到右的语言环境中，前缘出现在左边；在从右到左的语言环境中，前缘出现在右边。

## 获取底点

- **bottom**：视图底部边缘水平居中的点。
- **bottomTrailing**：位于视图底部尾角的点。

# bottomLeading

**Type Property**

A point that’s in the bottom, leading corner of a view.

## Declaration

```swift
static let bottomLeading: UnitPoint
```

## Discussion

This point occupies the position where the horizontal and vertical alignment guides intersect for [bottomLeading](../Alignment/bottomLeading.zh-Hans.md) alignment. The leading edge appears on the left in a left-to-right language environment and on the right in a right-to-left environment.

## Getting bottom points

- **bottom**: A point that’s centered horizontally on the bottom edge of a view.
- **bottomTrailing**: A point that’s in the bottom, trailing corner of a view.
