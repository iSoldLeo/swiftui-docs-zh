# BottomTrailing

**类型属性**

位于视图底部尾角的点。

## 声明

```swift
static let bottomTrailing: UnitPoint
```

## 讨论

在[bottomTrailing](../Alignment/bottomTrailing.zh-Hans.md)对齐方式中，该点位于水平和垂直对齐指导相交的位置。在从左到右的语言环境中，尾边出现在右侧，而在从右到左的语言环境中，尾边出现在左侧。

### 获取底点

- **bottomLeading**：位于视图底部前角的点。
- **bottom**：位于视图底部边缘水平居中的点。

# bottomTrailing

**Type Property**

A point that’s in the bottom, trailing corner of a view.

## Declaration

```swift
static let bottomTrailing: UnitPoint
```

## Discussion

This point occupies the position where the horizontal and vertical alignment guides intersect for [bottomTrailing](../Alignment/bottomTrailing.zh-Hans.md) alignment. The trailing edge appears on the right in a left-to-right language environment and on the left in a right-to-left environment.

## Getting bottom points

- **bottomLeading**: A point that’s in the bottom, leading corner of a view.
- **bottom**: A point that’s centered horizontally on the bottom edge of a view.
