# 顶部

**类型属性**

视图顶部边缘水平居中的点。

## 声明

```swift
static let top: UnitPoint
```

## 讨论

该点位于[top](../Alignment/top.zh-Hans.md)对齐时水平和垂直对齐导线相交的位置。

## 获取顶点

- **topLeading**：位于视图顶部前角的点。
- **topTrailing**：位于视图顶部尾角的点。

# top

**Type Property**

A point that’s centered horizontally on the top edge of a view.

## Declaration

```swift
static let top: UnitPoint
```

## Discussion

This point occupies the position where the horizontal and vertical alignment guides intersect for [top](../Alignment/top.zh-Hans.md) alignment.

## Getting top points

- **topLeading**: A point that’s in the top, leading corner of a view.
- **topTrailing**: A point that’s in the top, trailing corner of a view.
