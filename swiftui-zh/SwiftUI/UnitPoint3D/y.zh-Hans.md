# y

**实例属性**

从原点到垂直方向上的点的归一化距离。

## 声明

```swift
var y: CGFloat
```

## 获取点的坐标

- **x**：水平方向上从原点到点的归一化距离。
- **z**：从原点到点在深度方向上的归一化距离。

# y

**Instance Property**

The normalized distance from the origin to the point in the vertical dimension.

## Declaration

```swift
var y: CGFloat
```

## Getting the point’s coordinates

- **x**: The normalized distance from the origin to the point in the horizontal direction.
- **z**: The normalized distance from the origin to the point in the depth dimension.
