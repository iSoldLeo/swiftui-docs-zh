# VerticalDirection.Set

**Structure**

一个高效的垂直方向集合。

## 声明

```swift

```

## 初始化器

- **init(_:)**：创建一个仅包含指定方向的方向集合。

## 类型属性

- **all**：一个包含向上和向下垂直方向的集合。

- **down**：一个仅包含尾部水平方向的集合。

- **up**：一个仅包含前部水平方向的集合。

# VerticalDirection.Set

**Structure**

An efficient set of vertical directions.

## Declaration

```swift
@frozen struct Set
```

## Initializers

- **init(_:)**: Creates a set of directions containing only the specified direction.

## Type Properties

- **all**: A set containing the upward and downward vertical directions.
- **down**: A set containing only the trailing horizontal direction.
- **up**: A set containing only the leading horizontal direction.

## Conforms To

- BitwiseCopyable
- Copyable
- Equatable
- ExpressibleByArrayLiteral
- OptionSet
- RawRepresentable
- Sendable
- SendableMetatype
- SetAlgebra
