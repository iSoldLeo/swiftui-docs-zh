# 垂直边缘

**Enumeration**

垂直轴上的一条边。

## 声明

```swift
@frozen enum VerticalEdge
```

## 获取边缘

- **VerticalEdge.top**：顶边。
- **VerticalEdge.bottom**：底边。

## 访问边缘集

- **VerticalEdge.Set**：一组有效的垂直边缘。

## 访问边缘、区域和布局

- **Edge**：枚举表示矩形的一条边。
- **Edge3D**：三维体的一条边或一个面。
- **HorizontalEdge**：水平轴上的边缘。
- **EdgeInsets**：矩形边的嵌入距离。
- **EdgeInsets3D**：三维体的面的嵌入距离。

# VerticalEdge

**Enumeration**

An edge on the vertical axis.

## Declaration

```swift
@frozen enum VerticalEdge
```

## Getting the edges

- **VerticalEdge.top**: The top edge.
- **VerticalEdge.bottom**: The bottom edge.

## Accessing sets of edges

- **VerticalEdge.Set**: An efficient set of vertical edges.

## Accessing edges, regions, and layouts

- **Edge**: An enumeration to indicate one edge of a rectangle.
- **Edge3D**: An edge or face of a 3D volume.
- **HorizontalEdge**: An edge on the horizontal axis.
- **EdgeInsets**: The inset distances for the sides of a rectangle.
- **EdgeInsets3D**: The inset distances for the faces of a 3D volume.
