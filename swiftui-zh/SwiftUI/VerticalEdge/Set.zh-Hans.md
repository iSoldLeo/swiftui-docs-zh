# VerticalEdge.Set

**Structure**

An efficient set of vertical edges.

## Declaration

```swift
@frozen struct Set
```

## Getting edge sets

- **all**: A set containing the top and bottom vertical edges.
- **top**: A set containing only the top vertical edge.
- **bottom**: A set containing only the bottom vertical edge.

## Creating an edge set

- **init(_:)**: Creates a set of edges containing only the specified vertical edge.
