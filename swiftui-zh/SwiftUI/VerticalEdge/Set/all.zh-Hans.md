# all

**Type Property**

A set containing the top and bottom vertical edges.

## Declaration

```swift
static let all: VerticalEdge.Set
```

## Getting edge sets

- **top**: A set containing only the top vertical edge.
- **bottom**: A set containing only the bottom vertical edge.
