#底部

**类型属性**

仅包含底部垂直边缘的集合。

## 声明

```swift
static let bottom: VerticalEdge.Set
```

## 获取边缘集

- **all**：包含顶部和底部垂直边缘的集合。
- **top**：仅包含顶部垂直边缘的集合。

# bottom

**Type Property**

A set containing only the bottom vertical edge.

## Declaration

```swift
static let bottom: VerticalEdge.Set
```

## Getting edge sets

- **all**: A set containing the top and bottom vertical edges.
- **top**: A set containing only the top vertical edge.
