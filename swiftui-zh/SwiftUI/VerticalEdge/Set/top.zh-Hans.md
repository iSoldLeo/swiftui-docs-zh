# 顶部

**类型属性**

仅包含顶部垂直边缘的集合。

## 声明

```swift
static let top: VerticalEdge.Set
```

## 获取边缘集

- **all**：包含顶部和底部垂直边缘的集合。
- **bottom**：仅包含底部垂直边缘的集合。

# top

**Type Property**

A set containing only the top vertical edge.

## Declaration

```swift
static let top: VerticalEdge.Set
```

## Getting edge sets

- **all**: A set containing the top and bottom vertical edges.
- **bottom**: A set containing only the bottom vertical edge.
