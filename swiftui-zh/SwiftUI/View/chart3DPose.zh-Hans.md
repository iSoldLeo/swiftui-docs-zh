# chart3DPose(_:)

**Instance Method**

Associates a binding to be updated when the 3D chart’s pose is changed by an interaction.

## Declaration

```swift
@MainActor @preconcurrency func chart3DPose(_ pose: Binding<Chart3DPose>) -> some View

```

## Parameters

- **pose**: The 3D chart’s current pose.
