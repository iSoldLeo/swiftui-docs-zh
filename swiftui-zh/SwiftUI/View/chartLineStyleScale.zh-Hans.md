# chartLineStyleScale(_:)

**实例方法**

配置图表的线条样式比例。

## 声明

```swift
nonisolated func chartLineStyleScale<DataValue>(_ mapping: KeyValuePairs<DataValue, StrokeStyle>) -> some View where DataValue : Plottable

```

## 参数

- **mapping**：将数据类别映射到线条样式。

## 线条样式比例

- **chartLineStyleScale(domain:)**：配置图表的线条样式比例。

- **chartLineStyleScale(domain:range:)**：配置图表的线条样式比例。

- **chartLineStyleScale(range:)**：配置图表的线条样式比例。

- **chartLineStyleScale(domain:mapping:)**：配置图表的线条样式刻度。

- **chartLineStyleScale(mapping:)**：配置图表的线条样式刻度。
