# chartLineStyleScale(range:)

**实例方法**

配置图表的线条样式比例。

## 声明

```swift
nonisolated func chartLineStyleScale<Range>(range: Range) -> some View where Range : ScaleRange, Range.VisualValue == StrokeStyle

```

## 参数

- **range**：与比例域对应的线条样式范围。

## 线条样式比例

- **chartLineStyleScale(_:)**：配置图表的线条样式比例。

- **chartLineStyleScale(domain:)**：配置图表的线条样式比例。

- **chartLineStyleScale(domain:range:)**：配置图表的线条样式比例。

- **chartLineStyleScale(domain:mapping:)**：配置图表的线条样式刻度。

- **chartLineStyleScale(mapping:)**：配置图表的线条样式刻度。
