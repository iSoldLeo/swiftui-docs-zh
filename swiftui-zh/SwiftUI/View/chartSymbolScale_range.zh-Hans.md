# chartSymbolScale(range:)

**实例方法**

配置图表符号样式比例。

## 声明

```swift

```

## 参数

- **range**：与比例域对应的符号范围。

## 符号比例

- **chartSymbolScale(_:)**：配置图表符号比例。

- **chartSymbolScale(domain:)**：配置图表符号样式比例。

- **chartSymbolScale(domain:range:)**：配置图表符号样式比例。

- **chartSymbolScale(domain:mapping:)**：配置图表符号刻度。

- **chartSymbolScale(mapping:)**：配置图表符号刻度。
