# controlGroupStyle(_:)

**实例方法**

设置此视图中控件组的样式。

## 声明

```swift
nonisolated func controlGroupStyle<S>(_ style: S) -> some View where S : ControlGroupStyle

```

## 参数

- **style**：要应用于此视图中控件的样式。

## 显示一组控件

- **ControlGroup**：一个容器视图，用于以适合上下文的视觉方式显示语义相关的控件。

# controlGroupStyle(_:)

**Instance Method**

Sets the style for control groups within this view.

## Declaration

```swift
nonisolated func controlGroupStyle<S>(_ style: S) -> some View where S : ControlGroupStyle

```

## Parameters

- **style**: The style to apply to controls within this view.

## Presenting a group of controls

- **ControlGroup**: A container view that displays semantically-related controls in a visually-appropriate manner for the context
