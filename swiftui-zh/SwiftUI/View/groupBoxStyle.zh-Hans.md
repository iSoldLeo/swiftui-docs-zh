# groupBoxStyle(_:)

**实例方法**

设置此视图中分组框的样式。

## 声明

```swift
nonisolated func groupBoxStyle<S>(_ style: S) -> some View where S : GroupBoxStyle

```

## 参数

- **style**：要应用于此视图中框的样式。

## 将视图分组到一个框中

- **GroupBox**：一个带有可选标签的样式化视图，用于以视觉方式将内容进行逻辑分组。

# groupBoxStyle(_:)

**Instance Method**

Sets the style for group boxes within this view.

## Declaration

```swift
nonisolated func groupBoxStyle<S>(_ style: S) -> some View where S : GroupBoxStyle

```

## Parameters

- **style**: The style to apply to boxes within this view.

## Grouping views into a box

- **GroupBox**: A stylized view, with an optional label, that visually collects a logical grouping of content.
