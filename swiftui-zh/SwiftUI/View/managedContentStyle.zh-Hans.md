# managedContentStyle(_:)

**实例方法**

将托管内容样式应用于视图。

## 声明

```swift
@MainActor @preconcurrency func managedContentStyle(_ style: ManagedContentStyle) -> some View

```

## 参数

- **style**：要应用于视图的样式。

## 返回值

应用样式后的视图。

## 处理托管设备

- **automatedDeviceEnrollmentAddition(isPresented:)**：显示一个模态视图，使用户能够将设备添加到其组织。

# managedContentStyle(_:)

**Instance Method**

Applies a managed content style to the view.

## Declaration

```swift
@MainActor @preconcurrency func managedContentStyle(_ style: ManagedContentStyle) -> some View

```

## Parameters

- **style**: The style to apply to the view.

## Return Value

The styled view.

## Working with managed devices

- **automatedDeviceEnrollmentAddition(isPresented:)**: Presents a modal view that enables users to add devices to their organization.
