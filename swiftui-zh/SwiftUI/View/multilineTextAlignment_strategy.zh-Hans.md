# multilineTextAlignment(strategy:)

**实例方法**

用于修饰视图层级结构中的默认文本对齐策略。

## 声明

```swift
nonisolated func multilineTextAlignment(strategy: Text.AlignmentStrategy) -> some View

```

## 说明

要在视图级别显式控制对齐方式，请选择 [layoutBased](../Text/AlignmentStrategy/layoutBased.zh-Hans.md) 模式，并将 [multilineTextAlignment](../EnvironmentValues/multilineTextAlignment.zh-Hans.md) 设置为相应的值。

# multilineTextAlignment(strategy:)

**Instance Method**

A modifier for the default text alignment strategy in the view hierarchy.

## Declaration

```swift
nonisolated func multilineTextAlignment(strategy: Text.AlignmentStrategy) -> some View

```

## Discussion

To control the alignment explicitly at a view level, choose the [layoutBased](../Text/AlignmentStrategy/layoutBased.zh-Hans.md) mode and set the [multilineTextAlignment](../EnvironmentValues/multilineTextAlignment.zh-Hans.md) to the appropriate value.
