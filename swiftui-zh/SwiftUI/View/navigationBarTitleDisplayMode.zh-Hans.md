# navigationBarTitleDisplayMode(_:)

**实例方法**

配置此视图的标题显示模式。

## 声明

```swift
nonisolated func navigationBarTitleDisplayMode(_ displayMode: NavigationBarItem.TitleDisplayMode) -> some View

```

## 参数

- **displayMode**：用于显示标题的样式。

## 配置导航栏

- **navigationBarBackButtonHidden(_:)**：隐藏视图的导航栏返回按钮。

- **NavigationBarItem**：导航栏的配置，该导航栏代表导航堆栈顶部的视图。

# navigationBarTitleDisplayMode(_:)

**Instance Method**

Configures the title display mode for this view.

## Declaration

```swift
nonisolated func navigationBarTitleDisplayMode(_ displayMode: NavigationBarItem.TitleDisplayMode) -> some View

```

## Parameters

- **displayMode**: The style to use for displaying the title.

## Configuring the navigation bar

- **navigationBarBackButtonHidden(_:)**: Hides the navigation bar back button for the view.
- **NavigationBarItem**: A configuration for a navigation bar that represents a view at the top of a navigation stack.
