# pointerVisibility(_:)

**实例方法**

设置指针悬停在视图上时的可见性。

## 声明

```swift
nonisolated func pointerVisibility(_ visibility: Visibility) -> some View

```

## 参数

- **visibility**：要应用的指针可见性。

## 返回值

一个视图，当指针悬停在视图上时，该视图会改变指针的可见性。

## 修改指针外观

- **pointerStyle(_:)**：设置指针悬停在视图上时要显示的指针样式。

- **PointerStyle**：一种样式，用于描述鼠标悬停在视图上时指针（也称光标）的外观。

# pointerVisibility(_:)

**Instance Method**

Sets the visibility of the pointer when it’s over the view.

## Declaration

```swift
nonisolated func pointerVisibility(_ visibility: Visibility) -> some View

```

## Parameters

- **visibility**: The pointer visibility to apply.

## Return Value

A view that changes the visibility of the pointer when hovered.

## Modifying pointer appearance

- **pointerStyle(_:)**: Sets the pointer style to display when the pointer is over the view.
- **PointerStyle**: A style describing the appearance of the pointer (also called a cursor) when it’s hovered over a view.
