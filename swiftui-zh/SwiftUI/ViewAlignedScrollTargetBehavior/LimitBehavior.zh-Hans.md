# ViewAlignedScrollTargetBehavior.LimitBehavior

**Structure**

定义一次可滚动的视图数量的类型。

## 声明

```swift
struct LimitBehavior
```

## 获取极限行为

- **automatic**：自动限制行为。
- **always**：始终限幅行为。
- **never**：从不限制行为。

### 类型属性

- **alwaysByFew**：总是逐少限制行为。
- **alwaysByOne**：总是逐一限制行为。

## 创建目标行为

- **init(limitBehavior:)**：创建视图对齐滚动行为。

# ViewAlignedScrollTargetBehavior.LimitBehavior

**Structure**

A type that defines the amount of views that can be scrolled at a time.

## Declaration

```swift
struct LimitBehavior
```

## Getting the limit behavior

- **automatic**: The automatic limit behavior.
- **always**: The always limit behavior.
- **never**: The never limit behavior.

## Type Properties

- **alwaysByFew**: The always-by-few limit behavior.
- **alwaysByOne**: The always-by-one limit behavior.

## Creating the target behavior

- **init(limitBehavior:)**: Creates a view aligned scroll behavior.
