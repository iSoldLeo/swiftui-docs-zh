# init(limitBehavior:)

**Initializer**

Creates a view aligned scroll behavior.

## Declaration

```swift
init(limitBehavior: ViewAlignedScrollTargetBehavior.LimitBehavior = .automatic)
```

## Creating the target behavior

- **ViewAlignedScrollTargetBehavior.LimitBehavior**: A type that defines the amount of views that can be scrolled at a time.
