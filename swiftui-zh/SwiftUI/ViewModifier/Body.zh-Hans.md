# Body

**Associated Type**

The type of view representing the body.

## Declaration

```swift
associatedtype Body : View
```

## Creating a view modifier

- **body(content:)**: Gets the current body of the caller.
- **ViewModifier.Content**: The content view type passed to `body()`.
