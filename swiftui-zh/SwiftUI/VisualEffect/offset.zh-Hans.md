# offset(_:)

**实例方法**

按照偏移参数中指定的水平和垂直量偏移视图。

## 声明

```swift
func offset(_ offset: CGSize) -> some VisualEffect

```

## 参数

- **offset**：偏移视图的距离。

## 返回值

将视图偏移 `offset` 的效果。

## 翻译

- **offset(x:y:)**：按指定的水平和垂直距离偏移视图。
- **offset(z:)**：以点为单位将视图在 Z 轴向前移动指定的距离。

# offset(_:)

**Instance Method**

Offsets the view by the horizontal and vertical amount specified in the offset parameter.

## Declaration

```swift
func offset(_ offset: CGSize) -> some VisualEffect

```

## Parameters

- **offset**: The distance to offset the view.

## Return Value

An effect that offsets the view by `offset`.

## Translating

- **offset(x:y:)**: Offsets the view by the specified horizontal and vertical distances.
- **offset(z:)**: Brings a view forward in Z by the provided distance in points.
