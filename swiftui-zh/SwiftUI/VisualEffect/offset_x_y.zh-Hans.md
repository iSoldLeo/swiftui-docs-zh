# offset(x:y:)

**实例方法**

按指定的水平和垂直距离偏移视图。

## 声明

```swift
func offset(x: CGFloat = 0, y: CGFloat = 0) -> some VisualEffect

```

## 参数

- **x**：偏移视图的水平距离。
- **y**：偏移视图的垂直距离。

## 返回值

将视图偏移 `x` 和 `y` 的效果。

## 翻译

- **offset(_:)**：按偏移参数中指定的水平和垂直量偏移视图。
- **offset(z:)**：以点为单位将视图在 Z 轴向前移动所提供的距离。

# offset(x:y:)

**Instance Method**

Offsets the view by the specified horizontal and vertical distances.

## Declaration

```swift
func offset(x: CGFloat = 0, y: CGFloat = 0) -> some VisualEffect

```

## Parameters

- **x**: The horizontal distance to offset the view.
- **y**: The vertical distance to offset the view.

## Return Value

An effect that offsets the view by `x` and `y`.

## Translating

- **offset(_:)**: Offsets the view by the horizontal and vertical amount specified in the offset parameter.
- **offset(z:)**: Brings a view forward in Z by the provided distance in points.
