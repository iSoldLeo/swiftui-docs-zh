# Body

**Associated Type**

The type of configuration representing the content of the widget.

## Declaration

```swift
associatedtype Body : WidgetConfiguration
```

## Discussion

When you create a custom widget, Swift infers this type from your implementation of the required [body-swift.property](body-swift_property.zh-Hans.md) property.

## Implementing a widget

- **body**: The content and behavior of the widget.
