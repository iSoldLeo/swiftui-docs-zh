# 隐藏标题栏

**类型属性**

一种窗口样式，用于隐藏窗口标题和标题栏区域的背景，以便显示更多的窗口内容。

### 声明

```swift
static var hiddenTitleBar: HiddenTitleBarWindowStyle { get }
```

## 获取内置窗口样式

- **automatic**：默认窗口样式。
- **plain**：普通窗口样式。
- **titleBar**：默认窗口样式：显示窗口标题栏部分的窗口样式。
- **volumetric**：一种窗口样式，用于创建三维体积窗口。

# hiddenTitleBar

**Type Property**

A window style which hides both the window’s title and the backing of the titlebar area, allowing more of the window’s content to show.

## Declaration

```swift
static var hiddenTitleBar: HiddenTitleBarWindowStyle { get }
```

## Getting built-in window styles

- **automatic**: The default window style.
- **plain**: The plain window style.
- **titleBar**: A window style which displays the title bar section of the window.
- **volumetric**: A window style that creates a 3D volumetric window.
