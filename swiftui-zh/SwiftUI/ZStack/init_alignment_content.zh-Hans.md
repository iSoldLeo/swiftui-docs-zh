# init(alignment:content:)

**Initializer**

创建一个具有指定对齐方式的实例。

## 声明

```swift
init(alignment: Alignment = .center, @ViewBuilder content: () -> Content)
```

## 参数

- **alignment**：用于在 x 轴和 y 轴上对齐此堆栈中子视图的指南。

- **content**：用于创建此堆栈内容的视图构建器。

# init(alignment:content:)

**Initializer**

Creates an instance with the given alignment.

## Declaration

```swift
init(alignment: Alignment = .center, @ViewBuilder content: () -> Content)
```

## Parameters

- **alignment**: The guide for aligning the subviews in this stack on both the x- and y-axes.
- **content**: A view builder that creates the content of this stack.
