# init(alignment:)

**Initializer**

以指定的对齐方式创建堆栈。

## 声明

```swift
init(alignment: Alignment = .center)
```

## 参数

- **alignment**：用于在 x 轴和 y 轴上对齐该堆栈中的子视图的向导。

# init(alignment:)

**Initializer**

Creates a stack with the specified alignment.

## Declaration

```swift
init(alignment: Alignment = .center)
```

## Parameters

- **alignment**: The guide for aligning the subviews in this stack on both the x- and y-axes.
