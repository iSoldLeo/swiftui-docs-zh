# AccessibilityRotorContentBuilder

**Structure**

Result builder you use to generate rotor entry content.

## Declaration

```swift
@resultBuilder struct AccessibilityRotorContentBuilder
```

## Building navigation content

- **buildBlock(_:)**
- **buildIf(_:)**
- **buildExpression(_:)**: Builds an expression within the builder.

## Creating rotors

- **AccessibilityRotorContent**: Content within an accessibility rotor.
- **AccessibilityRotorEntry**: A struct representing an entry in an Accessibility Rotor.
