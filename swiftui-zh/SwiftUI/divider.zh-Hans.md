# 分隔线

**Structure**

一种可用于分隔其他内容的视觉元素。

## 声明

```swift
struct Divider
```

## 概述

当分隔线包含在堆栈中时，它会沿着堆栈的短轴扩展；如果不包含在堆栈中，则会水平扩展。

## 创建分隔线

- **init()**

## 分隔符

- **Spacer**：一个灵活的空间，它会沿着其所在的堆栈布局的长轴扩展；如果不包含在堆栈中，则会沿着两个轴扩展。

## 符合以下规范

- View

# Divider

**Structure**

A visual element that can be used to separate other content.

## Declaration

```swift
struct Divider
```

## Overview

When contained in a stack, the divider extends across the minor axis of the stack, or horizontally when not in a stack.

## Creating a divider

- **init()**

## Separators

- **Spacer**: A flexible space that expands along the major axis of its containing stack layout, or on both axes if not contained in a stack.
