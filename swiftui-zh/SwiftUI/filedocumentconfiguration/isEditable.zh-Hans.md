# isEditable

**实例属性**

一个布尔值，指示是否可以编辑文档。

## 声明

```swift
var isEditable: Bool
```

## 说明

如果文档处于查看模式或文件不可写，则此值为 `false`。

## 获取文档属性

- **fileURL**：已打开文件的 URL。

# isEditable

**Instance Property**

A Boolean that indicates whether you can edit the document.

## Declaration

```swift
var isEditable: Bool
```

## Discussion

This value is `false` if the document is in viewing mode, or if the file is not writable.

## Getting document properties

- **fileURL**: The URL of the open file document.
