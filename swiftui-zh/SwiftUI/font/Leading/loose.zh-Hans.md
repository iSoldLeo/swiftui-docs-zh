# Font.Leading.loose

**Case**

增加行间距。

## 声明

```swift
case loose
```

## 讨论

此值通常会将 watchOS 上的行间距增加 1 磅，其他平台上的行间距增加 2 磅。

## 获取前导行间距选项

- **Font.Leading.standard**：字体的默认行间距。

- **Font.Leading.tight**：减少行间距。

# Font.Leading.loose

**Case**

Increased line spacing.

## Declaration

```swift
case loose
```

## Discussion

This value typically increases line spacing by 1 point for watchOS and 2 points on other platforms.

## Getting leading line spacing options

- **Font.Leading.standard**: The font’s default line spacing.
- **Font.Leading.tight**: Reduced line spacing.
