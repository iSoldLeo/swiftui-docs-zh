# Font.Leading.standard

**Case**

The font’s default line spacing.

## Declaration

```swift
case standard
```

## Discussion

If you modify a font to use a nonstandard line spacing like [tight](tight.zh-Hans.md) or [loose](loose.zh-Hans.md), you can use this value to return to the font’s default line spacing.

## Getting leading line spacing options

- **Font.Leading.loose**: Increased line spacing.
- **Font.Leading.tight**: Reduced line spacing.
