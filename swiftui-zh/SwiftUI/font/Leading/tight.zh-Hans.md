# Font.Leading.tight

**Case**

缩小行距。

## 声明

```swift
case tight
```

## 讨论

此值通常会将 watchOS 上的行距减少 1 磅，其他平台上的行距减少 2 磅。

## 获取前导行距选项

- **Font.Leading.standard**：字体的默认行距。

- **Font.Leading.loose**：增大行距。

# Font.Leading.tight

**Case**

Reduced line spacing.

## Declaration

```swift
case tight
```

## Discussion

This value typically reduces line spacing by 1 point for watchOS and 2 points on other platforms.

## Getting leading line spacing options

- **Font.Leading.standard**: The font’s default line spacing.
- **Font.Leading.loose**: Increased line spacing.
