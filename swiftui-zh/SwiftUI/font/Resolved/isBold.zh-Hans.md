# isBold

**实例属性**

如果解析后的字体根据 CoreText 具有粗体特性，或者字体的字重为半粗或更粗，则返回 `true`。

## 声明

```swift
var isBold: Bool { get }
```

# isBold

**Instance Property**

Returns `true` if the resolved font has a bold trait according to CoreText or the font’s weight is semi-bold or greater.

## Declaration

```swift
var isBold: Bool { get }
```
