# Font.TextStyle.subheadline

**Case**

The font used for subheadings.

## Declaration

```swift
case subheadline
```

## Getting font text styles

- **Font.TextStyle.extraLargeTitle2**: The font used for second level extra large titles.
- **Font.TextStyle.extraLargeTitle**: The font used for extra large titles.
- **Font.TextStyle.largeTitle**: The font style for large titles.
- **Font.TextStyle.title**: The font used for first level hierarchical headings.
- **Font.TextStyle.title2**: The font used for second level hierarchical headings.
- **Font.TextStyle.title3**: The font used for third level hierarchical headings.
- **Font.TextStyle.headline**: The font used for headings.
- **Font.TextStyle.body**: The font used for body text.
- **Font.TextStyle.callout**: The font used for callouts.
- **Font.TextStyle.caption**: The font used for standard captions.
- **Font.TextStyle.caption2**: The font used for alternate captions.
- **Font.TextStyle.footnote**: The font used in footnotes.
