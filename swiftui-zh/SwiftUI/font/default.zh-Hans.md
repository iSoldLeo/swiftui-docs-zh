# default

**类型属性**

在任何给定环境下使用的有效 SwiftUI 字体。

## 声明

```swift
static var `default`: Font { get }
```

## 说明

此属性指定环境使用的字体，优先选择开发者指定的字体（通过 [font](../EnvironmentValues/font.zh-Hans.md)），其次是框架指定的字体，最后是 SwiftUI 默认字体。

# default

**Type Property**

The effective SwiftUI font used in any given environment.

## Declaration

```swift
static var `default`: Font { get }
```

## Discussion

The font specified by environment, preferring first any developer spedified font, via [font](../EnvironmentValues/font.zh-Hans.md), then any framework specified font, and finally the default SwiftUI font.
