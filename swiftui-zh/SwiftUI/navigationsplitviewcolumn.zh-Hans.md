# NavigationSplitViewColumn

**Structure**

A view that represents a column in a navigation split view.

## Declaration

```swift
struct NavigationSplitViewColumn
```

## Overview

A [NavigationSplitView](NavigationSplitView.zh-Hans.md) collapses into a single stack in some contexts, like on iPhone or Apple Watch. Use this type with the `preferredCompactColumn` parameter to control which column of the navigation split view appears on top of the collapsed stack.
