# PlainTextEditorStyle

**Structure**

A text editor style with no decoration.

## Declaration

```swift
struct PlainTextEditorStyle
```

## Overview

You can also use [plain](TextEditorStyle/plain.zh-Hans.md) to create this style.

## Creating the text editor style

- **init()**

## Supporting types

- **AutomaticTextEditorStyle**: The default text editor style, based on the text editor’s context.
- **RoundedBorderTextEditorStyle**: A text editor style with a system-defined rounded border.
