# RoundedBorderTextFieldStyle

**Structure**

A text field style with a system-defined rounded border.

## Declaration

```swift
struct RoundedBorderTextFieldStyle
```

## Overview

You can also use [roundedBorder](TextFieldStyle/roundedBorder.zh-Hans.md) to construct this style.

## Creating the text field style

- **init()**

## Supporting types

- **DefaultTextFieldStyle**: The default text field style, based on the text field’s context.
- **PlainTextFieldStyle**: A text field style with no decoration.
- **SquareBorderTextFieldStyle**: A text field style with a system-defined square border.

## Conforms To

- TextFieldStyle
