# Body

**Associated Type**

The type of scene that represents the body of this scene.

## Declaration

```swift
associatedtype Body : Scene
```

## Discussion

When you create a custom scene, Swift infers this type from your implementation of the required [body-swift.property](body-swift_property.zh-Hans.md) property.

## Creating a scene

- **body**: The content and behavior of the scene.
