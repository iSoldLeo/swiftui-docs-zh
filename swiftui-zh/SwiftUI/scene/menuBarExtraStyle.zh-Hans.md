# menuBarExtraStyle(_:)

**Instance Method**

Sets the style for menu bar extra created by this scene.

## Declaration

```swift
nonisolated func menuBarExtraStyle<S>(_ style: S) -> some Scene where S : MenuBarExtraStyle

```

## Creating a menu bar extra

- **MenuBarExtra**: A scene that renders itself as a persistent control in the system menu bar.
- **MenuBarExtraStyle**: A specification for the appearance and behavior of a menu bar extra scene.
