# centered

**Type Property**

The target view is centered within the container

## Declaration

```swift
static var centered: ScrollTransitionConfiguration.Threshold { get }
```

## Getting the threshold

- **hidden**
- **visible**
- **visible(_:)**: The target view is visible by the given amount, where zero is fully hidden, and one is fully visible.
