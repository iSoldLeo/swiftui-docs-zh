# orientation

**实例属性**

一个 3D 旋转，旋转中心点位于四个边之一的中心。

## 声明

```swift
var orientation: Rotation3D { get }
```

## 讨论

该角度将等于 `0°`、`90°`、`180°` 或 `270°`。

# orientation

**Instance Property**

A 3D rotation that is snapped to the center of one of the four sides.

## Declaration

```swift
var orientation: Rotation3D { get }
```

## Discussion

The angle will be equal to `0°`, `90°`, `180°`, or `270°`.
