- **SystemFormatStyle.Stopwatch**：系统秒表格式样式。

# SystemFormatStyle

**Enumeration**

A namespace for format styles that implement designs used across Apple’s platformes.

## Declaration

```swift
enum SystemFormatStyle
```

## Structures

- **SystemFormatStyle.DateOffset**: A system format to display the offset to a certain date.
- **SystemFormatStyle.DateReference**: A system format style to refer to a date in the most natural way.
- **SystemFormatStyle.Stopwatch**: The system stopwatch format style.
- **SystemFormatStyle.Timer**: The system timer format style.

## Formatting date and time

- **TimeDataSource**: A source of time related data.
