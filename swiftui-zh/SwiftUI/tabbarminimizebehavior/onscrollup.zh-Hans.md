# onScrollUp

**类型属性**

当开始向上滚动时，最小化标签栏。只有 iPhone 支持将标签栏最小化。

## 声明

```swift
static let onScrollUp: TabBarMinimizeBehavior
```

# onScrollUp

**Type Property**

Minimize the tab bar when upwards scrolling starts. Minimizing is supported for tab bars on only iPhone.

## Declaration

```swift
static let onScrollUp: TabBarMinimizeBehavior
```
