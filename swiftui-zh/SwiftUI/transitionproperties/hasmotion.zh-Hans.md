# hasMotion

**实例属性**

转换是否包含动作。

## 声明

```swift
var hasMotion: Bool
```

## 讨论

如果在过渡效果中包含此行为，则在启用 "减少运动 "时，该过渡效果将被不透明度取代。

默认值为 `true`。

## 创建过渡属性

- **init(hasMotion:)**

# hasMotion

**Instance Property**

Whether the transition includes motion.

## Declaration

```swift
var hasMotion: Bool
```

## Discussion

When this behavior is included in a transition, that transition will be replaced by opacity when Reduce Motion is enabled.

Defaults to `true`.

## Creating the transition properties

- **init(hasMotion:)**
