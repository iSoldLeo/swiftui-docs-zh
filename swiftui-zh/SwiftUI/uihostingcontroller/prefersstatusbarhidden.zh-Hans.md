# prefersStatusBarHidden

**实例属性**

布尔值，表示视图控制器更喜欢隐藏还是显示状态栏。

## 声明

```swift
@MainActor @preconcurrency override dynamic var prefersStatusBarHidden: Bool { get }
```

## 配置状态栏

- **preferredStatusBarStyle**：视图控制器首选的状态栏样式。
- **preferredStatusBarUpdateAnimation**：隐藏或显示该视图控制器的状态栏时要使用的动画样式。
- **childForStatusBarStyle**：隐藏或显示该视图控制器的状态栏时使用的动画样式。
- **childForStatusBarHidden**：该视图控制器隐藏或显示状态栏时使用的动画样式。

# prefersStatusBarHidden

**Instance Property**

A Boolean value that indicates whether the view controller prefers the status bar to be hidden or shown.

## Declaration

```swift
@MainActor @preconcurrency override dynamic var prefersStatusBarHidden: Bool { get }
```

## Configuring the status bar

- **preferredStatusBarStyle**: The preferred status bar style for the view controller.
- **preferredStatusBarUpdateAnimation**: The animation style to use when hiding or showing the status bar for this view controller.
- **childForStatusBarStyle**
- **childForStatusBarHidden**
