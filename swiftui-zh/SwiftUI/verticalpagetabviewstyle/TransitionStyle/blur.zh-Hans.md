# 模糊

**类型属性**

在每个选项卡之间模糊内容的过渡样式

## 声明

```swift
static let blur: VerticalPageTabViewStyle.TransitionStyle
```

## 获取过渡样式

- **automatic**：自动过渡样式
- **identity**：每个选项卡之间没有动画的过渡样式

# blur

**Type Property**

A transition style that blurs content between each tab

## Declaration

```swift
static let blur: VerticalPageTabViewStyle.TransitionStyle
```

## Getting the transition styles

- **automatic**: Automatic transition style
- **identity**: A transition style that has no animation between each tab
